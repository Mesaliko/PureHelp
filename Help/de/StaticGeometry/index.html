PureBasic - StaticGeometry

Einleitung

Eine "statische Geometrie" (englisch "static geometry") ist eine vordefinierte und vorgerenderte Geometrie-Form, die sehr komplex sein kann und trotzdem noch ein sehr schnelles Rendering besitzt. Sobald sie erstellt wurde, kann die Geometrie-Form nicht mehr bewegt werden.

InitEngine3D() muss erfolgreich aufgerufen worden sein, bevor die Funktionen der "statischen Geometrie" verwendet werden.

Befehlsübersicht

AddStaticGeometryEntity
BuildStaticGeometry
CreateStaticGeometry
FreeStaticGeometry
IsStaticGeometry
