GetRuntimeDouble()

Syntax

Ergebnis = GetRuntimeDouble(Objekt$)
Beschreibung
Gibt den Double-Wert des Runtime-Objekts zurück.

Parameter

Objekt$ Name des Objekts, von dem der Wert ermittelt werden soll. Die folgenden Objekte werden unterstützt:
- Variable: der Objektname ist 'VariablenName' (ohne Beachtung von Groß- und Kleinschreibung).
- Konstante: der Objektname ist '#KonstantenName' (ohne Beachtung von Groß- und Kleinschreibung).

Beim Zugriff auf öffentliche Module-Elemente ist der Modulpräfixname obligatorisch, auch wenn UseModule verwendet wird.

Rückgabewert

Gibt den Double-Wert des angegebenen Objekts zurück oder Null, falls es nicht gefunden wurde. Da Null ein gültiger Double-Wert ist, kann IsRuntime() verwendet werden, um festzustellen, ob das Runtime-Objekt tatsächlich existiert. Ist die Variable vom Typ Integer oder Float, wird sie automatisch in Double umgewandelt.
