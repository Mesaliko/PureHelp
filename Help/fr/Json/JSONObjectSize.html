JSONObjectSize()

Syntaxe

Resultat = JSONObjectSize(JSONValeur)
Description
Renvoie le nombre de membres dans une valeur JSON de type #PB_JSON_Object.

Arguments

JSONValeur La valeur JSON.

La valeur doit être de type #PB_JSON_Object.

Valeur de retour

Le nombre de membres dans l'objet JSON.

Exemple

  Donnee$ = "{ " + Chr(34) + "x" + Chr(34) + ": 10, " + 
                   Chr(34) + "y" + Chr(34) + ": 20, " + 
                   Chr(34) + "z" + Chr(34) + ": 30 }"
  
  ParseJSON(0, Donnee$)
  Debug JSONObjectSize(JSONValue(0))
  
  ; Resultat : 3
