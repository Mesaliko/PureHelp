Mid()

Syntaxe

Resultat$ = Mid(Chaine$, PositionDepart [, Longueur])
Description
Extrait une sous-chaîne d'une chaîne.

Arguments

Chaine$ La chaîne de caractères à utiliser.
PositionDepart La position de départ (commence à 1).
Longueur (optionnel) Précise combien de caractères seront extraits.

Si le paramètre 'Longueur' est omis, les caractères sont extraits jusqu'à la fin de la chaîne 'Chaine$'.

Valeur de retour

Renvoie une sous-chaîne de longueur 'Longueur', à partir du caractère situé à la position 'PositionDepart'.

Exemple

  Debug Mid("Hello", 2) ; Affichera "ello"
  Debug Mid("Hello", 2, 1) ; Affichera "e"
