ScreenID()

Syntax

ID = ScreenID()
Beschreibung
Gibt die ScreenID des OS zurück.

Parameter

Diese Funktion hat keine Parameter.

Rückgabewert

Die ScreenID des OS. Unter Windows ist dies die normale WindowID. Damit kann jeder Befehl diesen Wert verwenden, der eine solche ID benutzt (wie zum Beispiel PlayMovie()).

Unterstützte OS

Windows (DirectX), Linux (SDL)
