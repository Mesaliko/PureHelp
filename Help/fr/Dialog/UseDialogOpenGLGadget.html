UseDialogOpenGLGadget()

Syntaxe

UseDialogOpenGLGadget()
Description
Active la prise en charge de OpenGLGadget() dans la bibliothèque de dialogue.
Ceci n'est pas activé par défaut pour réduire la taille de la bibliothèque de dialogue si OpenGLGadget() n'est pas nécessaire.

Valeur de retour

Aucune.
