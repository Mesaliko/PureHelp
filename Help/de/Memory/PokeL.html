PokeL()

Syntax

PokeL(*Speicherpuffer, Nummer)
Beschreibung
Schreibt einen Long-Wert (4 Bytes) an die angegebene Speicheradresse.

Parameter

*Speicherpuffer Die Adresse, an die geschrieben werden soll.
Nummer Der zu schreibende Wert.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.
