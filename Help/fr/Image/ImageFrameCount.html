ImageFrameCount()

Syntaxe

Resultat = ImageFrameCount(#Image)
Description
Renvoie le nombre de trames de l'image.

Arguments

#Image L'image à utiliser.

Valeur de retour

Renvoie le nombre de trames de l'image. Renvoie toujours 1 si l'image n'est pas multi-trames.

Voir aussi

CreateImage(), AddImageFrame(), RemoveImageFrame(), SetImageFrame(), GetImageFrame()

OS Supportés

Tous
