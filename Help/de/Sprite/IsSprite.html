IsSprite()

Syntax

Ergebnis = IsSprite(#Sprite)
Beschreibung
Überprüft, ob die angegebene #Sprite Nummer gültig und ein korrekt initialisiertes Sprite ist.

Parameter

#Sprite Das zu verwendende Sprite.

Rückgabewert

Ein Wert ungleich Null, wenn #Sprite ein gültiges Sprite ist, andernfalls Null.

Anmerkungen

Diese Funktion ist "kugelsicher" und kann mit jedem Wert benutzt werden. Ist das 'Ergebnis' ungleich Null, dann ist das Objekt gültig und initialisiert, andernfalls wird Null zurückgegeben. Dies ist der richtige Weg, um sicher zu stellen, dass ein Sprite bereit zur Verwendung ist.
