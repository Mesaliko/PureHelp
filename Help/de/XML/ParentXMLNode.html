ParentXMLNode()

Syntax

*Ergebnis = ParentXMLNode(*Knoten)
Beschreibung
Gibt den übergeordneten Knoten (englisch "parent node") des angegebenen XML-Knotens zurück. Jeder XML-Knoten hat einen übergeordneten Knoten, außer der Wurzelknoten.

Parameter

*Knoten Der XML-Knoten, von dem der übergeordnete Knoten ("parent") ermittelt werden soll.

Rückgabewert

Gibt einen Zeiger auf den übergeordneten Knoten zurück, oder Null wenn '*Knoten' bereits der Wurzelknoten ist.
