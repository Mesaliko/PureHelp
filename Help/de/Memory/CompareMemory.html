CompareMemory()

Syntax

Ergebnis = CompareMemory(*MemoryID1, *MemoryID2, Größe)
Beschreibung
Vergleicht den Inhalt von zwei Speicherbereichen.

Parameter

*MemoryID1, *MemoryID2 Die Adressen der zwei zu vergleichenden Speicherbereiche.
Größe Die Anzahl Bytes, welche verglichen werden sollen.

Rückgabewert

Gibt einen Wert ungleich Null zurück, wenn die zwei Bereiche die gleichen Bytes enthalten, oder Null wenn der Inhalt nicht übereinstimmt.
