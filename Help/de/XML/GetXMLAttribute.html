GetXMLAttribute()

Syntax

Ergebnis$ = GetXMLAttribute(*Knoten, Attribut$)
Beschreibung
Gibt den Wert eines Attributs vom angegebenen XML-Knoten zurück.

Parameter

*Knoten Der XML-Knoten, von dem das Attribut ermittelt werden soll.
Attribut$ Der Attribut-Name.

Rückgabewert

Der Wert des Attributs im angegebenen XML-Knoten. Wenn das Attribut nicht existiert, wird ein leerer String zurückgegeben.

Nur Knoten vom Typ #PB_XML_Normal können Attribute haben. Für alle anderen Knoten-Typen erzeugt der Compiler eine Fehlermeldung.
