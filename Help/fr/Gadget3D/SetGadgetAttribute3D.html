SetGadgetAttribute3D()

Syntaxe

SetGadgetAttribute3D(#Gadget3D, Attribut, Valeur)
Description
Change la valeur d'un attribut.

Arguments

#Gadget3D Le gadget 3D à utiliser.
Attribut L'attribut à utiliser.
Valeur La valeur à donner à l'attribut.

Valeur de retour

Aucune.

Remarques

Cette fonction est disponible avec les gadgets suivants:

- EditorGadget3D()
- ProgressBarGadget3D()
- ScrollAreaGadget3D()
- ScrollBarGadget3D()
- SpinGadget3D()

OS Supportés

Tous

<- SetActiveGadget3D() - Gadget3D Index - SetGadgetData3D() ->
