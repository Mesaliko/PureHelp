TransparentSpriteColor()

Syntax

TransparentSpriteColor(#Sprite, Farbe)
Beschreibung
Verwendet die angegebene Farbe als transparente Sprite-Farbe (bei Anzeige mit DisplayTransparentSprite()). Es kann nur eine Farbe als transparente Farbe festgelegt werden. Die vorherigen Alphakanalwerte gehen verloren. Wenn das Sprite bereits Alpha-Informationen enthält, wird dieser Befehl wahrscheinlich nicht benötigt. Um diesen Befehl verwenden zu können, muss das Sprite mit dem Flag #PB_Sprite_AlphaBlending erstellt werden.

Parameter

#Sprite Das zu verwendende Sprite. Wenn #PB_Default verwendet wird, dann wird die Standardfarbe (schwarz - RGB(0,0,0)) auf die neu angegebene Farbe geändert und alle zukünftig geladenen oder erstellten Sprites (mittels LoadSprite() bzw. CreateSprite()) benutzen diese Farbe als transparente Farbe.
Farbe Die neue Farbe, welche als transparente Farbe behandelt werden soll. RGB() kann verwendet werden, um einen gültigen Farbwert zu erhalten. Eine Tabelle gebräuchlicher Farbwerte ist hier verfügbar.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.
