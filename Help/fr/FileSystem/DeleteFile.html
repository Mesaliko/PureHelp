DeleteFile()

Syntaxe

Resultat = DeleteFile(NomFichier$ [, Mode]))
Description
Supprime un fichier.

Arguments

NomFichier$ Le fichier à supprimer.
Mode (optionnel) Peut être une des valeurs suivantes: 
  #PB_FileSystem_Force: Supprime aussi les fichiers qui sont protégés (en lecture seule).

Valeur de retour

Renvoie une valeur non nulle si l'opération a réussi, zéro sinon.

Exemple

  If FileSize("c:\PB") = -2
    DeleteFile("c:\PB\ok.txt")
    Debug "Fichier supprimé"
  EndIf
