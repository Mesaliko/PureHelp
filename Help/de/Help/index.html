PureBasic - Help

Einleitung

Die Hilfe ist ein sehr bedeutender Bestandteil jeder Art von Software. Sie gibt den Endanwendern die Möglichkeit, die Software-Funktionen schnell und einfach zu entdecken.
Die PureBasic Help-Funktionen ermöglichen dem Programmierer, Standard Hilfe-Dateien mit globaler und kontextbezogener Unterstützung anzuzeigen.

Auf Microsoft Windows gibt es zwei verschiedene Formate von Hilfe-Dateien, welche unterstützt werden: Das ".hlp" Format (was das alte Format ist) und das ".chm" Format (was das neue Format ist). Das ".chm" Format basiert auf HTML, was für "Hyper Text Markup Language" steht.
