CountGadgetItems3D()

Syntax

Ergebnis = CountGadgetItems3D(#Gadget3D)
Beschreibung
Gibt die Anzahl der Einträge im angegebenen #Gadget3D zurück.

Parameter

#Gadget3D Das zu verwendende 3D-Gadget.

Rückgabewert

Die Anzahl an Einträgen im angegebenen #Gadget3D.

Anmerkungen

Diese Funktion wird von den folgenden 3D-Gadgets unterstützt:

- ComboBoxGadget3D()
- ListViewGadget3D()
- PanelGadget3D()

Unterstützte OS

Alle

<- ContainerGadget3D() - Gadget3D Inhaltsverzeichnis - DisableGadget3D() ->
