RemoveXMLAttribute()

Syntax

RemoveXMLAttribute(*Knoten, Attribut$)
Beschreibung
Entfernt das Attribut vom angegebenen XML-Knoten.

Parameter

*Knoten Der XML-Knoten, von dem das Attribut entfernt werden soll.
Attribut$ Der zu entfernende Attribut-Name. Wenn das Attribut nicht existiert, passiert nichts.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.

Anmerkungen

Nur Knoten vom Typ #PB_XML_Normal können Attribute haben. Für alle anderen Knoten-Typen wird diese Funktion ignoriert.
