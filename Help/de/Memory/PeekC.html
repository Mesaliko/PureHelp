PeekC()

Syntax

Wert.c = PeekC(*Speicherpuffer)
Beschreibung
Liest einen Character-Wert (2 Bytes in Unicode) von der angegebenen Speicheradresse.

Parameter

*Speicherpuffer Die Adresse, von der gelesen werden soll.

Rückgabewert

Gibt den Wert des Characters (Zeichen) zurück.
