ScaleMaterial()

Syntax

ScaleMaterial(#Material, x, y [, Layer])
Beschreibung
Skaliert das Material. Die Parameter 'x' und 'y' sind Skalierungsfaktoren.

Parameter

#Material Das zu verwendende Material.
x, y Die Skalierungsfaktoren (die aktuelle Materialgröße wird mit diesen Werten multipliziert):
- ein Wert von 1.0 bedeutet, dass die Größe nicht verändert wird
- ein Wert zwischen 0.0 und 1.0 bedeutet, dass das Material verkleinert wird (z.B. ein Faktor von 0.5 wird die Größe halbieren)
- ein Wert oberhalb von 1.0 bedeutet, dass das Material vergrößert wird (z.B. ein Faktor von 2.0 wird die Größe verdoppeln)
Layer (optional) Der zu skalierende Layer. Der erste Layer hat den Wert Null (wird dieser Parameter weggelassen, dann wird der Layer Null skaliert).

Rückgabewert

Diese Funktion gibt keinen Wert zurück.

Anmerkungen

Um die aktuellen Skalierungs-Werte zu erhalten, verwenden Sie GetMaterialAttribute().
