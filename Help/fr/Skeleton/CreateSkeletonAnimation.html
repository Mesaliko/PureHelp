CreateSkeletonAnimation()

Syntaxe

CreateSkeletonAnimation(#Mesh, NomAnimation$, Duree)
Description
Crée une nouvelle animation de squelette.

Arguments

#Mesh Le mesh à utiliser.
NomAnimation$ Le nom de l'animation.
Duree La durée de l'animation (en millisecondes).

Valeur de retour

Aucune.

Remarques

Pour ce mesh, un squelette doit être créé avec CreateSkeleton().
La nouvelle animation est vide et les étapes doivent être créées avec AddSkeletonAnimationKeyFrame().

AddSkeletonAnimationKeyFrame(), CreateSkeleton()
