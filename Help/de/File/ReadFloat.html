ReadFloat()

Syntax

Zahl.f = ReadFloat(#Datei)
Beschreibung
Liest einen Fließkomma-Wert (4 Bytes) aus einer Datei ein, beginnend an der aktuellen Dateiposition.

Parameter

#Datei Die Datei, aus der gelesen werden soll.

Rückgabewert

Gibt den gelesenen Fließkomma-Wert zurück, oder Null bei einem Fehler.

Anmerkungen

Für ein Beispiel, wie aus einer Datei gelesen wird, siehe den ReadFile() Befehl - mit ReadFloat() wird einfach ein Fließkomma-Wert anstelle einer kompletten Zeile (String) gelesen.
