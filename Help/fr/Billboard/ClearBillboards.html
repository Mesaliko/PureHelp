ClearBillboards()

Syntaxe

ClearBillboards(#BillboardGroup)
Description
Détruit tous les billboards contenus dans un groupe de billboards.

Arguments

#BillboardGroup Numéro du groupe de billboards.

Valeur de retour

Aucune.
