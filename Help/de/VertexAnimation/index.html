PureBasic - VertexAnimation

Einleitung

Vertex-Animation ermöglicht die Steuerung der Animations-Posen (auch Haltung, Stellung) von Meshes. Das Mesh muss eine Liste vordefinierter Posen enthalten.

Befehlsübersicht

AddVertexPoseReference
CreateVertexAnimation
CreateVertexPoseKeyFrame
CreateVertexTrack
MeshPoseCount
MeshPoseName
UpdateVertexPoseReference
VertexPoseReferenceCount
