MeshRadius()

Syntaxe

Resultat = MeshRadius(#Mesh)
Description
Renvoie le rayon de la plus petite sphère capable de contenir le mesh.

Arguments

Mesh Le mesh à utiliser.

Valeur de retour

Renvoie le rayon du mesh.
