ShowLibraryViewer()

Syntax

ShowLibraryViewer([Bibliothek$ [, #Objekt]])
Beschreibung
Öffnet das Library-Anzeiger-Fenster oder bringt es in den Vordergrund, wenn es bereits offen ist. Wenn 'Bibliothek$' angegeben wird, dann wird der Anzeiger die Objekte dieser Bibliothek (Library) anzeigen. Wenn eine '#Objekt'-Nummer angegeben wird, dann wird der Anzeiger das angegebene Objekt dieser Bibliothek anzeigen.

Parameter

Bibliothek$ (optional) Die Bibliothek, welche im Anzeiger dargestellt werden soll. Wenn dieser Parameter nicht angegeben wird, dann wird der Anzeiger geöffnet, ohne eine bestimmte Bibliothek anzuzeigen.
#Objekt (optional) Das Objekt der Bibliothek, welches im Anzeiger dargestellt werden soll. Wenn dieser Parameter nicht angegeben wird, dann wird kein Objekt angezeigt.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.

Beispiel

  If CreateImage(0, 200, 200) And StartDrawing(ImageOutput(0))
    DrawingMode(#PB_2DDrawing_Transparent)
    Box(0, 0, 200, 200, RGB(255, 255, 255))
    For i = 1 To 30
      DrawText(Random(200), Random(200), "Hello World!", RGB(Random(255), Random(255), Random(255)))
    Next i
    StopDrawing() 
    
    ShowLibraryViewer("Image", 0)  ; das Bild anzeigen
    CallDebugger     ; das Programm anhalten, damit es nicht sofort beendet wird
  EndIf
