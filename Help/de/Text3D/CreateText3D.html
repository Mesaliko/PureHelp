CreateText3D()

Syntax

Ergebnis = CreateText3D(#Text3D, Beschriftung$ [, Font$, Höhe, Farbe])
Beschreibung
Erstellt einen neuen 3D-Text. Um angezeigt zu werden, muss der Text mit einem Node oder einem Entity verknüpft werden.

Parameter

#Text3D Die Nummer zum Identifizieren des neuen 3D-Textes. #PB_Any kann verwendet werden, um diese Zahl automatisch zu generieren.
Beschriftung$ Die anzuzeigende Text-Beschriftung (englisch "Caption"). Sie kann mittels Text3DCaption() geändert werden.
Font$ (optional) Der zu verwendende Zeichensatz-Name. Der Zeichensatz muss im Add3DArchive() Pfad vorhanden sein, und muss wie in der 'proper-definitions.fontdef' Datei definiert werden.
Höhe (optional) Die zu verwendende Zeichensatz-Höhe.
Farbe (optional) Die zu verwendende RGBA-Farbe. Um eine gültige Farbe zu erhalten, verwenden Sie RGBA(). Die Farbe kann mittels Text3DColor() geändert werden.

Rückgabewert

Gibt einen Wert ungleich Null zurück, wenn der Text erfolgreich erstellt wurde, und Null bei einem Fehler. Wenn #PB_Any als #Text3D Parameter verwendet wurde, dann wird bei Erfolg die neu generierte Nummer zurückgegeben.
