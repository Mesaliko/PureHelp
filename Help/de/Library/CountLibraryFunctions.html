CountLibraryFunctions()

Syntax

Ergebnis = CountLibraryFunctions(#Library)
Beschreibung
Ermittelt die Anzahl der in der Library verfügbaren Funktionen. Die Library muss geöffnet sein, wenn dieser Befehl aufgerufen wird.

Parameter

#Library Die Nummer der Library, welche die zu zählenden Funktionen enthält. Diese Nummer muss identisch zu der vorher mit OpenLibrary() verwendeten sein.

Rückgabewert

Gibt die Anzahl der in der Library verfügbaren Funktionen zurück.

Unterstützte OS

Windows, Linux
