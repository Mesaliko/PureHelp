GetRuntimeInteger()

Syntax

Ergebnis = GetRuntimeInteger(Objekt$)
Beschreibung
Gibt den Integer-Wert des Runtime-Objekts zurück. Ist das Runtime-Objekt eine Prozedur, wird die Prozedur-Adresse zurückgegeben.

Parameter

Objekt$ Name des Objekts, von dem der Wert ermittelt werden soll. Die folgenden Objekte werden unterstützt:
- Variable: der Objektname ist 'VariablenName' (ohne Beachtung von Groß- und Kleinschreibung).
- Konstante: der Objektname ist '#KonstantenName' (ohne Beachtung von Groß- und Kleinschreibung).
- Prozedur: der Objektname ist 'ProzedurName()' (ohne Beachtung von Groß- und Kleinschreibung).

Beim Zugriff auf öffentliche Module-Elemente ist der Modulpräfixname obligatorisch, auch wenn UseModule verwendet wird.

Rückgabewert

Gibt den Integer-Wert des angegebenen Objekts zurück oder Null, falls es nicht gefunden wurde. Da Null ein gültiger Integer-Wert ist, kann IsRuntime() verwendet werden, um festzustellen, ob das Runtime-Objekt tatsächlich existiert. Ist die Variable vom Typ Float oder Double, wird sie automatisch in Integer umgewandelt. Handelt es sich bei dem Runtime-Objekt um eine Prozedur, wird die Prozeduradresse zurückgegeben.
