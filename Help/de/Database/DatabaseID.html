DatabaseID()

Syntax

DatenbankID = DatabaseID(#Datenbank)
Beschreibung
Gibt die einmalige ID zurück, welche die angegebene '#Datenbank' im Betriebssystem identifiziert. Diese Funktion ist nützlich, wenn eine andere Bibliothek einen Datenbank-Bezug benötigt.

Parameter

#Datenbank Die zu verwendende Datenbank.

Rückgabewert

Gibt die ID für diese Datenbank-Verbindung zurück.

Unterstützte OS

Alle

<- DatabaseError() - Database Inhaltsverzeichnis - DatabaseQuery() ->
