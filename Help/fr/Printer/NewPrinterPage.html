NewPrinterPage()

Syntaxe

NewPrinterPage()
Description
Créé une nouvelle page vide.

Arguments

Aucun.

Valeur de retour

Aucune.

Remarques

La page précédente est envoyée à l'imprimante et ne peut plus être modifiée.

Cette commande doit être appelée à l'intérieur d'un bloc StartDrawing()/StopDrawing().

Exemple

  If PrintRequester()
    If StartPrinting("Deux feuilles")
      If StartDrawing(PrinterOutput())
        DrawingMode(#PB_2DDrawing_Transparent)
        DrawText(10, 10, "Première page !", RGB(0, 0, 0))
      
        ; Informe l'imprimante qu'elle doit commencer une nouvelle page
        NewPrinterPage()
        
        DrawText(10, 10, "Deuxième page !", RGB(0, 0, 0))
      
        StopDrawing()
      EndIf
      
      StopPrinting()
    EndIf
  EndIf
