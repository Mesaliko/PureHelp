Valeur de retour

Renvoie une valeur non nulle si la touche a été appuyée puis relâchée, zéro sinon.

Exemple

  If InitSprite() And InitKeyboard() And OpenScreen(800,600,16,"")
    Paused = #False
    Repeat
      FlipBuffers()
      
      If StartDrawing(ScreenOutput())
        
        ExamineKeyboard()
        If KeyboardReleased(#PB_Key_P)
          If Paused = #False
            Paused = #True
          Else
            Paused = #False
          EndIf
        EndIf
        
        DrawingMode(0)

        If Paused = #False
          DrawText(20, 20, "Le programme tourne...        ")
        Else
          DrawText(20, 20, "Le programme est à l'arrêt...")
        EndIf

        StopDrawing()
      EndIf
    Until KeyboardPushed(#PB_Key_Escape)
  EndIf
