OpenGadgetList3D()

Syntax

OpenGadgetList3D(#Gadget3D [, Eintrag])
Beschreibung
Verwendet das angegebene 3D-Gadget als Gadgetliste, um darauf dynamisch neue Gadgets hinzufügen. Sobald alle notwendigen Änderungen vorgenommen wurden, sollte CloseGadgetList3D() aufgerufen werden.

Parameter

#Gadget3D Das zu verwendende 3D-Gadget.
Eintrag (optional) Der zu verwendende 3D-Gadget-Eintrag. Dies wird nur beim PanelGadget3D() unterstützt.

Anmerkungen

Diese Funktion wird von den folgenden 3D-Gadgets unterstützt:

- ContainerGadget3D()
- PanelGadget3D() : Der optionale 'Eintrag' Parameter muss angegeben werden.
- ScrollAreaGadget3D()

Unterstützte OS

Alle

<- ListViewGadget3D() - Gadget3D Inhaltsverzeichnis - OptionGadget3D() ->
