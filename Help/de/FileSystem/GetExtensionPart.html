GetExtensionPart()

Syntax

Endung$ = GetExtensionPart(DateiPfad$)
Beschreibung
Ermittelt die Dateiendung aus einem kompletten Dateipfad.

Parameter

DateiPfad$ Der volle Dateipfad, von dem die Endung ermittelt werden soll.

Rückgabewert

Gibt die Dateiendung zurück. Zum Beispiel: wenn der volle Pfad "C:\PureBasic\PB.exe" lautet, wird das Resultat "exe" sein.

Anmerkungen

Um den Dateinamen oder den Pfad aus einem kompletten Dateipfad zu ermitteln, verwenden Sie die GetFilePart() und GetPathPart() Befehle.
Wenn der Pfad keinen Dateinamen enthält, kann das Ergebnis zufällig sein. Zum Beispiel sind GetExtensionPart(".htaccess") oder GetExtensionPart(".access.") ungültig.
