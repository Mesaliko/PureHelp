WebViewProxy()

Syntaxe

WebViewProxy(URL$, Port [NomUtilisateur$, MotDePasse$])
Description
Configure un proxy pour tous les futurs WebViewGadget() créés.

Arguments

URL$ L'URL du serveur proxy.
Port Le port du serveur proxy.
NomUtilisateur$, MotDePasse$ (optionnel) Si le serveur proxy nécessite une authentification, un nom d'utilisateur et un mot de passe peuvent être définis ici.

Valeur de retour

Aucune.

Remarques

Tous les liens externes et URL passeront par ce proxy avant d'atteindre la destination. Cela peut être utile dans un environnement d'entreprise où le niveau de sécurité est élevé.

Lorsqu'il est utilisé sur MacOS, la configuration minimale requise est MacOS 14 (Sonoma).
