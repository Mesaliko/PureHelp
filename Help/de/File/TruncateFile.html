TruncateFile()

Syntax

TruncateFile(#Datei)
Beschreibung
Schneidet die Datei an der aktuellen Schreib-/Lese-Position ab und löscht alle folgenden Daten.

Parameter

#Datei Die zu verwendende Datei.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.

Anmerkungen

Diese Funktion kann verwendet werden, um eine Datei kürzer zu machen, ohne diese vollständig neu erstellen zu müssen. Um eine Datei länger zu machen, fügen Sie einfach weitere Daten mit den Write-Befehlen dieser Bibliothek hinzu.
