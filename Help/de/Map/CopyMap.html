CopyMap()

Syntax

Ergebnis = CopyMap(QuellMap(), ZielMap())
Beschreibung
Kopiert alle Elemente der QuellMap in die ZielMap.

Parameter

QuellMap() Die Map, aus der die Elemente kopiert werden.
ZielMap() Die Map, in welche die Elemente kopiert werden. Die sich vor dem Kopieren in dieser Map befindlichen Elemente werden gelöscht. Nach einem erfolgreichen Kopiervorgang sind die beiden Maps identisch.

Rückgabewert

Gibt bei Erfolg einen Wert ungleich Null zurück, und Null bei einem Fehler. Wenn die beiden Maps nicht den gleichen Typ (nativ oder strukturiert) haben, dann wird das Kopieren fehlschlagen.

Beispiel

  NewMap Age()
  NewMap AgeCopy()
  
  Age("John")  = 15
  Age("Elise") = 30
  
  CopyMap(Age(), AgeCopy())
  
  Debug AgeCopy("John")
  Debug AgeCopy("Elise")
