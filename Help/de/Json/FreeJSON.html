FreeJSON()

Syntax

FreeJSON(#JSON)
Beschreibung
Gibt die JSON-Daten und ihren beinhalteten Werte frei.

Parameter

#JSON Die freizugebenden JSON-Daten. Wird hier #PB_All angegeben, dann werden alle verbleibenden JSON-Objekte freigegeben.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.

Anmerkungen

Alle verbliebenen JSON-Objekte werden automatisch freigegeben, wenn das Programm endet.
