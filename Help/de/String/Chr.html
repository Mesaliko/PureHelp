Chr()

Syntax

Text$ = Chr(ZeichenWert)
Beschreibung
Gibt einen String zurück, der aus dem angegebenen Zeichenwert erstellt wurde.

Parameter

ZeichenWert Der Unicode-Zeichenwert. Dies muss ein Wert zwischen 0 und $D7FF oder zwischen $E000 und $FFFF sein. Der Bereich zwischen $D800 und $DFFF ist für Ersatzzeichen reserviert.

Rückgabewert

Gibt einen String zurück, der aus dem angegebenen Zeichenwert erstellt wurde. Ist der Wert ungültig, wird ein leerer String zurückgegeben.

Beispiel

  Debug Chr(33) ; wird "!" ausgeben

Anmerkungen

Eine Tabelle mit allen ASCII-Werten und ihren zugehörigen Zeichen finden Sie hier.

Dieser Befehl arbeitet mit Unicode und gibt dann die zum angegebenen Wert gehörenden Zeichen zurück. 
  Debug Chr(8364) ; wird "€" ausgeben
  Debug Chr($BC)  ; wird "¼" ausgeben
  Debug Chr($BD)  ; wird "½" ausgeben

Beispiel: Spezielle Symbole (Icons) und eine Schleife mit Ausgabe lateinischer Buchstaben

  Debug Chr($2630) ; ☰
  Debug Chr($25BC) ; ▼
  For i = 65 To 90
      Debug Chr(i)
  Next
