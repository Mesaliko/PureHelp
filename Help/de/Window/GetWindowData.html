GetWindowData()

Syntax

Ergebnis = GetWindowData(#Window)
Beschreibung
Gibt den 'Daten' Wert zurück, der mittels der SetWindowData() Funktion für dieses Fenster gespeichert wurde. Dies ermöglicht es, jedem Fenster einen eigenen Wert zuzuordnen.

Parameter

#Window Das zu verwendende Fenster.

Rückgabewert

Gibt den aktuellen Daten-Wert zurück. Wurde für dieses Fenster niemals ein Daten-Wert gespeichert, dann wird der Rückgabewert 0 lauten.
