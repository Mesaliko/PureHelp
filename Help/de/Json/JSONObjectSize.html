JSONObjectSize()

Syntax

Ergebnis = JSONObjectSize(JSONWert)
Beschreibung
Gibt die Anzahl Elemente in einem JSON-Wert des Typs #PB_JSON_Object zurück.

Parameter

JSONWert Der JSON-Wert. Der Wert muss vom Typ #PB_JSON_Object sein.

Rückgabewert

Die Anzahl Elemente (Mitglieder) im JSON-Objekt.

Beispiel

  Input$ = "{ " + Chr(34) + "x" + Chr(34) + ": 10, " + 
                  Chr(34) + "y" + Chr(34) + ": 20, " + 
                  Chr(34) + "z" + Chr(34) + ": 30 }"
  
  ParseJSON(0, Input$)
  Debug JSONObjectSize(JSONValue(0))
