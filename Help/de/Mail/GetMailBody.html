GetMailBody()

Syntax

Ergebnis$ = GetMailBody(#Mail)
Beschreibung
Gibt den angegebenen Email-Inhalt (auch "Email-Körper") zurück, welcher zuvor mit SetMailBody() festgelegt wurde.

Parameter

#Mail Die zu verwendende Email.

Rückgabewert

Gibt den Inhalt (Körper) als einen String zurück.

Beispiel

  If CreateMail(0, "test@purebasic.com", "Hello")
    SetMailBody(0, "Dies ist der Hauptteil")
    Debug GetMailBody(0) ; wird "Dies ist der Hauptteil" ausgeben
  EndIf
