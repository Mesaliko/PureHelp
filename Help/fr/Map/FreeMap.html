FreeMap()

Syntaxe

FreeMap(Map())
Description
Détruit une Map et libère toutes les ressources associées.

Arguments

Map() La map à utiliser.

Valeur de retour

Aucune.

Remarques

Pour accéder à cette map à nouveau, NewMap doit être appelé.
