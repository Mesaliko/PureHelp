PureBasic - Cipher

Einleitung

Die 'Cipher' ("Verschlüsselungs"-) Bibliothek ist ein Set an Funktionen, die nützlich für das Kodieren sowie Ver- und Entschlüsseln von Daten sind. Zum Beispiel ist SHA-2 eine sehr populäre "Fingerabdruck" Routine, welche wegen ihres hohen Schutzes gegen Attacken auf vielen Gebieten zum Einsatz kommt.
