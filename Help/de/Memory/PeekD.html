PeekD()

Syntax

Wert.d = PeekD(*Speicherpuffer)
Beschreibung
Liest einen Double-Wert (8 Bytes) von der angegebenen Speicheradresse.

Parameter

*Speicherpuffer Die Adresse, von der gelesen werden soll.

Rückgabewert

Gibt den Wert des Double-Werts zurück.
