ConsoleLocate()

Syntax

ConsoleLocate(x, y)
Beschreibung
Bewegt den Cursor an die angegebene Position, in Zeichen-Koordinaten. Jeder von Ihnen nach diesem Befehl ausgegebene Text beginnt an den angegebenen Koordinaten.

Parameter

x Die horizontale Position in der Konsole (beginnend bei 0), zu der der Cursor bewegt werden soll.
y Die vertikale Position in der Konsole (beginnend bei 0), zu der der Cursor bewegt werden soll.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.

Anmerkungen

Die Konsole muss sich im Grafikmodus befinden, siehe EnableGraphicalConsole().

Beispiel

  If OpenConsole()
    EnableGraphicalConsole(1)
    
    For i = 0 To 200
      ConsoleLocate(Random(79), Random(24))
      Print("*")
    Next
    
    ConsoleLocate(30, 10)
    PrintN("Druecken Sie Return zum Beenden.")
    Input()
  EndIf
