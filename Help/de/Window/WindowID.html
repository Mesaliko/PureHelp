WindowID()

Syntax

WindowID = WindowID(#Window)
Beschreibung
Ermittelt die einmalige ID des Fensters im System (englisch "system identifier").

Parameter

#Window Das zu verwendende Fenster.

Rückgabewert

Die System-ID des Fensters. Dieses Ergebnis ist manchmal auch als 'Handle' bekannt. Schauen Sie in das Extra-Kapitel Handles und Nummern für weitere Informationen.

Unterstützte OS

Alle

<- WindowHeight() - Window Inhaltsverzeichnis - WindowMouseX() ->
