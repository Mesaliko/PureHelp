SetTerrainTileHeightAtPoint()

Syntax

SetTerrainTileHeightAtPoint(#Terrain, TileX, TileY, x, y, Höhe)
Beschreibung
Legt die Höhe des Terrain-Tile (Kachel im Gelände) an der angegebenen Position fest. Diese Änderung wird nicht sofort sichtbar - UpdateTerrain() muss aufgerufen werden, sobald alle Änderungen durchgeführt wurden.

Parameter

#Terrain Das zu verwendende Terrain.
TileX Der 'x' Index des Tile (Kachels).
TileY Der 'y' Index des Tile (Kachels).
x Die 'x' Position im Tile (in Pixel), um die Terrain-Höhe festzulegen.
y Die 'y' Position im Tile (in Pixel), um die Terrain-Höhe festzulegen.
Height Die neue Terrain-Höhe (in Welt-Einheiten) an der angegebenen Position im Tile.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.
