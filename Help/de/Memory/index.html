PureBasic - Memory

Einleitung

Manchmal ist es sehr nützlich, direkten bzw. "rohen" ("raw") Zugriff auf den Systemspeicher (RAM) zu haben, um einige zeitintensive Operationen durchzuführen und sie zu beschleunigen. Diese Bibliothek ermöglicht das Reservieren einer beliebigen Anzahl an Speicherbereichen (Memory-Puffer) und deren direkte Benutzung in PureBasic.

Hinweis: Die direkte Speicher-Manipulation muss vorsichtig erfolgen. Der Zugriff auf Speicherbereiche außerhalb der reservierten Speicherpuffer wird das Programm zum Absturz bringen.
