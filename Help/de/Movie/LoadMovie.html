LoadMovie()

Syntax

Ergebnis = LoadMovie(#Movie, Dateiname$)
Beschreibung
Lädt das angegebene Movie (oder anderes Medium) und bereitet für das Abspielen vor.

Parameter

#Movie Die Nummer, mit welcher das geladene Movie identifiziert wird. #PB_Any kann verwendet werden, um diese Nummer automatisch zu generieren.
Dateiname$ Der Dateiname des Movies.

Rückgabewert

Gibt einen Wert ungleich Null zurück, wenn das Movie korrekt geladen wurde, und Null wenn das Laden des Movies fehlgeschlagen ist (Format nicht unterstützt oder Datei nicht gefunden). Wenn #PB_Any als #Movie Parameter verwendet wurde, dann wird die generierte Nummer zurückgegeben.

Anmerkungen

InitMovie() muss einmalig aufgerufen werden, bevor Movies geladen werden. Weitere Informationen über das geladene Movie können mit den MovieInfo(), MovieLength(), MovieWidth() und MovieHeight() Befehlen gelesen werden.
