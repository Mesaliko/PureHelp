UseDialogOpenGLGadget()

Syntax

UseDialogOpenGLGadget()
Description
Enable OpenGLGadget() support in the dialog library. This is not enabled by default to reduce the dialog library size footprint if OpenGLGadget() is not needed.

Return value

None.

See Also

OpenXMLDialog()

Supported OS

All
