HideEntity()

Syntax

HideEntity(#Entity, Status)
Beschreibung
Versteckt oder zeigt das angegebene Objekt '#Entity'.

'Status' kann einen der folgenden Werte annehmen: 
  1: das #Entity wird versteckt
  0: das #Entity wird angezeigt
