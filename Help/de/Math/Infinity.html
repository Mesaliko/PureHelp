Infinity()

Syntax

Ergebnis.f(.d) = Infinity()
Beschreibung
Gibt die spezielle Fließkommazahl, welche positive Unendlichkeit repräsentiert, zurück. Negative Unendlichkeit kann mittels "-Infinity()" berechnet werden.

Parameter

Diese Funktion hat keine Parameter.

Rückgabewert

Gibt den Wert zurück, welcher Unendlichkeit repräsentiert. Das Ergebnis ist ein Float- oder Double-Wert, was davon abhängig ist, ob es einer Float oder Double Variable zugewiesen wird.

Anmerkungen

Unendlichkeit und negative Unendlichkeit sind spezielle Werte. Sie verhalten sich in Berechnungen auf eine Weise, die Sie allgemein erwarten würden. Zum Beispiel beim Teilen von Unendlichkeit durch jede positive Zahl (außer 0 bzw. Unendlichkeit) wird erneut in Unendlichkeit resultieren. Die IsInfinity() Funktion kann verwendet werden, um zu überprüfen, ob ein Wert positive oder negative Unendlichkeit repräsentiert.

Beispiel

  Debug IsInfinity(Infinity() / 1000) ; gibt 1.0 aus
