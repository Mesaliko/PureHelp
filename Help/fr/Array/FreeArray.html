FreeArray()

Syntaxe

FreeArray(Tableau())
Description
Détruit un tableau et libère toutes les ressources associées.

Arguments

Tableau() Le tableau à libérer.

Valeur de retour

Aucune.

Remarques

Pour continuer à l'utiliser, Dim doit être appelé.
