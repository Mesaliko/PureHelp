ShowMemoryViewer()

Syntaxe

ShowMemoryViewer([*AdresseMemoire, Longueur])
Description
Ouvre le visualisateur de mémoire ou le ramène au premier plan si il était déjà ouvert.

Arguments

*AdresseMemoire, Longueur (optionnel) Si une zone mémoire (Buffer) représentée par *AdresseMemoire' de longueur 'Longueur' est spécifiée alors le contenu de cette zone mémoire sera affiché dans le visualisateur de mémoire.

Valeur de retour

Aucune.

Exemple

  *Memoire = AllocateMemory(1000)
  If *Memoire
    RandomData(*Memoire, 1000)       ; Remplir la mémoire avec des données
  
    ShowMemoryViewer(*Memoire, 1000) ; Ouvrir le visualisateur de mémoire
    CallDebugger                     ; Stoppe le programme qui ne se termine pas correctement
  EndIf
