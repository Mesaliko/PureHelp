PauseAudioCD()

Syntaxe

PauseAudioCD()
Description
Met sur pause la lecture d'un CD-Audio.

Arguments

Aucun.

Valeur de retour

Aucune.

Remarques

La lecture peut être reprise en appelant la fonction ResumeAudioCD().
