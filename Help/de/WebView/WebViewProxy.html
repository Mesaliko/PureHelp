WebViewProxy()

Syntax

WebViewProxy(URL$, Port [Benutzername$, Passwort$])
Beschreibung
Richtet einen Proxy für alle künftig erstellten WebViewGadget() ein. Das bedeutet, dass alle externen Links und URLs diesen Proxy durchlaufen, bevor sie das Ziel erreichen. Dies kann in einer Unternehmensumgebung mit hohem Sicherheitsniveau nützlich sein.

Parameter

URL$ Die URL des Proxy-Servers.
Port Der Port des Proxy-Servers.
Benutzername$, Passwort$ (optional) Wenn der Proxy-Server eine Authentifizierung erfordert, können ein Benutzername und ein Passwort festgelegt werden.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.

Anmerkungen

Bei der Nutzung unter MacOS ist mindestens MacOS 14 (Sonoma) erforderlich.
