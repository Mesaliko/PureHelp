JSONErrorPosition()

Syntax

Ergebnis = JSONErrorPosition()
Beschreibung
Gibt die Zeichenposition innerhalb der Zeile bei der letzten bei ParseJSON(), CatchJSON() oder LoadJSON() fehlgeschlagenen JSON-Parsing-Operation.

Parameter

Diese Funktion hat keine Parameter.

Rückgabewert

Die Zeichenposition (ausgehend von 1) des letzten JSON-Parser-Fehlers innerhalb der von JSONErrorLine() gemeldeten Zeile.

Siehe auch

JSONErrorLine(), JSONErrorMessage(), ParseJSON(), CatchJSON(), LoadJSON()
