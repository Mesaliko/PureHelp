StickyWindow()

Syntaxe

StickyWindow(#Fenetre, Etat)
Description
Affiche une fenêtre toujours au premier plan, devant tous les autres programmes, même si elle n'est pas active.

Arguments

#Fenetre La fenêtre à utiliser.
Etat Peut prendre l'une des valeurs suivantes: 
  #True : La fenêtre sera toujours au premier plan.
  #False: Le fenêtre ne restera pas au premier plan si elle n'a pas le focus.

Valeur de retour

Aucune.

Exemple

  If OpenWindow(0, 0, 0, 220, 100, "Exemple...", #PB_Window_SystemMenu | #PB_Window_ScreenCentered)
    
   ButtonGadget  (1, 10, 60, 200, 30, "Fenêtre toujours au premier plan")
   
   Repeat
     Event = WaitWindowEvent()
          
     Select Event
     
       Case #PB_Event_Gadget
         Select EventGadget()
           Case 1
             
             StickyWindow(0, 1)
             RunProgram("notepad.exe")
                                                          
           EndSelect
            
     EndSelect
   Until Event = #PB_Event_CloseWindow
 EndIf
