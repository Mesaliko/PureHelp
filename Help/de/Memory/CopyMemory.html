CopyMemory()

Syntax

CopyMemory(*QuellMemoryID, *ZielMemoryID, Größe)
Beschreibung
Kopiert einen Speicherbereich beginnend von der *QuellMemoryID zur *ZielMemoryID.

Parameter

*QuellMemoryID Die Adresse, von der die Bytes kopiert werden sollen.
*ZielMemoryID Die Adresse, wohin die Bytes kopiert werden sollen.
Größe Die Anzahl der zu kopierenden Bytes.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.

Anmerkungen

Die Quell- und Zielpuffer dürfen sich nicht überlappen. Um Speicher an eine Ziel-Adresse zu kopieren, welche sich mit dem Quell-Puffer überschneidet, verwenden Sie MoveMemory().
