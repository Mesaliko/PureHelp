UserName()

Syntaxe

Resultat$ = UserName()
Description
Renvoie le nom de l'utilisateur en cours.

Arguments

Aucun.

Valeur de retour

Renvoie le nom de l'utilisateur actuellement loggé.

Exemple

  Debug "Utilisateur actuellement connecté: " + UserName()
