DisplayTransparentSprite()

Syntax

DisplayTransparentSprite(#Sprite, x, y [, Intensität [, Farbe]])
Beschreibung
Zeigt das #Sprite an der angegebenen Position auf dem aktuellen Bildschirm an. Das Sprite muss mit dem #PB_Sprite_AlphaBlending Flag erstellt worden sein, um diesen Befehl zu verwenden.

Standardmäßig wird die Farbe 0 (schwarz) als transparente Farbe betrachtet (diese Farbe wird nicht angezeigt). Es ist möglich, die transparente Farbe mittels TransparentSpriteColor() zu verändern.

Dieser Befehl ist ge'clipped' (wird automatisch auf den Bildschirm zugeschnitten), daher ist es völlig in Ordnung, das Sprite außerhalb des Bildschirms anzuzeigen.

Parameter

#Sprite Das darzustellende Sprite.
x, y Die Koordinaten (in Pixel) auf dem Bildschirm, an denen das Sprite dargestellt werden soll.
Intensität (optional) Der zum Darstellen des Sprites verwendete Intensitätslevel. Gültige Werte reichen von 0 (vollständig transparent) bis 255 (völlig undurchsichtig). Der Standardwert ist 255.
Farbe (optional) Die eine Farbe, welche für das einfarbige Darstellen des Sprites verwendet wird. Das Sprite wird in nur einer Farbe gerendet. Um eine gültige Farbe zu erhalten, verwenden Sie RGB().

Rückgabewert

Diese Funktion gibt keinen Wert zurück.
