OpenSubMenu()

Syntax

OpenSubMenu(Text$ [, ImageID])
Beschreibung
Erstellt ein leeres Unter-Menü im aktuellen Menü.

Parameter

Text$ Der Text für das Unter-Menü.

Windows:
Im Parameter Text$ können Sie das spezielle '&' Zeichen benutzen, um einen bestimmten Buchstaben zu unterstreichen:
"&Datei" ergibt in Wirklichkeit: Datei
ImageID (optional) Ein optionales Bild (Image), welches nahe des Unter-Menüs angezeigt werden soll. Dieser Parameter hat nur einen Effekt, wenn das aktuelle Menü mittels CreateImageMenu() oder CreatePopupImageMenu() erstellt worden ist. Dieser Wert kann mit der ImageID() Funktion ermittelt werden.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.

Anmerkungen

Es ist nicht möglich, ein OpenSubMenu einfach umzubenennen - außer auf Windows, welches eine Menü-Nummer zurückgibt.
Beispiel:
SubMenu = OpenSubMenu ("New")
SetMenuItemText (0, SubMenu, "Open")
Auf Linux und macOS müssen Sie mit mehreren Menüs "jonglieren" oder es "zerstören" und neu erstellen.

Beispiel

  If OpenWindow(0, 200, 200, 220, 100, "SubMenu Example")
    If CreateMenu(0, WindowID(0))
      MenuTitle("Project") 
        MenuItem(1, "Open")  
        MenuItem(2, "Close")
        MenuBar()
        OpenSubMenu("Recent files")       ; beginne Erstellung des Unter-Menüs
          MenuItem( 3, "C:\Autoexec.bat")
          MenuItem( 4, "D:\Test.txt")
        CloseSubMenu()                    ; beende Unter-Menü
    EndIf
    Repeat : Until WaitWindowEvent() = #PB_Event_CloseWindow
  EndIf
