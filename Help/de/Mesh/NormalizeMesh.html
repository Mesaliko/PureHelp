NormalizeMesh()

Syntax

NormalizeMesh(#Mesh [, SubMesh])
Beschreibung
"Normalisiert" das Mesh oder das Sub-Mesh. Es wird automatisch der Normalen-Vektor für jeden Eckpunkt (Knoten) des angegebenen Meshs oder Sub-Meshs berechnet. Dynamische Meshes (die mit dem #PB_Mesh_Dynamic Flag erstellt wurden), werden nicht unterstützt.

Parameter

Mesh Gibt das zu verwendende Mesh-Objekt an.
SubMesh (optional) Wenn angegeben, wird nur das Sub-Mesh "normalisiert". Der Index des ersten Sub-Mesh lautet 0 (Haupt-Mesh).

Rückgabewert

Diese Funktion gibt keinen Wert zurück.
