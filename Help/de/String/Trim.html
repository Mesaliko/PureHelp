Trim()

Syntax

Ergebnis$ = Trim(String$ [, Zeichen$])
Beschreibung
Entfernt alle angegebenen Zeichen, die sich am Beginn und am Ende eines Strings befinden.

Parameter

String$ Der zu kürzende String.
Zeichen$ (optional) Das für das Kürzen des Strings verwendete Zeichen. Das Standardzeichen ist das Leerzeichen ('Space'). 'Zeichen$' muss ein ein Zeichen langer String sein.

Rückgabewert

Ein neuer String, welcher den originalen String ohne die entfernten Zeichen beinhaltet.

Beispiel

  Debug Trim("   Hello     ")  ; wird "Hello" anzeigen
  Debug Trim("!!Hello!!", "!") ; wird "Hello" anzeigen
