ClearScreen()

Syntax

ClearScreen(Farbe)
Beschreibung
Löscht den gesamten Bildschirm mit der angegebenen Farbe.

Parameter

Farbe Die für das Löschen des Bildschirms zu verwendende Farbe. RGB() kann verwendet werden, um einen gültigen Farbwert zu erhalten. Eine Tabelle gebräuchlicher Farbwerte finden Sie hier.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.

Anmerkungen

ClearScreen() muss stets außerhalb eines StartDrawing() : StopDrawing() Blocks aufgerufen werden.
