EjectAudioCD()

Syntaxe

EjectAudioCD(Etat)
Description
Ouvre ou ferme le tiroir du lecteur de CD.

Arguments

Etat 
   =1 : Le tiroir est ouvert. 
  <>1 : Le tiroir est fermé.

Valeur de retour

Aucune.
