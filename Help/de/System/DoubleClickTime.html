DoubleClickTime()

Syntax

Ergebnis = DoubleClickTime()
Beschreibung
Gibt die System-Einstellung für die Zeit eines Doppelklicks zurück. Wenn zwei Mausklicks innerhalb dieser Zeit stattfinden, werden sie als Doppelklick erkannt.

Parameter

Diese Funktion hat keine Parameter.

Rückgabewert

Gibt die Doppelklick-Zeit in Millisekunden zurück.

Anmerkungen

Diese Funktion kann verwendet werden, um das Maus-Handling auf einem Screen (Bildschirm) oder in einem CanvasGadget() konsistent zu den globalen System-Einstellungen zu machen.
