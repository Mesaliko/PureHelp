HideBillboardGroup()

Syntax

HideBillboardGroup(#BillboardGruppe, Status)
Beschreibung
Ändert die Sichtbarkeit (versteckt oder angezeigt) einer Billboard-Gruppe und aller darin enthaltenen Billboards.

Parameter

#BillboardGruppe Die Nummer der Billboard-Gruppe, welche versteckt oder angezeigt werden soll.
Status Dieser kann folgende Werte annehmen: 
  #True : die Billboard-Gruppe wird versteckt
  #False: die Billboard-Gruppe wird angezeigt

Rückgabewert

Diese Funktion gibt keinen Wert zurück.

Unterstützte OS

Alle

<- FreeBillboardGroup() - Billboard Inhaltsverzeichnis - IsBillboardGroup() ->
