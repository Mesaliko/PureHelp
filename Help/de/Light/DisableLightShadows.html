DisableLightShadows()

Syntax

DisableLightShadows(#Light, Status)
Beschreibung
Deaktiviert oder aktiviert den Schattenwurf des Lichts.

Parameter

#Light Das zu verwendende Licht.
Status Der neue Status des Licht-Schattenwurfs (englisch "light shadow casting"). Wird dieser auf #True gesetzt, wird der Schattenwurf des Lichts ausgeschalten. Wird er auf #False gesetzt, wird er aktiviert.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.

Unterstützte OS

Alle

<- CreateLight() - Light Inhaltsverzeichnis - FreeLight() ->
