IsRuntime()

Syntax

Ergebnis = IsRuntime(Objekt$)
Beschreibung
Überprüft, ob das angegebene Objekt als Runtime deklariert ist.

Parameter

Objekt$ Name des zu überprüfenden Objekts. Die folgenden Objekte werden unterstützt:
- Variable: der Objektname ist 'VariablenName' (ohne Beachtung von Groß- und Kleinschreibung).
- Konstante: der Objektname ist '#KonstantenName' (ohne Beachtung von Groß- und Kleinschreibung).
- Prozedur: der Objektname ist 'ProzedurName()' (ohne Beachtung von Groß- und Kleinschreibung).

Beim Zugriff auf öffentliche Module-Elemente ist der Modulpräfixname obligatorisch, auch wenn UseModule verwendet wird.

Rückgabewert

Gibt einen Wert ungleich Null zurück, wenn das Objekt als Runtime deklariert wurde, andernfalls Null.
