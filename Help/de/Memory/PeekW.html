PeekW()

Syntax

Wert.w = PeekW(*Speicherpuffer)
Beschreibung
Liest einen Word-Wert (2 Bytes) von der angegebenen Speicheradresse.

Parameter

*Speicherpuffer Die Adresse, von der gelesen werden soll.

Rückgabewert

Gibt den Wert des Word-Werts zurück.
