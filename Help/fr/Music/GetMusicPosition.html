GetMusicPosition()

Syntaxe

Resultat = GetMusicPosition(#Music)
Description
Renvoie la position en cours.

Arguments

#Music Le module musical à utiliser.

Valeur de retour

Renvoie la position dans le module musical en cours de lecture.
La première plage commence à 0.
