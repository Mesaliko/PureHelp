FirstDatabaseRow()

Syntax

Ergebnis = FirstDatabaseRow(#Datenbank)
Beschreibung
Gibt eine Information über die erste Zeile der #Datenbank zurück. Das Flag #PB_Database_DynamicCursor muss bei DatabaseQuery() angegeben werden, damit dieser Befehl funktioniert.

Parameter

#Datenbank Die zu verwendende Datenbank.

Rückgabewert

Ist das Ergebnis gleich Null, dann ist keine Datenbank-Zeile (kein Eintrag) verfügbar.

Anmerkungen

Um auf die Informationen innerhalb des Eintrags (bzw. der Zeile) zuzugreifen, können GetDatabaseLong(), GetDatabaseFloat() und GetDatabaseString() verwendet werden.

NextDatabaseRow(), PreviousDatabaseRow(), GetDatabaseLong()
