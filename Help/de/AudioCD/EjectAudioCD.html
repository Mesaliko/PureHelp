EjectAudioCD()

Syntax

EjectAudioCD(Status)
Beschreibung
Öffnet (wirft aus) oder schließt das aktuelle CD-Laufwerk.

Parameter

Status Die auszuführende Aktion. Ist der Status = 1, ist das CD-Laufwerk geöffnet, andernfalls ist es geschlossen.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.

Siehe auch

UseAudioCD()

Unterstützte OS

Windows, Linux
