ExamineDatabaseDrivers()

Syntax

Ergebnis = ExamineDatabaseDrivers()
Beschreibung
Untersucht die auf dem System verfügbaren Datenbank-Treiber.

Parameter

Diese Funktion hat keine Parameter.

Rückgabewert

Wenn ODBC nicht installiert ist oder keine Treiber vorhanden sind, wird 0 zurückgegeben. Sonst kann NextDatabaseDriver() zum Auflisten der Treiber benutzt werden.

Anmerkungen

Dies ist ein spezifischer Befehl für ODBC-Datenbanken.
