AvailableSerialPortOutput()

Syntaxe

Resultat = AvailableSerialPortOutput(#PortSerie)
Description
Renvoie le nombre d'octets restants dans le tampon de sortie.

Arguments

#PortSerie Le port série à utiliser.

Valeur de retour

Renvoie le nombre d'octets restants dans le tampon de sortie.
