ValD()

Syntax

Ergebnis.d = ValD(String$)
Beschreibung
Wandelt einen String in einen Double-Wert (Fließkommazahl doppelter Genauigkeit) um. Der String muss ein Double im dezimalen oder wissenschaftlichen (Exponent) Format sein. Das Parsen (Suchen) nach Zahlen stoppt beim ersten nicht numerischen Zeichen.

Parameter

String$ Der zu konvertierende String.

Rückgabewert

Der Double-Wert des Strings.

Anmerkungen

Strings mit einer enthaltenen Ganzzahl können auch mit Val() und 32 Bit-Fließkommazahlen mit ValF() (mit geringerer Genauigkeit als ValD()) konvertiert werden.

Beispiel

  Debug ValD("10.000024") ; wird 10.000024 ausgeben
  Debug ValD("1.2345e-2")  ; wird 0.012345 ausgeben
