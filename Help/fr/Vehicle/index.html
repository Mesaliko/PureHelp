PureBasic - Vehicle

Généralités

Les véhicules (vehicle) sont habituellement composés d'un seul corps et d'une ou de plusieurs roues (voitures, camions, vélos, etc.). Le nouveau véhicule créé est une simple entité, de sorte que toutes les fonctions concernant les entités peuvent être utilisées pour le manipuler.

InitEngine3D() doit être appelé avec succès avant d'utiliser les fonctions de cette bibliothèque.
