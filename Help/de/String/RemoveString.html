RemoveString()

Syntax

String$ = RemoveString(String$, LöschString$ [, Modus [, StartPosition [, AnzahlVorkommen]]])
Beschreibung
Ermittelt alle Vorkommen des 'LöschString$' im angegebenen 'String$' und entfernt diese.

Parameter

String$ Der zu verwendende String.
LöschString$ Der zu entfernende String.
Modus (optional) Dies kann einer der folgenden Werte sein: 
  #PB_String_CaseSensitive : Suchen/Entfernen berücksichtigt Groß-/Kleinschreibung (a=a). (Standard)
  #PB_String_NoCase        : Suchen/Entfernen unabhängig von Groß-/Kleinschreibung (A=a).
StartPosition (optional) Gibt die Zeichenposition an, bei das Entfernen starten soll. Die Position des ersten Zeichens ist die 1. Wird dieser Parameter weggelassen, wird der gesamte String verwendet.
AnzahlVorkommen (optional) Gibt an, wieviele Strings bis zum Anhalten der Operation entfernt werden sollen. Wird dieser weggelassen, werden alle Strings entfernt.

Rückgabewert

Ein neuer String ohne die entfernten Strings.

Beispiel

  Debug RemoveString("This is Art", "is") ; wird "Th  Art" anzeigen
  Debug RemoveString("This is Art", "is", #PB_String_CaseSensitive, 1, 1) ; wird "Th is Art" anzeigen
