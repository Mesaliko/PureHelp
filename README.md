# PureHelp
PureHelp is a PureBasic help file reader in HTML format for PureBasic.

<img width="1600" height="900" alt="Image" src="https://github.com/user-attachments/assets/70ce25b4-d6e8-4118-a2f1-36936ec2445c" />

PureHelp is a PureBasic help file reader in HTML format.
It's a clone of the help software offered by PureBasic on Windows, but with many additional features, such as the ability to choose the font and especially its size, as well as the ability to choose the icon size.

It works on all Windows systems since Windows XP.
It also works on Linux and MacOS.

It is multilingual, French and English, and is planned for German (translation to be done).
For other languages, you'll need to send me the appropriate language file, or read the Note.txt file.

You can integrate PureHelp into the Tools menu of the PureBasic IDE and assign it a keyboard shortcut.

For updates from one version of PureBasic to another, read the Notes.txt file.
However, I plan to offer one PureHelp per version of PureBasic on this GitHub.

PureHelp is an alternative to the current PureBasic help (while waiting for an official PB software?!)




FRANÇAIS
========
PureHelp est un lecteur de fichier d'aide de PureBasic au format HTML.
C'est un clône du logiciel d'aide proposé par PureBasic sous Windows mais avec beaucoup de fonctionnalité en plus, comme le choix de la police de caractère et surtout de sa taille, ainsi que le choix de la taille des icônes.

Il fonctionne sur tous les Windows depuis WIndows XP.
Il fonctionne aussi sous linux et MacOS.

Il est multilanguage, français et anglais et il est prévu pour l'allemand (traduction à faire).
Pour les autres langgues, il faudra m'envoyer le fichier langue ad-hoc.

Vous pourrez intégrer PureHelp au menu Outils de l'IDE de PureBasic et lui assigner un raccourci clavier.

Pour les mises à jour d'une version de PureBasic vers une autre, lire le fichier Notes.txt.
Toutefois, il est prévu que je propose un PureHelp par version de PureBasic dans ce github.

PureHelp est une alternative à l'aide actuelle de PureBasic (en attendant un logiciel officiel fait maison ?!)
