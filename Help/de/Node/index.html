PureBasic - Node

Einleitung

Nodes ("Knoten" oder auch "Verbindungen") sind Container, welche verwendet werden können, um Objekte wie Entities, Sound, Camera, Billboard und sogar Nodes selbst zusammen zu gruppieren. Sobald ein Objekt zu einem Node hinzugefügt wurde, ist seine Position und Bewegung relativ zur Node-Position. Dies erlaubt einfaches hierarchisches Objekt-Management.

InitEngine3D() muss erfolgreich aufgerufen worden sein, bevor die Node Funktionen verwendet werden.
