CreateVehicle()

Syntaxe

Resultat = CreateVehicle(#Entity)
Description
Crée une nouvelle entité véhicule.

Arguments

#Entity Le numéro d'identification la nouvelle entité.

#PB_Any peut être utilisé pour générer automatiquement ce numéro.

Valeur de retour

Renvoie zéro si l'entité véhicule ne peut pas être créée.

Si #PB_Any est utilisé comme paramètre '#Entity', le nouveau numéro d'entité de véhicule est renvoyé.
