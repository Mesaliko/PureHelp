ReadData()

Syntax

Ergebnis = ReadData(#Datei, *Speicherpuffer, ZuLesendeLänge)
Beschreibung
Liest den Inhalt der Datei in den angegebenen Speicherbereich, beginnend an der aktuellen Dateiposition.

Parameter

#Datei Die Datei, aus der gelesen werden soll.
*Speicherpuffer Die Adresse, an welche die gelesenen Daten geschrieben werden sollen.
ZuLesendeLänge Die Anzahl der einzulesenden Bytes. Die maximale Länge beträgt 2 GByte.

Rückgabewert

Gibt die Anzahl an Bytes zurück, welche tatsächlich aus der Datei gelesen wurden. Sofern ein Fehler auftrat, ist der Rückgabewert gleich Null.

Anmerkungen

Für ein Code-Beispiel siehe die Lof() Funktion.
