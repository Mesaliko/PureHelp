Minute()

Syntaxe

Resultat = Minute(Date.q)
Description
Renvoie les minutes.

Arguments

Date.q La date à utiliser.

Valeur de retour

Renvoie les minutes de la date spécifiée ou -1 en cas d'erreur.
Le résultat est compris entre 0 et 59.

Exemple

  Debug Minute(Date(1970, 1, 1, 11, 3, 45))  ; Le résultat est '3'.
