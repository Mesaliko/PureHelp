AmbientColor()

Syntaxe

AmbientColor(Couleur)
Description
Change la couleur ambiante du monde 3D.

Arguments

Couleur Nouvelle couleur ambiante.

RGB() peut être utilisée pour obtenir une couleur valide.

Valeur de retour

Aucune.
