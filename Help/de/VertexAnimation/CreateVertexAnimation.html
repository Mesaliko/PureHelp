CreateVertexAnimation()

Syntax

Ergebnis = CreateVertexAnimation(#Mesh, Animation$, Länge)
Beschreibung
Erstellt eine neue Vertex-Animation für das angegebene Mesh.

Parameter

#Mesh Das zu verwendende Mesh, um die neue Animation zu erstellen.
Animation$ Der neue Animations-Name. Der Name beachtet die Groß- und Kleinschreibung.
Länge Die Länge der neuen Animation (in Millisekunden).

Rückgabewert

Gibt einen Wert ungleich Null zurück, wenn die Animation erfolgreich erstellt wurde, andernfalls Null.
