SplineX()

Syntaxe

Resultat = SplineX(#Spline)
Description
Renvoie la position 'X' d'une spline dans le monde.

Arguments

#Spline La spline à utiliser.

Valeur de retour

Renvoie la position 'X' de la spline dans le monde après exécution de ComputeSpline().
