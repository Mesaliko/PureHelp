UseAudioCD()

Syntax

UseAudioCD(AudioCDLaufwerk)
Beschreibung
Wählt das Laufwerk aus, auf das sich die folgenden AudioCD-Befehle beziehen. Es ist möglich, mehrere CDs gleichzeitig abzuspielen.

Parameter

AudioCDLaufwerk Das zu verwendende Laufwerk. Das erste Laufwerk hat die Nummer 0. Die Anzahl der im System verfügbaren Laufwerke wird von der InitAudioCD()-Funktion zurückgegeben.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.
