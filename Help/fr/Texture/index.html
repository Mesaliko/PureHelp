PureBasic - Texture

Généralités

Les textures permettent aux objets 3D (Meshs) d'avoir un aspect réaliste.
En effet, sans textures les objets 3D seraient affichés avec une seule couleur.

PureBasic offre la possibilité de créer des textures directement à l'aide des outils 2D de base (bibliothèque 2DDrawing) ou de les charger à partir de fichiers.

InitEngine3D() doit être appelé avec succès avant de pouvoir utiliser les commandes relatives aux textures.
