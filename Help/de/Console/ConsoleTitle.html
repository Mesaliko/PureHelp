ConsoleTitle()

Syntax

ConsoleTitle(Titel$)
Beschreibung
Ändert den Konsolentitel auf den angegebenen 'Titel$'. Der Konsolentitel wird typischerweise in der Titelzeile des Konsolen-Fensters angezeigt (wenn Sie die Konsole in einer grafischen Umgebung wie Ihrem Desktop sehen).

Unter Windows ist der Konsolentitel auch der Text, welcher in der Startleiste und im Taskmanager für Ihre Konsolen-Anwendung angezeigt wird.

Parameter

Titel$ Der neue Titel für die Konsole.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.

Beispiel

  If OpenConsole()
    ConsoleTitle("Das ConsoleTitle Beispielprogramm")
    PrintN("Druecken Sie Return zum Beenden.")
    Input()
  EndIf
