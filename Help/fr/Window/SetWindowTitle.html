SetWindowTitle()

Syntaxe

SetWindowTitle(#Fenetre, NouveauTitre$)
Description
Change le texte actuellement affiché dans la barre de titre d'une fenêtre.

Arguments

#Fenetre La fenêtre à utiliser.
NouveauTitre$ Le nouveau titre à utiliser.

Valeur de retour

Aucune.

Remarques

Le titre de la fenêtre est récupérable avec la commande GetWindowTitle().

Exemple

  If OpenWindow(0, 0, 0, 220, 100, "Exemple...", #PB_Window_SystemMenu | #PB_Window_ScreenCentered)
    
   ButtonGadget  (1, 10, 60, 200, 30, "Changer le titre de la fenêtre")
   
   Repeat
     Event = WaitWindowEvent()
          
     Select Event
     
       Case #PB_Event_Gadget
         Select EventGadget()
           Case 1
             
            SetWindowTitle(0, "Nouveau Titre") 
                                                          
           EndSelect
            
     EndSelect
   Until Event = #PB_Event_CloseWindow
 EndIf
