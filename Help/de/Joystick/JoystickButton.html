JoystickButton()

Syntax

Ergebnis = JoystickButton(#Joystick, Button)
Beschreibung
Gibt den Status des Joystick-Buttons (Knopfes) zurück.

Parameter

#Joystick Der zu verwendende Joystick. Der Index des ersten Joysticks lautet 0. Die Anzahl der verfügbaren Joysticks wird von InitJoystick() zurückgegeben.
Button Der abzufragende Joystick-Knopf. Der Index des ersten Knopfes lautet 1.
Gibt Null zurück, wenn der angegebene Knopf nicht gedrückt ist, andernfalls wird ein Wert ungleich Null zurückgegeben. Eine beliebige Anzahl an Knöpfen kann zur gleichen Zeit gedrückt sein.

Anmerkungen

ExamineJoystick() muss vor der Verwendung dieser Funktion aufgerufen werden, um den momentanen Joystick-Status zu aktualisieren.
