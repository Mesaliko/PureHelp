DisplayPopupMenu()

Syntax

DisplayPopupMenu(#Menü, WindowID [, x, y])
Beschreibung
Stellt ein Popup-Menü an der aktuellen Maus-Position oder der angegebenen Bildschirm-Position dar.

Parameter

#Menü Das anzuzeigende Menü. Es muss zuvor mittels CreatePopupMenu() oder CreatePopupImageMenu() erstellt worden sein.
WindowID Das Fenster, welchem das Popup-Menü zugewiesen werden soll. Dieser Wert kann mit Hilfe der WindowID() Funktion ermittelt werden.
x, y (optional) Die Position (in Bildschirm-Koordinaten), an der das Menü angezeigt werden soll. Dies sind Koordinaten in Pixel - relativ zur linken oberen Ecke des Haupt-Monitors.

Wenn diese Parameter nicht angegeben werden, wird das Menü an der aktuellen Maus-Position dargestellt.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.

Anmerkungen

Das Popup-Menü wird wieder versteckt, wenn der Anwender einen Eintrag auswählt oder irgendwo außerhalb des Popup-Menü-Bereichs klickt.

Für ein Beispiel und ein Vorschaubild siehe die CreatePopupMenu() Hilfe.
