HideGadget3D()

Syntax

HideGadget3D(#Gadget3D, Status)
Beschreibung
Versteckt oder zeigt ein 3D-Gadget.

Parameter

#Gadget3D Das zu verwendende 3D-Gadget.
Status Der neue Status des 3D-Gadgets. Ist der Status = 1, dann wird das Gadget versteckt. Ist der Status = 0, wird es angezeigt.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.
