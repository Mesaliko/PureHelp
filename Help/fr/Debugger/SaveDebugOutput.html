SaveDebugOutput()

Syntaxe

SaveDebugOutput(NomFichier$)
Description
Enregistre le contenu de la fenêtre de déboguage dans un fichier.

Arguments

NomFichier$ Le nom du fichier dans lequel sera enregistré le contenu de la fenêtre du débogueur.

Valeur de retour

Aucune.

Remarques

Une erreur est rapportée si le fichier ne peut pas être enregistré.

Exemple

  For i = 1 To 100
    Debug Random(i)
  Next i
  SaveDebugOutput("C:\log.txt")
