PureBasic - Dialog

Einleitung

Die Dialog Befehlsbibliothek ermöglicht das einfache Erstellen komplexer Benutzeroberflächen (GUI) - basierend auf einer XML-Definition. Sie verfügt über ein automatisches Gadget-Layout, was sehr nützlich beim Erstellen von Oberflächen ist, die auf verschiedenen Betriebssystemen laufen oder mit verschiedenen Zeichensatz-Größen arbeiten sollen.

Die XML-Definition kann Datei-basierend sein, oder mit Hilfe der XML-Bibliothek während der Laufzeit ("on the fly") im Speicher erstellt werden.

Eine Liste aller unterstützten Schlüsselwörter und Attribute dieser Bibliothek finden Sie hier.
