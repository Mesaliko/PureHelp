ErrorTargetAddress()

Syntaxe

Resultat = ErrorTargetAddress()
Description
Après une erreur de type #PB_OnError_InvalidMemory, cette commande renvoie l'adresse mémoire qui a été lue ou écrite quand l'erreur est survenue.

Arguments

Aucun.

Valeur de retour

Après une erreur de type #PB_OnError_InvalidMemory, cette commande renvoie l'adresse mémoire qui a été lue ou écrite quand l'erreur est survenue. Cette commande n'a pas d'utilité pour les autres types d'erreur.

OS Supportés

Tous
