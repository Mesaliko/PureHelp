AudioCDStatus()

Syntaxe

Resultat = AudioCDStatus()
Description
Renvoie l'état actuel du lecteur de CD.

Arguments

Aucun.

Valeur de retour

Peut prendre les valeurs suivantes: 
  -1 : Lecteur de CD non prêt (vide ou avec tiroir ouvert)
   0 : Lecteur de CD arrêté (mais un CD est à l'intérieur et détecté)
  >0 : Plage audio en cours de lecture.
