UseDialogScintillaGadget()

Syntaxe

UseDialogScintillaGadget()
Description
Active la prise en charge de ScintillaGadget() dans la bibliothèque de dialogue.
Ceci n'est pas activé par défaut pour réduire la taille de la bibliothèque de dialogue si ScintillaGadget() n'est pas nécessaire.

Valeur de retour

Aucune.
