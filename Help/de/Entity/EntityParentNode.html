EntityParentNode()

Syntax

Ergebnis = EntityParentNode(#Entity)
Beschreibung
Gibt die übergeordnete ("parent") NodeID() zurück.

Parameter

#Entity Das zu verwendende Entity.

Rückgabewert

Gibt die übergeordnete ("Parent-") NodeID() zurück, sofern vorhanden. Dies kann entweder ein echter Knoten ("Node") sein - oder ein Knochen ("bone"), wenn das Entity einem Knochen ("bone") zugeordnet ist. Wenn das Entity keinen übergeordneten Knoten hat, wird 0 zurückgegeben.

Unterstützte OS

Alle

<- EntityLookAt() - Entity Inhaltsverzeichnis - EntityPitch() ->
