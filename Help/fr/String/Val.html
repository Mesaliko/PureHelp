Val()

Syntaxe

Resultat.q = Val(Chaine$)
Description
Convertit une chaîne en une valeur numérique de type quad.

Arguments

Chaine$ La chaîne de caractères à utiliser.

Valeur de retour

Renvoie une valeur numérique entière.

La chaîne doit représenter un entier au format décimal, hexadécimal (avec le préfixe '$') ou binaire (avec le préfixe '%').
La conversion s'arrêtera au premier caractère non numérique.

Remarques

Les chaînes représentant une valeur flottante 32 bits devraient être converties avec ValF(), et 64 bits avec ValD().

Exemple

  Debug Val("1024102410241024") ; affichera '1024102410241024'.
  Debug Val("$10FFFFFFFF")      ; affichera '73014444031'.
  Debug Val("%1000")            ; affichera '8'.
  Debug Val("1.2345e+3")        ; affichera '1'. Utiliser ValF ou ValD.
