EventDropAction()

Syntax

Ergebnis = EventDropAction()
Beschreibung
Nachdem ein #PB_Event_GadgetDrop oder #PB_Event_WindowDrop Ereignis von WaitWindowEvent() oder WindowEvent() empfangen wurde, gibt diese Funktion zurück, welche Aktion mit diesen Daten stattfinden sollte.

Parameter

Diese Funktion hat keine Parameter.

Rückgabewert

Gibt einen der folgende Werte zurück: 
  #PB_Drag_Copy: Die Daten sollten kopiert werden
  #PB_Drag_Move: Die Daten sollten verschoben werden (die Quelle der Drag & Drop Operation ist verantwortlich für das Entfernen der originalen Daten)
  #PB_Drag_Link: Die Daten sollten verlinkt werden
