HideParticleEmitter()

Syntaxe

HideParticleEmitter(#EmetteurParticule, Etat)
Description
Cache ou affiche un émetteur de particules.

Arguments

#EmetteurParticule L'émetteur de particules à utiliser.
Etat 
  #True : L'émetteur est caché
  #False: L'émetteur est affiché

Valeur de retour

Aucune.
