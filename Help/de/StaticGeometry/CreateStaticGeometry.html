CreateStaticGeometry()

Syntax

Ergebnis = CreateStaticGeometry(#StaticGeometry, Breite, Höhe, Länge, SchattenAktivieren)
Beschreibung
Erstellt eine leere statische Geometrie-Form.

Parameter

#StaticGeometry Die Nummer, durch welche die neue statische Geometrie-Form identifiziert wird. #PB_Any kann verwendet werden, um diese Nummer automatisch zu generieren.
Breite Breite (in Welt-Einheiten, englisch "world units") der statischen Geometrie-Form.
Höhe Höhe (in Welt-Einheiten) der statischen Geometrie-Form.
Länge Länge (in Welt-Einheiten) der statischen Geometrie-Form.
SchattenAktivieren Aktiviert oder deaktiviert das dynamische Werfen von Schatten der statischen Geometrie-Form. Auf #True gesetzt zum Aktivieren, andernfalls auf #False.

Rückgabewert

Gibt einen Wert ungleich Null zurück, wenn die statische Geometrie-Form erfolgreich erstellt wurde, und Null wenn nicht. Sofern #PB_Any als #StaticGeometry Parameter verwendet wurde, dann wird bei Erfolg die generierte Nummer zurückgegeben.

Anmerkungen

Wenn zuvor eine andere statische Geometrie-Form mit der gleichen #StaticGeometry Nummer geladen wurde, dann wird die ältere statische Geometrie-Form beim Erstellen der neuen automatisch freigegeben.
