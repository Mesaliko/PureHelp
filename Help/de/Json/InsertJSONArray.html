InsertJSONArray()

Syntax

InsertJSONArray(JSONWert, Array())
Beschreibung
Fügt das angegebene Array() in den angegebenen JSON-Wert sein. Der JSON-Wert wird auf den Typ #PB_JSON_Array geändert.

Parameter

JSONWert Der JSON-Wert. Der vorherige Inhalt des Werts wird auf den Inhalt des Arrays geändert.
Array() Das in den JSON-Wert einzufügende Array.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.

Anmerkungen

Wenn das angegebene Array() mehr als eine Dimension hat, dann wird der JSON-Wert mit einem verschachtelten Array von Arrays gefüllt, um die mehrdimensionalen Daten wiederzugeben. Siehe das Beispiel unten für weitere Details.

Beispiel

  Dim Colors.s(3)
  Colors(0) = "red"
  Colors(1) = "yellow"
  Colors(2) = "green"
  Colors(3) = "blue"
  
  If CreateJSON(0)
    InsertJSONArray(JSONValue(0), Colors())
    Debug ComposeJSON(0)
  EndIf

Beispiel

  Dim matrix(2, 2)
  matrix(0, 0) = 1
  matrix(1, 1) = 1
  matrix(2, 2) = 1
  
  If CreateJSON(0)
    InsertJSONArray(JSONValue(0), matrix())
    Debug ComposeJSON(0)
  EndIf
