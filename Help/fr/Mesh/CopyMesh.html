CopyMesh()

Syntaxe

Resultat = CopyMesh(#Mesh, #NouveauMesh)
Description
Copie un mesh.

Arguments

#Mesh Le numéro d'identification du mesh à copier.
#NouveauMesh Le numéro d'identification du nouveau mesh. Si #PB_Any est utilisé à la place de '#NouveauMesh' alors le numéro du nouveau mesh sera renvoyé dans 'Resultat'.

Valeur de retour

Renvoie une valeur non nulle en cas de succès, zéro sinon.

Remarques

Si le #NouveauMesh était déjà créé, il est automatiquement supprimé et remplacé par le nouveau.

Les meshes dynamiques créés avec l'option #PB_Mesh_Dynamic, ne sont pas autorisés.
