ParticleAngle()

Syntax

ParticleAngle(#ParticleEmitter, RangeStart.f, RangeEnd.f [, SpeedRangeStart.f, SpeedRangeEnd.f])
Description
Changes the particle angle when emitted.

Parameters

#ParticleEmitter The particle emitter to use.
RangeStart, RangeEnd The particle range (in degree) between 0 and 360. For example a range of 45 to 90 will only emit particle with an random angle between 45 and 90 degrees.
SpeedRangeStart, SpeedRangeEnd Specify the rotation speed range of the emitted particle. A random rotation speed will be picked from this range when a new particle is emitted.

Remarks

To avoid display glitchs, the particle material should be configured to be non-repeatable using SetMaterialAttribute(Material, #PB_Material_TAM, #PB_Material_ClampTAM).

Return value

None.

Supported OS

All
