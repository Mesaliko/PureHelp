DESFingerprint()

Syntaxe

Resultat$ = DESFingerprint(MotDePasse$, Cle$)
Description
Attention
Cette fonction est dépréciée, elle sera peut-être supprimée dans une future version de PureBasic. Elle ne doit pas être utilisée dans du nouveau code.


Renvoie une version cryptée DES du mot de passe.
Cette commande est obsolète car elle n'est plus suffisamment sécurisée pour les utilisations modernes.
Utilisez plutôt la fonction CreatePasswordHash().
