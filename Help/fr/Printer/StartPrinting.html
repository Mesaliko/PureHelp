StartPrinting()

Syntaxe

Resultat = StartPrinting(Impression$)
Description
Initialise l'imprimante et démarre l'impression.

Arguments

Impression$ Le nom qui apparaîtra dans la file d'attente et qui identifie la tâche d'impression.

Valeur de retour

Renvoie une valeur non nulle en cas de succès, zéro sinon.

Exemple

  If PrintRequester()
  
  If StartPrinting("Test")
    
      If StartDrawing(PrinterOutput())
        DrawingMode(#PB_2DDrawing_Transparent)
        DrawText(10, 10, "Première page !", RGB(0, 0, 0))
        DrawingMode(#PB_2DDrawing_Outlined)
        Circle(500,500,500 , RGB(255,0,0))     
        StopDrawing()
      EndIf
      
      StopPrinting()
    EndIf
    
  EndIf
