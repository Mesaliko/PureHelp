OpenHelp()

Syntax

OpenHelp(Dateiname$, Thema$)
Beschreibung
Öffnet ein Hilfe-Fenster und zeigt dieses an.

Parameter

Dateiname$ Der Name der zu öffnenden .chm oder .hlp Datei.
Thema$ Der interne Name der darzustellenden Seite. Wenn der Parameter 'Thema$' kein leerer String ist, stellt die OpenHelp-Funktion direkt die angegebene Seite dar (sehr nützlich für Online/kontext-sensitive Hilfe).

Rückgabewert

Diese Funktion gibt keinen Wert zurück.

Beispiel

  OpenHelp("help.chm", "index.htm")
  OpenHelp("help.chm", "print.txt")
