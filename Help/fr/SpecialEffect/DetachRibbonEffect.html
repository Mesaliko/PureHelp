DetachRibbonEffect()

Syntaxe

DetachRibbonEffect(#Effet, NoeudID)
Description
Détache le ruban du noeud donné.

Arguments

#Effet L'effet à utiliser.

Cet effet a été créé avec CreateRibbonEffect().
NoeudID Le nœud à détacher du ruban.

Si le ruban n'a pas été attaché au nœud, la fonction n'a aucun effet.

Pour obtenir un ID de noeud valide, utiliser NodeID().

Valeur de retour

Aucune.

Voir aussi

CreateRibbonEffect(), AttachRibbonEffect()

OS Supportés

Tous
