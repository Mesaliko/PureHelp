IsLibrary()

Syntax

Ergebnis = IsLibrary(#Library)
Beschreibung
Überprüft, ob die angegebene Library-Nummer gültig und die Library korrekt initialisiert wurde.

Parameter

#Library Die zu überprüfende Library-Nummer.

Rückgabewert

Gibt einen Wert ungleich Null zurück, wenn es sich um eine gültige Library handelt, und Null wenn nicht.

Anmerkungen

Diese Funktion ist "kugelsicher" und kann mit jedem Wert benutzt werden. Dies ist der richtige Weg, um sicher zu stellen, dass eine Library bereit zur Verwendung ist.
