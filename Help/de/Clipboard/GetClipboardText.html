GetClipboardText()

Syntax

Text$ = GetClipboardText()
Beschreibung
Gibt den gerade in der Zwischenablage befindlichen Text-String zurück.

Parameter

Diese Funktion hat keine Parameter.

Rückgabewert

Gibt den Inhalt der Zwischenablage zurück, wenn diese Text enthält. Wenn die Zwischenablage keine Daten oder keine Text-Daten enthält, dann wird ein leerer String zurückgegeben.
