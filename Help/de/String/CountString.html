CountString()

Syntax

Ergebnis = CountString(String$, StringToCount$)
Beschreibung
Gibt die Anzahl zurück, wie oft 'StringToCount$' im 'String$' vorkommt.

Parameter

String$ Der zu verwendende Eingabestring.
StringToCount$ Der String, dessen Häufigkeit im Eingabestring gezählt werden soll.

Rückgabewert

Die Anzahl, wie oft 'StringToCount$' im 'String$' gefunden wurde.

Anmerkungen

Das Zählen erfolgt nicht Wort-basiert, was bedeutet: wenn 'StringToCount$' Teil eines Wortes ist, wird es trotzdem gezählt, wie im folgenden Beispiel dargestellt.

Beispiel

  Debug CountString("How many 'ow' contains Bow ?", "ow") ; wird 3 ausgeben
