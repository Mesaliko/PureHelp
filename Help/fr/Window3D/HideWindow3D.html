HideWindow3D()

Syntaxe

HideWindow3D(#Fenetre3D, Etat)
Description
Affiche ou cache une fenêtre 3D.

Arguments

#Fenetre3D La fenêtre 3D à utiliser.
Etat Peut prendre une des valeurs suivantes: 
  #True : La fenêtre 3D est cachée.
  #False: La fenêtre 3D est rendue visible.

Valeur de retour

Aucune.

OS Supportés

Tous
