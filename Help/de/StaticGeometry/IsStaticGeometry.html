IsStaticGeometry()

Syntax

Ergebnis = IsStaticGeometry(#StaticGeometry)
Beschreibung
Überprüft, ob die angegebene #StaticGeometry eine gültige und korrekt initialisierte statische Geometrie-Form ist.

Parameter

#StaticGeometry Die zu verwendende statische Geometrie-Form.

Rückgabewert

Gibt einen Wert ungleich Null zurück, wenn #StaticGeometry eine gültige statische Geometrie-Form ist, und andernfalls Null.

Anmerkungen

Diese Funktion ist "kugelsicher" und kann mit jedem Wert benutzt werden. Ist das 'Ergebnis' ungleich Null, dann ist das Objekt gültig und initialisiert, andernfalls wird Null zurückgegeben. Dies ist ein guter Weg, um sicher zu stellen, dass eine statische Geometrie-Form bereit zur Verwendung ist.
