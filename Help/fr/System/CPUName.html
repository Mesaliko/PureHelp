CPUName()

Syntaxe

Resultat$ = CPUName()
Description
Renvoie le nom du processeur.

Arguments

Aucun.

Valeur de retour

Renvoie le nom du CPU (la totalité des informations du fournisseur). Il n'y a pas de norme à ce propos, alors cette fonction ne doit pas être utilisée pour identifier le CPU lors de l'exécution, mais peut être utile pour du deboguage par exemple.

Exemple

  Debug CPUName() ; Ex: "Intel(R) Core(TM) i7 CPU 860  @ 2.80GHz"
