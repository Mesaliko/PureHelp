ExamineXMLAttributes()

Syntax

Ergebnis = ExamineXMLAttributes(*Knoten)
Beschreibung
Startet die Untersuchung der Attribute des angegebenen XML-Knotens.

Parameter

*Node Der zu untersuchende XML-Knoten.

Rückgabewert

Ein Wert ungleich Null, wenn der Knoten vom Typ #PB_XML_Normal ist, andernfalls Null (da solche Knoten keine Attribute haben können).
