CopyTexture()

Syntax

Ergebnis = CopyTexture(#Textur, #NeueTextur)
Beschreibung
Erstellt eine neue Textur, welche eine exakte Kopie der angegebenen Textur ist.

Parameter

#Textur Die zu kopierende Textur.
#NeueTextur Eine Nummer zum Identifizieren der neuen Textur. #PB_Any kann verwendet werden, um diese Zahl automatisch zu generieren.

Rückgabewert

Ein Wert ungleich Null, wenn die Textur erfolgreich dupliziert wurde, andernfalls Null. Wenn #PB_Any als #NeueTextur Parameter verwendet wurde, dann wird im Erfolgsfall die generierte Nummer zurückgegeben.

Unterstützte OS

Alle
