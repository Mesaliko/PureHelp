NetworkClientEvent()

Syntax

Ergebnis = NetworkClientEvent(Verbindung)
Beschreibung
Überprüft, ob auf einer mit OpenNetworkConnection() erstellten Netzwerk-Verbindung ein Ereignis (Event) auftrat.

Parameter

Verbindung Die zu überprüfende Verbindung.

Rückgabewert

Gibt einen der folgenden Werte zurück: 
  #PB_NetworkEvent_None      : Nichts ist passiert
  #PB_NetworkEvent_Data      : Roh-Daten wurden empfangen (können mittels ReceiveNetworkData() gelesen werden)
  #PB_NetworkEvent_Disconnect: Der Client wurde getrennt (Verbindung geschlossen).
