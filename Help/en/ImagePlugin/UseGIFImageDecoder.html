UseGIFImageDecoder()

Syntax

UseGIFImageDecoder()
Description
Enables the GIF image support for the CatchImage(), LoadImage(), CatchSprite() and LoadSprite() functions. Only LoadImage() and CatchImage() support multi-frame GIF (it will result in a multi-frame image).

Parameters

None.

Return value

None.

Remarks

All formats are supported, including the progressive and interlaced format.

See Also

LoadImage(), CatchImage(), LoadSprite(), CatchSprite()

Supported OS

All
