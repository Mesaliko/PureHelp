SetClipboardText()

Syntax

SetClipboardText(Text$)
Beschreibung
Speichert einen String in der Zwischenablage. Wenn die Zwischenablage bereits Text enthält, wird dieser überschrieben.

Parameter

Text$ Der String, den Sie in der Zwischenablage speichern möchten.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.
