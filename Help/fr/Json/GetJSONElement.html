GetJSONElement()

Syntaxe

Resultat = GetJSONElement(JSONValeur, Index)
Description
Renvoie l'élément de tableau JSON à l'index donné d'une valeur JSON de type #PB_JSON_Array.

Arguments

JSONValeur La valeur JSON.

La valeur doit être de type #PB_JSON_Array.
Index L'indice de l'élément de tableau à renvoyer.

L'indice doit être compris entre 0 et JSONArraySize() - 1.

Valeur de retour

L'adresse de la valeur JSON à l'index de tableau spécifié.
Si la donnée 'Index' est hors de portée, le résultat est zéro.

Exemple

  ParseJSON(0, "[1, 2, 3, 4, 5]")
    
  For i = 0 To JSONArraySize(JSONValue(0)) - 1
    Debug GetJSONInteger(GetJSONElement(JSONValue(0), i))
  Next i
  
  ; Resultat : 1
  ;            2
  ;            3
  ;            4
  ;            5
