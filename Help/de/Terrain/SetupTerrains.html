SetupTerrains()

Syntax

SetupTerrains(LightID, CompositeMapDistanz.f, Flags)
Beschreibung
Legt die Standard-Parameter für alle Zukunft geschaffenen Terrains (Gelände) fest.

Parameter

LightID Das für das Terrain-Rendering zu verwendende Licht. Um eine gültige Licht-ID zu erhalten, verwenden Sie LightID().
CompositeMapDistanz Der Abstand (in Welt-Einheiten), ab dem - falls vorhanden - mit der Verwendung einer zusammengesetzten Karte (englisch "composite map") begonnen wird.
Flags Kann eine Kombination der folgenden Werte sein: 
  #PB_Terrain_Lightmap:      aktiviert Licht-"Mapping" (Anzeige) für die Terrains (eine Textur mit
                             statischen Schatten).
  #PB_Terrain_NormalMapping: aktiviert das normale Mapping (Anzeige) für die Terrains. Dies ist eine
                             spezielle Textur, welche Relief wie kleine Steine, etc. simuliert.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.
