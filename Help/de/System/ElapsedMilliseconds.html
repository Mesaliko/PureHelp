ElapsedMilliseconds()

Syntax

Ergebnis.q = ElapsedMilliseconds()
Beschreibung
Gibt die Anzahl an Millisekunden zurück, welche seit einer bestimmten Zeit in der Vergangenheit abgelaufen sind.

Parameter

Diese Funktion hat keine Parameter.

Rückgabewert

Gibt die abgelaufene Zeit in Millisekunden zurück.

Anmerkungen

Der zurückgegebene absolute Wert ist nutzlos, da er abhängig vom Betriebssystem variiert. Stattdessen sollte diese Funktion verwendet werden, um Zeitunterschiede zwischen mehreren ElapsedMilliseconds() Aufrufen zu berechnen.

Diese Funktion ist relativ genau: es kann eine leichte Differenz auftreten, je nachdem auf welchem Betriebssystem sie ausgeführt wird, da manche Systeme eine niedrigere Timer-Auflösung als andere haben.

Beispiel

  StartTime.q = ElapsedMilliseconds()     ; ermittelt den aktuellen Wert
  Delay(1000)                             ; wartet 1000 Millisekunden
  Debug ElapsedMilliseconds() - StartTime ; der angezeigte Wert sollte ca. 1000 Millisekunden betragen
