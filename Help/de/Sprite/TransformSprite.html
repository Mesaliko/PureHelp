TransformSprite()

Syntax

TransformSprite(#Sprite, x1, y1, [z1], x2, y2, [z2], x3, y3, [z3], x4, y4, [z4])
Beschreibung
Verformt ("transform") das Sprite auf die neu angegebenen Koordinaten. Dies erfolgt typischerweise beim Durchführen von Echtzeit-Verformungen. Warnung: Da ein Sprite eine Kombination von 2 Dreiecken darstellt), kann die Verformung seltsam aussehen.

Wenn einer der optionalen 'z'-Parameter verwendet wird, müssen alle angegeben werden.

Parameter

#Sprite Das zu verformende Sprite.
x1, y1 Die Koordinaten des ersten Punktes, in Pixel.
x2, y2 Die Koordinaten des zweiten Punktes, in Pixel.
x3, y3 Die Koordinaten des dritten Punktes, in Pixel.
x4, y4 Die Koordinaten des vierten Punktes, in Pixel.
z1, z2, z3, z4 Der 'z'-Wert, welcher die Tiefe der Punkte angibt.

Rückgabewert

Diese Funktion gibt keinen Wert zurück. 
  ;
  ; x1         x2
  ;   ---------
  ;   |      /|
  ;   |    /  |     
  ;   |  /    |
  ;   |/      |
  ;   ---------
  ; x4         x3
  ;
