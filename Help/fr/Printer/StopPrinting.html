StopPrinting()

Syntaxe

StopPrinting()
Description
Arrête toutes les opérations d'impression et envoie les données à l'imprimante.

Arguments

Aucun.

Valeur de retour

Aucune.

Exemple

  If PrintRequester()
  
  If StartPrinting("Test")
    
      If StartDrawing(PrinterOutput())
        DrawingMode(#PB_2DDrawing_Transparent)
        DrawText(10, 10, "Première page !", RGB(0, 0, 0))
        DrawingMode(#PB_2DDrawing_Outlined)
        Circle(500,500,500 , RGB(255,0,0))     
        StopDrawing()
      EndIf
      
      StopPrinting()
    EndIf
    
  EndIf
