FreeMesh()

Syntax

FreeMesh(#Mesh)
Beschreibung
Gibt das angegebene '#Mesh' frei. Sein gesamter zugehöriger Speicherplatz wird freigegeben und dieses Objekt kann nicht mehr benutzt werden.

Parameter

#Mesh Das freizugebende Mesh. Wenn #PB_All angegeben wird, dann werden alle verbliebenen Meshes freigegeben.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.

Anmerkungen

Alle verbleibenden Meshes werden automatisch freigegeben, wenn das Programm endet.

Unterstützte OS

Alle
