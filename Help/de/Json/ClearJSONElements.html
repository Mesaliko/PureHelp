ClearJSONElements()

Syntax

ClearJSONElements(JSONWert)
Beschreibung
Entfernt alle Array-Elemente von einem JSON-Wert des Typs #PB_JSON_Array.

Parameter

JSONWert Der JSON-Wert. Der Wert muss vom Typ #PB_JSON_Array sein.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.

Beispiel

  ParseJSON(0, "[1, 2, 3, 4]")
  
  ; löscht alle Werte und fügt einen neuen String hinzu
  ClearJSONElements(JSONValue(0))
  SetJSONString(AddJSONElement(JSONValue(0)), "Hello")
  
  Debug ComposeJSON(0)
