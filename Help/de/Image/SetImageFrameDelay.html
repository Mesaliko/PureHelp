SetImageFrameDelay()

Syntax

Ergebnis = SetImageFrameDelay(#Image, Delay)
Beschreibung
Legt die Anzeigeverzögerung (in Millisekunden) für das aktuelle "Frame" (Einzelbild) fest. Jedes Frame kann seine eigene Verzögerung haben.

Parameter

#Image Das Bild, dessen Frame-Verzögerung festgelegt werden soll.

Rückgabewert

Legt die Anzeigeverzögerung (in Millisekunden) für das aktuelle Bild-Frame fest.
