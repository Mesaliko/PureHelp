CopyEntity()

Syntax

Ergebnis = CopyEntity(#Entity, #NewEntity)
Beschreibung
Erstellt ein neues Objekt '#NewEntity', welches eine exakte Kopie des angegebenen Objekts '#Entity' ist. Wenn #PB_Any als '#NewEntity' Parameter verwendet wird, dann wird die Nummer des neuen Entity als 'Ergebnis' zurückgegeben.

Ist das 'Ergebnis' gleich 0, ist das Kopieren des Entity fehlgeschlagen.

Wurde das Objekt '#NewEntity' bereits erstellt, wird es automatisch freigegeben und durch das neue ersetzt.

Unterstützte OS

Alle

<- AttachEntityObject() - Entity Inhaltsverzeichnis - CreateEntity() ->
