ResizeWindow()

Syntax

ResizeWindow(#Window, x, y, Breite, Höhe)
Beschreibung
Verschiebt das Fenster und ändert dessen Größe auf die angegebene Position und Größe. Wenn einer der Parameter ignoriert werden kann (weil er nicht verändert werden soll), kann an dessen Stelle #PB_Ignore übergeben werden.

Parameter

#Window Das zu verändernde Fenster.
x, y, Breite, Höhe Die neue Position und Größe des Fensters, in Pixel. #PB_Ignore kann für jeden Parameter (x, y, Breite bzw. Höhe) übergeben werden, und dieser Parameter wird nicht verändert.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.
