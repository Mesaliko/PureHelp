UseMD5Fingerprint()

Syntax

UseMD5Fingerprint()
Beschreibung
Registriert das MD5 (Message Digest 5) Fingerprint Plugin für die weitere Nutzung.

Parameter

Diese Funktion hat keine Parameter.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.

Anmerkungen

Hier eine kurze Erläuterung von RFC 1321 über MD5:

'Der Algorithmus verwendet als Eingabe eine Mitteilung beliebiger Länge und produziert als Ausgabe einen 128-Bit "Fingerprint" (Fingerabdruck) oder "Message Digest" (deutsch in etwa "Nachrichten-Übersicht") der Eingabe. Es wird allgemein angenommen, dass es unmöglich ist: zwei Mitteilungen zu produzieren, die im selben "Message Digest" resultieren oder eine Mitteilung zu erstellen, wo vorher der daraus resultierende "Message Digest" bekannt ist. Der MD5 Algorithmus ist geeignet für Applikationen mit digitaler Unterschrift.'

MD5 Hash-Codes werden häufig für Passwort-Verschlüsselung benutzt, aber es sollte gemieden werden, da sich eine Verletzbarkeit durch verschiedentliche Attaken erwiesen hat. Weitere Informationen finden Sie im RFC 1321: http://www.ietf.org/rfc/rfc1321.txt.

Beispiel

  UseMD5Fingerprint()

  Debug StringFingerprint("yourpassword", #PB_Cipher_MD5)
