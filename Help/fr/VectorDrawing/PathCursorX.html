PathCursorX()

Syntaxe

Resultat.d = PathCursorX()
Description
Renvoie la coordonnée X du curseur.

Arguments

Aucun.

Valeur de retour

La coordonnée X du curseur dans le chemin.

Remarques

Ceci est l'endroit où de nouveaux segments de chemin seront ajoutés ou du texte sera dessiné.
