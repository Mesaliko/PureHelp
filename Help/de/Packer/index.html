PureBasic - Packer

Einleitung

Die Packer-Bibliothek bietet ein effizientes Set an Funktionen zum Packen und Entpacken von Daten ("Komprimieren/Dekomprimieren") und dem Umgang mit Archiven. Mehrere Packer-Formate werden unterstützt - von Zip bis Lzma.
