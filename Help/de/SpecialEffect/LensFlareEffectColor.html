LensFlareEffectColor()

Syntax

LensFlareEffectColor(#Effekt, FarbTyp, Farbe)
Beschreibung
Ändert die Farbe des angegebenen Lensflare-Effektteils.

Parameter

#Effect Der zu verwendende Effekt. Dieser Effekt muss mit CreateLensFlareEffect() erstellt worden sein.
FarbTyp Dies kann einer der folgenden Werte sein: 
  #PB_LensFlare_HaloColor  : ändert die Halo-Farbe
  #PB_LensFlare_CircleColor: ändert die Kreis-Farbe
  #PB_LensFlare_BurstColor : ändert die Burst-Farbe
Farbe Die neue Farbe für den ausgewählten Lensflare-Teil. Eine gültige Farbe kann mit RGB() ermittelt werden.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.
