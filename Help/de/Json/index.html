PureBasic - Json

Einleitung

Die JSON Bibliothek bietet Funktionen zum Analysieren ("parsen"), Erstellen oder Verändern von Daten im JSON-Format. JSON (JavaScript Object Notation) ist ein einfaches Datenaustauschformat, das von vielen Programmiersprachen unterstützt wird. Eine Einführung in das Format finden Sie hier.

Diese Bibliothek versteht und produziert das JSON-Format wie definiert vom RFC-7159.
