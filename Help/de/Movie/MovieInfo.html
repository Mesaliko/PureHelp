MovieInfo()

Syntax

Ergebnis = MovieInfo(#Movie, Flags)
Beschreibung
Gibt zusätzliche Informationen über das Movie zurück.

Parameter

#Movie Das zu verwendende Movie.
Flags Die zurückzugebenden Informationen. Unterstützte Werte sind:
0: gibt die Anzahl der "Frames pro Sekunde" (*1000) zurück.

Rückgabewert

Gibt den im 'Flags' Parameter angegebenen Wert zurück.
