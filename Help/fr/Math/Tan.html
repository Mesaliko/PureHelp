Tan()

Syntaxe

Resultat.f(d) = Tan(Angle.f(d))
Description
Renvoie la tangente d'un angle.

Arguments

Angle.f ou Angle.d L'angle en radians. (float ou double)

Valeur de retour

Renvoie la tangente de l'angle spécifié, exprimé en radian.

Remarques

La fonction réciproque est ATan().

Exemple

  Debug Tan(0.785398) ; Affiche '1.0' (0.99999967320515659)
  Debug Tan(0.0)      ; Affiche '0.0'
