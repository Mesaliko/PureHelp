DatabaseColumns()

Syntax

Ergebnis = DatabaseColumns(#Datenbank)
Beschreibung
Gibt die Anzahl an Spalten (Feldern) von der letzten mittels DatabaseQuery() ausgeführten Datenbank-Abfrage zurück.

Parameter

#Datenbank Die zu verwendende Datenbank.

Rückgabewert

Gibt die Anzahl an Spalten von der letzten Datenbank-Abfrage zurück.
