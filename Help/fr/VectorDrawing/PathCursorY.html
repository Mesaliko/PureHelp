PathCursorY()

Syntaxe

Resultat.d = PathCursorY()
Description
Renvoie la coordonnée Y du curseur.

Arguments

Aucun.

Valeur de retour

La coordonnée Y du curseur dans le chemin.

Remarques

Ceci est l'endroit où de nouveaux segments de chemin seront ajoutés ou du texte sera dessiné.
