GetJSONBoolean()

Syntaxe

Resultat = GetJSONBoolean(JSONValeur)
Description
Renvoie la valeur booléenne d'une valeur JSON de type #PB_JSON_Boolean.

Arguments

JSONValeur La valeur JSON.

La valeur doit être de type #PB_JSON_Boolean.

Valeur de retour

La valeur booléenne #True ou #False.

Remarques

Une valeur JSON peut être réglée sur une valeur booléenne avec SetJSONBoolean().

Exemple

  ParseJSON(0, "[true, true, false]")
  
  Debug GetJSONBoolean(GetJSONElement(JSONValue(0), 0))
  Debug GetJSONBoolean(GetJSONElement(JSONValue(0), 1))
  Debug GetJSONBoolean(GetJSONElement(JSONValue(0), 2))
  
  ; Résultat : 1
  ;            1
  ;            0
