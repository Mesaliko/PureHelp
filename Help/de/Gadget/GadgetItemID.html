GadgetItemID()

Syntax

Ergebnis = GadgetItemID(#Gadget, Eintrag)
Beschreibung
Gibt das OS-Handle für den angegebenen Eintrag in einem Gadget zurück. Dies ist insbesondere nützlich für die Verwendung mit der OS API.

Parameter

#Gadget Das zu verwendende Gadget.
Eintrag Der Eintrag, von dem das Handle zurückgegeben werden soll. Der erste Eintrag im Gadget hat den Index 0.

Rückgabewert

Gibt das OS-Handle für den Eintrag zurück, oder Null bei einem Fehler.

Anmerkungen

Diese Funktion wird gegenwärtig nur vom TreeGadget() und PanelGadget() auf Windows unterstützt. In anderen Fällen gibt sie Null zurück.
