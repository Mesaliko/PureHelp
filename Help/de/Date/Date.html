Date()

Syntax

Datumswert.q = Date([Jahr, Monat, Tag, Stunde, Minute, Sekunde])
Beschreibung
Gibt den aus den angegebenen Parametern erstellten Datumswert oder die lokale Systemzeit zurück, wenn keine Parameter angegeben wurden.

Parameter

Jahr, Monat, Tag, Stunde, Minute, Sekunde (optional) Die zur Erstellung des Datums verwendeten Komponenten. Werden diese Parameter nicht angegeben, wird die lokale Systemzeit zurückgegeben. Der zulässige Bereich für den Jahreswert liegt zwischen 1601 und 9999.

Rückgabewert

Gibt den angegebenen Datumswert zurück. Wenn die angegebenen Parameter ungültig sind (oder außerhalb des unterstützten Datumsbereichs liegen), dann wird -1 zurückgegeben.

Anmerkungen

FormatDate() kann benutzt werden, um den Datumswert in ein für den Anwender lesbares Format umzuwandeln.

Beispiel

  Debug Date() / (3600*24*365)  ; gibt die Anzahl der seit 01.01.1970 vergangenen Jahre aus
  Debug Date(1999, 12, 31, 23, 59, 59)  ; gibt '946684799' aus (zwischen 01.01.1970 0:00:00 Uhr und 31.12.1999 23:59:59 Uhr vergangene Sekunden)
