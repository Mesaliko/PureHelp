PureBasic - Window3D

Einleitung

Die Window3D-Bibliothek ermöglicht die Erstellung komplexer grafischer Benutzeroberflächen (englisch "Graphical User Interfaces" - kurz "GUI's") direkt auf dem Bildschirm (Screen) mithilfe der 3D-Engine. Sie ist hauptsächlich für Spiele oder Anwendungen gedacht, die im Vollbildmodus Benutzereingaben erfordern. Diese Bibliothek basiert auf der regulären PureBasic Window-Bibliothek und bietet eine ähnliche Syntax und ähnliches Verhalten. Die verwendete GUI-Engine ist CEGUI, die einige nützliche Optionen wie "Skinning", hohe Geschwindigkeit und zahlreiche integrierte Gadgets bietet. Weitere Informationen über CEGUI finden Sie hier: http://www.cegui.org.uk.

Diese Bibliothek verwendet die 3D-Engine, daher muss vor dem Verwenden dieser Funktionen InitEngine3D() erfolgreich aufgerufen worden sein.
