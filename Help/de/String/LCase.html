LCase()

Syntax

Ergebnis$ = LCase(String$)
Beschreibung
Gibt den originalen String - konvertiert in Kleinbuchstaben (wenn möglich) - zurück.

Parameter

String$ Der in Kleinbuchstaben zu konvertierende String.

Rückgabewert

Der in Kleinbuchstaben konvertierte String.

Anmerkungen

Diese Funktion unterstützt auch Buchstaben mit Betonung; wenn also ein großes 'É' gefunden wurde, wird dieses in ein kleines 'é' konvertiert.

Beispiel

  Debug LCase("This is Art") ; wird "this is art" anzeigen
