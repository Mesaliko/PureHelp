Red()

Syntax

Ergebnis = Red(Farbe)
Beschreibung
Gibt den Rot-Anteil eines Farbwertes zurück.

Parameter

Farbe Der Farbwert. Dies kann ein 24 Bit RGB oder ein 32 Bit RGBA Wert sein.

Rückgabewert

Gibt den Wert des Rot-Anteils zurück. Das Ergebnis wird zwischen 0 und 255 liegen.

Anmerkungen

Um für die Erstellung einer 24 Bit RGB-Farbe die Rot-, Grün- und Blau-Werte zu kombinieren, verwenden Sie die RGB()-Funktion. Diese Funktionen sind nützlich für die Ausführung von Zeichenoperationen.
