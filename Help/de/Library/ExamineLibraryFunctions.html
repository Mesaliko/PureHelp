ExamineLibraryFunctions()

Syntax

Ergebnis = ExamineLibraryFunctions(#Library)
Beschreibung
Initiiert den Prozess zur Untersuchung der in einer Library enthaltenen Funktionen.

Parameter

#Library Die Nummer der Library, welche die zu untersuchenden Funktionen enthält. Diese Nummer muss identisch zu der vorher mit OpenLibrary() verwendeten sein.

Rückgabewert

Gibt einen Wert ungleich Null zurück, wenn die Funktionen untersucht werden können, und Null wenn nicht.

Anmerkungen

Wenn dieser Befehl erfolgreich ist, dann kann der Programmierer schrittweise durch die Funktionen in der Library gehen (mittels NextLibraryFunction()). An dieser Stelle können weitere Informationen wie der Name und die Adresse jeder Funktion ermittelt werden (mittels der LibraryFunctionName() bzw. LibraryFunctionAddress() Befehle).

Siehe auch

NextLibraryFunction(), LibraryFunctionAddress(), LibraryFunctionName()

Unterstützte OS
