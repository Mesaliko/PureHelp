AudioCDLength()

Syntax

Ergebnis = AudioCDLength()
Beschreibung
Gibt die komplette Länge einer ganzen Audio-CD zurück.

Parameter

Diese Funktion hat keine Parameter.

Rückgabewert

Gibt die Länge in Sekunden zurück.

AudioCDTrackLength(), UseAudioCD()
