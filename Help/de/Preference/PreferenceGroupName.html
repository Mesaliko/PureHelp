PreferenceGroupName()

Syntax

Gruppe$ = PreferenceGroupName()
Beschreibung
Gibt den Name der aktuellen Gruppe, welche mittels ExaminePreferenceGroups() aufgezählt wird oder zuvor mit PreferenceGroup() ausgewählt wurde, zurück.

Parameter

Diese Funktion hat keine Parameter.

Rückgabewert

Der Name der aktuellen Gruppe, welche mittels ExaminePreferenceGroups() aufgezählt wird oder zuvor mit PreferenceGroup() ausgewählt wurde.

Beispiel

  ; Öffnet eine Preference-Datei
  OpenPreferences(#PB_Compiler_Home +"examples/sources/Data/test.pref")
  
  ; Gruppen untersuchen
  ExaminePreferenceGroups()
  While NextPreferenceGroup() ; Solange eine Gruppe existiert...
    MessageRequester("Gruppen", PreferenceGroupName())  ; ...diese Gruppe anzeigen
  Wend
  
  ; Schließt die Preference-Datei
  ClosePreferences()
