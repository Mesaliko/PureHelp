MusicVolume()

Syntaxe

Resultat = MusicVolume(#Music, Volume.f)
Description
Change le volume général en temps réel.

Arguments

#Music Le module musical à utiliser.
Volume.f Le nouveau volume sonore entre 0 et 100.

Utile pour les fondus enchaînés (fade in/fade out).

Valeur de retour

Aucune.

OS Supportés

Tous

<- LoadMusic() - Music Index - PlayMusic() ->
