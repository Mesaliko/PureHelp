MeshVertexPosition()

Syntax

MeshVertexPosition(x, y, z)
Beschreibung
Fügt einen neuen Eckpunkt (Vertex) zum aktuellen Mesh hinzu, welches zuvor mittels CreateMesh() erstellt wurde. Um bestimmte Attribute des neu erstellten Eckpunkts festzulegen, verwenden Sie die folgenden Befehle: MeshVertexNormal(), MeshVertexTangent(), MeshVertexColor() und MeshVertexTextureCoordinate(). Wenn mehrere Attribute angegeben werden müssen, kann MeshVertex() stattdessen verwendet werden. Um eine neue Oberfläche zu erstellen, verwenden Sie MeshFace().

Parameter

x, y, z Die Position des neuen Eckpunkts.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.
