ApplyEntityTorque()

Syntax

ApplyEntityTorque(#Entity, x, y, z [, Modus])
Beschreibung
Wendet eine Drehkraft (Rotation) auf das Entity an. Die neue Drehkraft ersetzt alle Kräfte, die vorher auf das Entity angewendet wurden.

Parameter

#Entity Das zu verwendende Entity-Objekt.
x, y, z Die Dreh-Kraftwerte (englisch "force values").
Modus (optional) Der anzuwendende Drehkraft-Modus. Dies kann einer der folgenden Werte sein: 
  #PB_Local : lokale Drehkraft.
  #PB_Parent: Drehkraft relativ zur übergeordneten ("parent") Position.
  #PB_World : Drehkraft relativ zur Welt.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.
