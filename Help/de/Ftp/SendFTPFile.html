SendFTPFile()

Syntax

Ergebnis = SendFTPFile(#Ftp, Dateiname$, RemoteDateiname$ [, Asynchron])
Beschreibung
Sendet eine Datei an einen FTP-Server.

Parameter

#Ftp Die zu verwendende FTP-Verbindung.
Dateiname$ Der Name der zu sendenden Datei. Wenn der Dateiname keinen vollständigen Pfad enthält, wird dieser relativ zum aktuellen Verzeichnis interpretiert.
RemoteDateiname$ Die Datei auf dem Server ("remote"). Sie muss sich im aktuellen FTP-Verzeichnis befinden (siehe GetFTPDirectory() und SetFTPDirectory()).
Asynchron (optional) Wenn auf #True gesetzt, wird der Transfer im Hintergrund erfolgen. Standardmäßig wird das Programm blockiert, bis die Datei vollständig gesendet wurde (= "synchron"). Der Fortschritt einer asynchronen Übertragung kann mit der Funktion FTPProgress() ermittelt werden, und sie kann mit Hilfe von AbortFTPFile() abgebrochen werden.

Rückgabewert

Gibt einen Wert ungleich Null zurück, wenn die Datei korrekt hochgeladen wurde, oder wenn der asynchrone Transfer korrekt initialisiert wurde. Bei einem Fehler ist der Rückgabewert gleich Null.

Anmerkungen

Nur eine Datei kann im Hintergrund auf einmal herunter- oder hochgeladen (siehe ReceiveFTPFile()) werden.
