Space()

Syntax

Ergebnis$ = Space(Länge)
Beschreibung
Erstellt einen leeren String in der angegebenen 'Länge', der mit Leerzeichen ('Space') gefüllt wird.

Parameter

Länge Die Länge (in Zeichen) des neuen Strings.

Rückgabewert

Ein neuer String, gefüllt mit Leerzeichen ('Space').

Beispiel

  Debug "-" + Space(5) + "-" ; wird "-     -" anzeigen

Unterstützte OS

Alle

<- Right() - String Inhaltsverzeichnis - Str() ->
