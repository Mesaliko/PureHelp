PreviousXMLNode()

Syntaxe

Resultat = PreviousXMLNode(Noeud)
Description
Renvoie le noeud précédent de même niveau.

Arguments

Noeud Le noeud XML à utiliser.

Valeur de retour

Renvoie le noeud précédent, ou zéro s'il n'y a plus de noeud avant celui-ci.
