StopDrawing()

Syntax

StopDrawing()
Beschreibung
Sobald alle benötigten Grafik-Operationen (gestartet mit StartDrawing()) durchgeführt wurden, muss dieser Befehl aufgerufen werden, um das Zeichnen abzuschließen und alle zugehörigen Ressourcen freizugeben.

Parameter

Diese Funktion hat keine Parameter.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.

Beispiel

Eine typische Zeichen-Sequenz sieht wie folgt aus: 
  If StartDrawing(WindowOutput(0))
    Box(10,10,20,20)
    Line(30,50,100,100)
    ....
    StopDrawing() 
  EndIf
