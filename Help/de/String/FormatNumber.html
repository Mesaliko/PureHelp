FormatNumber()

Syntax

Ergebnis$ = FormatNumber(Nummer.d [, Stellen [, DezimalPunkt$ [, TausenderTrennzeichen$]]])
Beschreibung
Formatiert eine Zahl in ein Geld-ähnliches Format.

Parameter

Nummer Die zu formatierende Zahl.
Stellen (optional) Anzahl an darzustellenden Dezimalstellen.
DezimalPunkt$ (optional) Der zu verwendende String, um den Dezimal- und Integerteil zu trennen. Dies kann auch ein String mit mehreren Zeichen sein. Standardwert ist ".".
TausenderTrennzeichen$ (optional) Der zu verwendende String, um die Tausender zu trennen. Dies kann auch ein String mit mehreren Zeichen sein. Standardwert ist ",".

Rückgabewert

Die formatierte Zahl.

Beispiel

  Debug FormatNumber(125400.25) ; Wird anzeigen: 125,400.25 (englischer Standard)
  Debug FormatNumber(1125400.25, 2, ",", ".")  ; Wird anzeigen: 1.125.400,25  (deutsches Format)
