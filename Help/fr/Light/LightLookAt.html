LightLookAt()

Syntaxe

LightLookAt(#Lumiere, X.f, Y.f, Z.f)
Description
Modifie l'orientation d'une lumière dans le monde et pointe vers le point X, Y, Z.

Arguments

#Lumiere La lumière à utiliser.
X.f, Y.f, Z.f Coordonnées du point.

Valeur de retour

Aucune.

Voir aussi

LightDirection(), LightDirectionX(), LightDirectionY(), LightDirectionZ()

OS Supportés

Tous
