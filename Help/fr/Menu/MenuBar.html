MenuBar()

Syntaxe

MenuBar()
Description
Crée une barre de séparation horizontale dans un menu déroulant.

Arguments

Aucun.

Valeur de retour

Aucune.

Exemple

  If OpenWindow(0, 200, 200, 200, 100, "Exemple MenuBar")
    If CreateMenu(0, WindowID(0))  ; la création du menu commence ici....
      MenuTitle("Projet")
        MenuItem(1, "Ouvrir")
        MenuBar()                 ; la barre de séparation sera insérée ici
        MenuItem(4, "Fermer")
    EndIf
    Repeat : Until WaitWindowEvent()=#PB_Event_CloseWindow
  EndIf
