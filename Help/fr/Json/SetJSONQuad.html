SetJSONQuad()

Syntaxe

SetJSONQuad(JSONValeur, Valeur.q)
Description
Change le type de la valeur JSON en #PB_JSON_Number et stocke la valeur 'Quad' donnée.

Arguments

JSONValeur La valeur JSON.
Valeur.q La valeur à stocker.

Valeur de retour

Aucune.

Exemple

  If CreateJSON(0)
    ValeurTableau = SetJSONArray(JSONValue(0))
    SetJSONQuad(AddJSONElement(ValeurTableau), 1)
    SetJSONQuad(AddJSONElement(ValeurTableau), 2)
    SetJSONQuad(AddJSONElement(ValeurTableau), 3)
    
    Debug ComposeJSON(0)
  EndIf
  
  ; Resultat : [1, 2, 3]
