JSONErrorMessage()

Syntax

Ergebnis$ = JSONErrorMessage()
Beschreibung
Gibt eine Nachricht zurück, welche die Ursache für den Fehler bei der letzten JSON-Parsing-Operation mittels ParseJSON(), CatchJSON() oder LoadJSON() enthält.

Parameter

Diese Funktion hat keine Parameter.

Rückgabewert

Die Fehlernachricht in Englisch.

Beispiel

  If ParseJSON(0, "[1, 2, 3 4]")
    ; arbeiten mit den Daten
  Else
    Debug JSONErrorMessage()
  EndIf
