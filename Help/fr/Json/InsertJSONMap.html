InsertJSONMap()

Syntaxe

InsertJSONMap(JSONValeur, Map())
Description
Insère une Map dans la valeur JSON donnée.

La valeur JSON sera modifiée en type #PB_JSON_Object.

Arguments

JSONValeur La valeur JSON.

Le contenu de la valeur sera remplacé par le contenu de la Map().
Map() La Map à insérer dans la valeur JSON.

Valeur de retour

Aucune.

Exemple

  NewMap Couleurs()
  Couleurs("rouge") = $0000FF
  Couleurs("vert")  = $00FF00
  Couleurs("bleu")  = $FF0000
  
  If CreateJSON(0)
    InsertJSONMap(JSONValue(0), Couleurs())
    Debug ComposeJSON(0, #PB_JSON_PrettyPrint)
  EndIf
  
  ; Resultat : 
  ;{
  ;  "bleu" : 16711680,
  ;  "vert" : 65280,
  ;  "rouge": 255
  ;}
