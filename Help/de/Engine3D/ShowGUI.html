ShowGUI()

Syntax

ShowGUI(Transparenz, ZeigeMaus [, #Kamera, Aktivieren])
Beschreibung
Zeigt oder versteckt die kompletten GUI-Elemente, welche sich aus 3D-Fenstern und 3D-Gadgets zusammensetzen.

Parameter

Transparenz Transparenz-Level des GUI. Gültige Werte liegen zwischen 0 (versteckt) und 255 (vollständig angezeigt).
ZeigeMaus Ändert die Sichtbarkeit des Mauspfeils. Ist ZeigeMaus = 1, wird der Mauspfeil angezeigt, ist ZeigeMaus = 0 wird er versteckt.
#Kamera (optional) Wenn angegeben, wird das GUI auf dieser Kamera angezeigt oder nicht. Arbeitet zusammen mit dem folgenden Parameter 'Aktivieren'.
Aktivieren (optional) Aktiviert oder deaktiviert die GUI-Anzeige auf der zuvor ausgewählten Kamera. Arbeitet zusammen mit dem vorherigen Parameter '#Kamera'.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.

Unterstützte OS

Alle
