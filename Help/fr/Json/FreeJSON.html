FreeJSON()

Syntaxe

FreeJSON(#JSON)
Description
Libère les données JSON.

Arguments

#JSON Les données JSON à libérer.

Si #PB_All est spécifié, tous les objets JSON restants sont libérés.

Valeur de retour

Aucune.

Remarques

Tous les objets JSON restants sont automatiquement libérés lorsque le programme se termine.
