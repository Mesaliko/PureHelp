CloseSerialPort()

Syntaxe

CloseSerialPort(#PortSerie)
Description
Ferme un port série.

Arguments

#PortSerie Le port série à utiliser.

Si #PB_All est spécifié, tous les ports série sont libérés.

Valeur de retour

Aucune.

Remarques

Tous les ports série ouverts sont fermés automatiquement quand le programme se termine.
