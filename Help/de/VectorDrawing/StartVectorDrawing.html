StartVectorDrawing()

Syntax

Ergebnis = StartVectorDrawing(Ausgabe)
Beschreibung
Bereitet die VectorDrawing Bibliothek zum Zeichnen auf die angegebene Ausgabe vor.

Parameter

Ausgabe Die Ausgabe, auf die gezeichnet werden soll. Diese Funktionen können verwendet werden, um eine Ausgabe für das Vektor-Zeichnen zu erhalten:

WindowVectorOutput(): Zeichnen wird direkt auf dem Fenster gerendert
ImageVectorOutput(): Zeichnen wird direkt auf den Bilddaten gerendert (siehe CreateImage())
PrinterVectorOutput(): Zeichnen wird direkt auf dem Drucker gerendert
CanvasVectorOutput(): Zeichnen wird direkt auf dem CanvasGadget() gerendert
PdfVectorOutput(): Zeichnen wird direkt in einer PDF-Datei gerendert
SvgVectorOutput(): Zeichnen wird direkt in einer SVG-Datei gerendert

Rückgabewert

Gibt einen Wert ungleich Null zurück, wenn das Zeichnen möglich ist. Oder Null, wenn die Operation fehlgeschlagen ist.

Anmerkungen

Das Zeichnen muss mittels StopVectorDrawing() abgeschlossen werden.

Wenn "Erstelle Thread-sicheres Executable" in den Compiler-Optionen aktiviert ist, dann hat jeder Thread seine eigene Zeichenausgabe. Dies bedeutet, dass zwei Threads auf verschiedenen Ausgaben zur gleichen Zeit zeichnen können.
