NextDatabaseDriver()

Syntax

Ergebnis = NextDatabaseDriver()
Beschreibung
Ermittelt Informationen über den nächsten verfügbaren Datenbank-Treiber. Dieser Befehl muss nach ExamineDatabaseDrivers() aufgerufen werden. Um Informationen über den aktuellen Treiber zu erhalten, können DatabaseDriverName() und DatabaseDriverDescription() benutzt werden.

Parameter

Diese Funktion hat keine Parameter.

Rückgabewert

Ist das Ergebnis gleich 0, dann sind keine weiteren Treiber verfügbar.

Anmerkungen

Dies ist ein spezifischer Befehl für ODBC-Datenbanken.
