ParticleColorRange()

Syntaxe

ParticleColorRange(#EmetteurParticule, PremiereCouleur, DerniereCouleur)
Description
Change la plage de couleurs des particules.

Arguments

#EmetteurParticule L'émetteur de particules à utiliser.
PremiereCouleur, DerniereCouleur Chaque particule émise obtiendra une valeur aléatoire dans la plage située entre 'PremiereCouleur' et 'DerniereCouleur'.

La commande RGB() peut être utilisée pour obtenir rapidement une couleur.

Valeur de retour

Aucune.
