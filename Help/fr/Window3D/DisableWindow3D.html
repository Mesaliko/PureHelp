DisableWindow3D()

Syntaxe

DisableWindow3D(#Fenetre3D, Etat)
Description
Active ou désactive les interactions de l'utilisateur avec une fenêtre 3D.

Arguments

#Fenetre3D La fenêtre 3D à utiliser.
Etat Peut prendre une des valeurs suivantes: 
  #True : La fenêtre 3D est désactivée.
  #False: La fenêtre 3D est activée.

Valeur de retour

Aucune.

OS Supportés

Tous
