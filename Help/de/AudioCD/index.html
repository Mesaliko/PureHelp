PureBasic - AudioCD

Einleitung

Audio-CDs sind eine hervorragende Möglichkeit zum Abspielen von qualitativ hochwertiger Musik während eines Spiels oder einer Anwendung - ohne dabei eine Menge an System-Ressourcen zu verbrauchen. Diese Bibliothek bietet alle notwendigen Funktionen, die für das Abspielen von Audio-CDs in PureBasic-Programmen benötigt werden.
