AddJSONMember()

Syntax

Ergebnis = AddJSONMember(JSONWert, Schlüssel$)
Beschreibung
Fügt ein neues Mitglied (englisch "member") zu einem JSON-Wert vom Typ #PB_JSON_Object. Wenn ein Mitglied mit dem angegebenen Schlüssel (englisch "key") bereits existiert, dann wird dieser ersetzt.

Parameter

JSONWert Der JSON-Wert. Der Wert muss vom Typ #PB_JSON_Object sein.
Schlüssel$ Der Schlüssel für das neue Mitglied. Wenn ein Mitglied mit dem gleichen Schlüssel im Objekt bereits existiert, dann wird es ersetzt.

Rückgabewert

Gibt die Adresse des hinzugefügten JSON-Werts zurück. Der neu hinzugefügte Wert hat anfänglich den Typ #PB_JSON_Null.

Beispiel

  If CreateJSON(0)
    ObjectValue = SetJSONObject(JSONValue(0))
    
    FirstName = AddJSONMember(ObjectValue, "FirstName")
    SetJSONString(FirstName, "John")
    
    LastName = AddJSONMember(ObjectValue, "LastName")
    SetJSONString(LastName, "Smith")    
    
    Debug ComposeJSON(0)
  EndIf
