GetClientIP()

Syntax

IP = GetClientIP(Client)
Beschreibung
Diese Funktion gibt die IP-Adresse des Clients zurück und sollte nach EventClient() aufgerufen werden. Wenn die Verbindung eine IPv6-Verbindung ist, dann muss die zurückgegebene Adresse mit FreeIP() freigegeben werden.

Parameter

Client Der Client, von dem die IP ermittelt werden soll.

Rückgabewert

Gibt die IP-Adresse des Clients zurück.

Anmerkungen

GetClientPort() ist verfügbar, um den Port des Client zu ermitteln. IPString() kann verwendet werden, um die IP-Adresse einfach in einen String zu konvertieren.
