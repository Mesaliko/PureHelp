FirstDatabaseRow()

Syntaxe

Resultat = FirstDatabaseRow(#BaseDeDonnees)
Description
Récupère les informations relatives au premier enregistrement d'une base de données.

Arguments

#BaseDeDonnees La base de données à utiliser.

Valeur de retour

Si le résultat est zéro alors aucun enregistrement n'est disponible.

Remarques

Attention, l'option #PB_Database_DynamicCursor doit être spécifiée pour que la fonction DatabaseQuery() puisse fonctionner.

Pour accéder aux informations à l'intérieur de l'enregistrement on peut utiliser GetDatabaseLong(), GetDatabaseFloat() et GetDatabaseString().

Voir aussi

NextDatabaseRow(), PreviousDatabaseRow(), GetDatabaseLong()

OS Supportés

Tous
