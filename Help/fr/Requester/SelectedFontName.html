SelectedFontName()

Syntaxe

Resultat$ = SelectedFontName()
Description
Renvoie dans le nom de la police sélectionnée par l'utilisateur avec la fonction FontRequester().

Arguments

Aucun.

Valeur de retour

Renvoie dans le nom de la police sélectionnée par l'utilisateur avec la fonction FontRequester().
Ce nom peut être utilisé directement par la fonction LoadFont().
