GetClientPort()

Syntaxe

Resultat = GetClientPort(Client)
Description
Renvoie le port du client.

Arguments

Client Le client pour lequel vous souhaitez obtenir le port.

Valeur de retour

Renvoie le port du client.

Remarques

Cette commande doit être appelée après EventClient().

Voir aussi

GetClientIP(), EventClient()
