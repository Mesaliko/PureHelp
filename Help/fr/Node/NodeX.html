NodeX()

Syntaxe

Resultat = NodeX(#Noeud [, Mode])
Description
Renvoie la position en 'X' d'un noeud dans le Monde.

Arguments

#Noeud Le noeud à utiliser.
Mode (optionnel) Peut être une des valeurs suivantes: 
  #PB_World:  Renvoie la position absolue en 'X' dans le monde (par défaut).
  #PB_Parent: Renvoie la position relative à son parent en 'X'.

Valeur de retour

Renvoie la position 'X' du noeud dans le Monde.

Voir aussi

NodeY(), NodeZ(), MoveNode()
