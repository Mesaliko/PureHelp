FreeMap()

Syntax

FreeMap(Map())
Beschreibung
Gibt die angegebene Map und all ihren zugewiesenen Speicher frei. Um erneut auf sie zuzugreifen, muss NewMap aufgerufen werden.

Parameter

Map() Die freizugebende Map.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.
