RegularExpressionError()

Syntax

Ergebnis$ = RegularExpressionError()
Beschreibung
Gibt eine vom Menschen lesbare Fehlermeldung (in Englisch) über den letzten Fehler von CreateRegularExpression() zurück.

Parameter

Diese Funktion hat keine Parameter.

Rückgabewert

Gibt eine vom Menschen lesbare Fehlermeldung (in Englisch) über den letzten Fehler von CreateRegularExpression() zurück.

Beispiel

  ; Hier geben wir eine zusätzliche eckige Klammer '[' an, wodurch sich ein Syntax-Fehler
  ; im regulären Ausdruck ergibt.
  ;    
  If CreateRegularExpression(0, "[a-z]b[[A-Z][]")
    Debug "Success"
  Else
    Debug RegularExpressionError()
  EndIf
