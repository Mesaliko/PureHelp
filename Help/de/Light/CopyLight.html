CopyLight()

Syntax

Ergebnis = CopyLight(#Light, #NewLight)
Beschreibung
Erstellt ein neues Licht als eine exakte Kopie des angegebenen Lichts. Alle Licht-Attribute, wie Farbe, Spiegel-Farbe, Position etc., werden dabei mit dupliziert.

Parameter

#Light Das zu kopierende Licht.
#NewLight Eine Nummer zum Identifizieren des neuen Lichts. #PB_Any kann verwendet werden, um diese Nummer automatisch zu generieren.

Rückgabewert

Ein Wert ungleich Null, wenn das Licht erfolgreich dupliziert wurde, andernfalls Null. Wenn #PB_Any als #NewLight Parameter verwendet wurde, dann wird im Erfolgsfall die generierte Nummer zurückgegeben.

Unterstützte OS

Alle
