MakeIPAddress()

Syntax

Ergebnis = MakeIPAddress(Feld0, Feld1, Feld2, Feld3 [, Feld4, Feld5, Feld6, Feld7])
Beschreibung
Gibt den entsprechenden numerischen Wert der angegebenen IP-Adresse zurück.

Parameter

Feld0, Feld1, Feld2, Feld3 Die individuellen Felder der Adresse. Jedes Feld kann nur einen Wert zwischen 0 und 255 haben.
Feld4, Feld5, Feld6, Feld7 (optional) Die verbleibenden einzelnen Felder für eine IPv6-Adresse. Jedes Feld kann nur einen Wert zwischen 0 und 255 haben. Wenn diese Felder angegeben werden, dann wird eine IPv6-Adresse erstellt. Wenn die Adresse nicht mehr benötigt wird, muss diese manuell mit FreeIP() freigegeben werden. Eine IPv4-Adreese muss nicht mit FreeIP() freigegeben werden.

Rückgabewert

Gibt die IP-Adresse zurück.

Anmerkungen

Diese Funktion ist insbesondere nützlich mit:

- IPAddressGadget()

Der numerische Wert kann mit dem IPString() Befehl in einen (mit Punkten unterteilten) String zurückkonvertiert werden.
