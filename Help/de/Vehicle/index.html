PureBasic - Vehicle

Einleitung

Fahrzeuge (englisch "vehicles") bestehen in der Regel aus einem Körper und einem oder mehreren Rädern (PKW, LKW, Fahrrad, etc.). Das neu erstellte Fahrzeug ist ein reguläres Entity, so dass alle Entity-Funktionen verwendet werden können, um ein Fahrzeug zu manipulieren.

InitEngine3D() sollte erfolgreich aufgerufen werden, bevor die Vehicle-Funktionen verwendet werden.
