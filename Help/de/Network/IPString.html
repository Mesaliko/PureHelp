IPString()

Syntax

String$ = IPString(IPAdresse [, Format])
Beschreibung
Gibt die angegebene numerische IP-Adresse in einem mit Punkten unterteilten String (d.h. "127.0.0.1" für IPv4 bzw. "::1" für IPv6) zurück.

Parameter

IPAdresse Die IP-Adresse. Bei IPv6 muss diese Adresse das Ergebnis von MakeIPAddress(), NextIPAddress() oder GetClientIP() sein.
Format (optional) Das Format der umzuwandelnden IP. Dies kann einer der folgenden Werte sein: 
  #PB_Network_IPv4: konvertiert eine IPv4-Adresse (Standard).
  #PB_Network_IPv6: konvertiert eine IPv6-Adresse.

Rückgabewert

Gibt die IP-Adresse als einen String zurück.
