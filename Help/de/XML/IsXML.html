IsXML()

Syntax

Ergebnis = IsXML(#XML)
Beschreibung
Testet, ob die angegebene #XML Nummer eine gültige und korrekt initialisierte XML-Struktur ist.

Parameter

#XML Die zu verwendende XML-Struktur (Baum).

Rückgabewert

Gibt einen Wert ungleich Null zurück, wenn #XML eine gültige XML-Struktur ist, andernfalls Null.

Anmerkungen

Diese Funktion ist "kugelsicher" und kann mit jedem Wert verwendet werden. Ist 'Ergebnis' ungleich Null, dann ist das Objekt gültig und initialisiert, andernfalls ist es gleich Null. Dies ist der korrekte Weg, um sicherzustellen, dass eine XML-Struktur bereit zur Verwendung ist.
