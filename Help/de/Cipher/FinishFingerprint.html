FinishFingerprint()

Syntax

Ergebnis$ = FinishFingerprint(#Fingerprint)
Beschreibung
Schließt die Berechnung der Prüfsumme ab, welche zuvor mittels StartFingerprint() gestartet wurde, und gibt die Prüfsumme als einen hexadezimalen String zurück.

Parameter

#FingerPrint Der abzuschließende Fingerprint ("Fingerabdruck").

Rückgabewert

Gibt die Prüfsumme als einen hexadezimalen String zurück.

Anmerkungen

Dieser Befehl sollte zum Abschluss der Prüfsummen-Berechnung aufgerufen werden, auch wenn die Prüfsumme aktuell nicht mehr benötigt wird, da er auch alle mit der Prüfsummen-Berechnung zusammenhängenden Daten frei gibt.

Siehe StartFingerprint() für ein Code-Beispiel und weitere Informationen.
