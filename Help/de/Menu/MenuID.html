MenuID()

Syntax

MenuID = MenuID(#Menü)
Beschreibung
Gibt den eindeutigen System-Identifier (Bezeichner) des angegebenen Menüs zurück.

Parameter

#Menü Das zu verwendende Menü.

Rückgabewert

Gibt die ID des Menüs zurück. Diese ist manchmal auch als 'Handle' bekannt. Schauen Sie in das Extra-Kapitel Handles und Nummern für weitere Informationen.

Beispiel

  If OpenWindow(0, 200, 200, 200, 100, "MenuID Example")
    If CreateMenu(0, WindowID(0))
      Debug "Handle of current menu is:"
      Debug MenuID(0)
    EndIf
    Repeat : Until WaitWindowEvent() = #PB_Event_CloseWindow
  EndIf
