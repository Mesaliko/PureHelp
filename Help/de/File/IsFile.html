IsFile()

Syntax

Ergebnis = IsFile(#Datei)
Beschreibung
Überprüft, ob die angegebene #Datei-Nummer gültig und eine korrekt initialisierte Datei ist.

Parameter

#Datei Die zu verwendende Datei.

Rückgabewert

Gibt einen Wert ungleich Null zurück, wenn #Datei eine gültige Datei ist, andernfalls Null.

Anmerkungen

Diese Funktion ist "kugelsicher" und kann mit jedem Wert benutzt werden. Dies ist der richtige Weg, um sicher zu stellen, dass eine Datei bereit zur Verwendung ist.
