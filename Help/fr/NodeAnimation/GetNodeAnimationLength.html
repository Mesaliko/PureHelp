GetNodeAnimationLength()

Syntaxe

Duree = GetNodeAnimationLength(#NodeAnimation)
Description
Renvoie la durée d'une animation.

Arguments

#NodeAnimation L'animation à utiliser.

Valeur de retour

Renvoie la durée de l'animation (en millisecondes).

Voir aussi

StartNodeAnimation(), SetNodeAnimationLength()

OS Supportés

Tous
