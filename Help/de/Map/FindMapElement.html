FindMapElement()

Syntax

Ergebnis = FindMapElement(Map(), Key$)
Beschreibung
Ändert das aktuelle Map-Element auf das dem angegebenen Schlüssel 'Key$' zugewiesene Element.

Parameter

Map() Die zu verwendende Map.
Key$ Der zu findende Schlüssel.

Rückgabewert

Gibt einen Wert ungleich Null zurück, wenn der Schlüssel gefunden wurde, andernfalls Null. Der zurückgegebene Wert ist ein Zeiger auf die Element-Daten.

Beispiel

  NewMap Country.s()

  Country("US") = "United States"
  Country("FR") = "France"
  Country("GE") = "Germany"

  If FindMapElement(Country(), "US")
    Debug "'US' ist in der Länder-Liste."    
  Else
    Debug "'US' ist NICHT in der Länder-Liste!"    
  EndIf
  
  If FindMapElement(Country(), "UK")
    Debug "'UK' ist in der Länder-Liste."
  Else
    Debug "'UK' ist NICHT in der Länder-Liste!"
  EndIf
