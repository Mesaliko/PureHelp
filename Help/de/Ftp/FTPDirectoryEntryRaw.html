FTPDirectoryEntryRaw()

Syntax

Eintrag$ = FTPDirectoryEntryRaw(#Ftp)
Beschreibung
Gibt die "rohen" (unbearbeiteten) Informationen des aktuellen Eintrags - in der mittels der ExamineFTPDirectory() und NextFTPDirectoryEntry() Funktionen aufgelisteten FTP-Quelle - so zurück, wie diese vom FTP-Server gesendet wurden. Dies kann nützlich sein, wenn der Server von ExamineFTPDirectory() nicht unterstützt wird, um trotzdem die Informationen zum Verzeichnis-Listing zu empfangen und entsprechend zu verarbeiten.

Parameter

#Ftp Die zu verwendende FTP-Verbindung.

Rückgabewert

Gibt die Informationen zum Eintrag zurück, wie diese vom FTP-Server gesendet wurden.

Beispiel

  If OpenFTP(0, "ftp.free.fr", "anonymous", "")
    If ExamineFTPDirectory(0)
      While NextFTPDirectoryEntry(0)
        Debug FTPDirectoryEntryRaw(0)
      Wend
    EndIf
  Else
    Debug "Konnte keine Verbindung zu ftp.free.fr aufbauen."
  EndIf
