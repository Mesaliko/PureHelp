StatusBarHeight()

Syntax

Ergebnis = StatusBarHeight(#Statusleiste)
Beschreibung
Gibt die Höhe (in Pixel) der '#Statusleiste' zurück. Dies ist nützlich für die korrekte Berechnung der Fensterhöhe, wenn eine Statusleiste verwendet wird.

Parameter

#Statusleiste Die zu verwendende Statusleiste.

Rückgabewert

Die Höhe (in Pixel) der Statusleiste.

Unterstützte OS

Alle

<- IsStatusBar() - StatusBar Inhaltsverzeichnis - StatusBarID() ->
