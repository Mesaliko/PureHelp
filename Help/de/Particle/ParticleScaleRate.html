ParticleScaleRate()

Syntax

ParticleScaleRate(#ParticleEmitter, Rate.f)
Beschreibung
Ändert die aktuelle Skalierrate des Partikelstrahlers.

Parameter

#ParticleEmitter Der zu verwendende Partikelstrahler.
Rate Die neue Partikelstrahler-Skalierrate. Die Standardrate ist 0, was bedeutet, dass das Partikel die gleiche Größe haben wird. Wenn ein Wert größer als 0 verwendet wird, erhöht sich die Partikelgröße im Laufe der Zeit (jede Sekunde wird die Rate zur aktuellen Größe addiert). Wenn ein Wert unter 0 verwendet wird, wird die Partikelgröße im Laufe der Zeit verringert (jede Sekunde wird die Rate von der aktuellen Größe abgezogen).

Zum Beispiel:
Eine Rate von 10 wird die Partikelgröße um 10 Einheiten pro Sekunde vergrößern.
Eine Rate von -2 wird die Partikelgröße um 2 Einheiten pro Sekunde verringern.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.
