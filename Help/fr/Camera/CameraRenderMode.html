CameraRenderMode()

Syntaxe

CameraRenderMode(#Camera, Mode)
Description
Change le mode de rendu d'une caméra.

Arguments

#Camera La caméra à utiliser.
Mode La manière dont le monde doit être rendu: 
  #PB_Camera_Plot     : Rend la scène en mode point (seuls les sommets des objets sont visibles)
  #PB_Camera_Wireframe: Rend la scène en mode ligne (seuls les triangles des objets sont visibles)
  #PB_Camera_Textured : Rend la scène avec un maximum de détails (textures, transparence etc..)

Valeur de retour

Aucune.

Remarques

Quand vous créez une nouvelle caméra avec CreateCamera(), le mode de rendu par défaut se fait avec un niveau de détails et de textures maximum.
