DefineTerrainTile()

Syntax

Ergebnis = DefineTerrainTile(#Terrain, TileX, TileY, HöhenKarte$, FlipX, FlipY)
Beschreibung
Definiert den Inhalt eines Tile (Kachel) im Gelände/Terrain-Raster.

Parameter

#Terrain Das zu verwendende Terrain.
TileX Der x-Index des Tile, relativ zum mittleren Tile des Terrains. Das mittlere Tile beginnt bei 0,0. Der Wert kann negativ sein.
TileY Der y-Index des Tile, relativ zum mittleren Tile des Terrains. Das mittlere Tile beginnt bei 0,0. Der Wert kann negativ sein.
HöhenKarte$ Der Name des Bildes mit der Höhenkarte (englisch "height map"), welches auf dieses Tile angewandt werden soll.
FlipX Wenn auf #True gesetzt, wird das Bild horizontal gedreht. Wenn auf #False gesetzt, wird keine horizontale Drehung ausgeführt.
FlipY Wenn auf #True gesetzt, wird das Bild vertikal gedreht. Wenn auf #False gesetzt, wird keine vertikale Drehung ausgeführt.

Rückgabewert

Gibt null zurück, wenn das #Terrain nicht erstellt werden konnte
