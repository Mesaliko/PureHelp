FinishFTPDirectory()

Syntaxe

FinishFTPDirectory(#FTP)
Description
Termine l'énumération préalablement commencée avec ExamineFTPDirectory().

Arguments

#FTP La connexion FTP à utiliser.

Valeur de retour

Aucune.

Remarques

Permet de libérer les ressources allouées lors de l'examen du répertoire.
