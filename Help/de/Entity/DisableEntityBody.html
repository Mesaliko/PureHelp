DisableEntityBody()

Syntax

DisableEntityBody(#Entity, Deaktivieren)
Beschreibung
Deaktiviert den (physischen) Körper eines Entity. Die Physik-Engine hat keinen Einfluss auf das Entity mehr, wenn ein Entity-Körper deaktiviert ist.

Parameter

#Entity Das zu verwendende Entity.
Deaktivieren Wenn auf #True gesetzt, wird der Entity-Körper deaktiviert. Wenn auf #False gesetzt, wird der Körper aktiviert.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.

Unterstützte OS

Alle
