PureBasic - Keyboard

Einleitung

PureBasic bietet einfachen und schnellen Zugriff auf die Tastatur. Diese Möglichkeit sollte nur in Applikationen genutzt werden, die einen extrem schnellen Zugriff auf die Tastatur im "Roh-Format" (raw) benötigen, wie z.B. Spielen. Es wird die DirectX Technologie benutzt.

Diese Library wurde für Spiele und Multimedia-Applikationen geschaffen, welche schnellen - auf DirectX basierenden - Tastaturzugriff benötigen. Für normale Windows-Applikationen ist es besser, den AddKeyboardShortcut() Befehl zu verwenden.
