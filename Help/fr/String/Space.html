Space()

Syntaxe

Resultat$ = Space(Longueur)
Description
Créé une chaîne de caractères de longueur 'Longueur' et ne contenant que des espaces.

Arguments

Longueur Le nombre d'espaces à ajouter.

Valeur de retour

Renvoie une chaîne de caractères de longueur 'Longueur' et ne contenant que des espaces.

Exemple

  Debug Space(10) ; Affichera "           ".

OS Supportés

Tous
