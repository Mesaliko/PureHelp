FreeLight()

Syntax

FreeLight(#Light)
Beschreibung
Gibt das angegebene Licht '#Light' frei. Sein gesamter zugehöriger Speicherplatz wird freigegeben und an dieser Stelle darf das Objekt nicht mehr verwendet werden.

Parameter

#Light Das freizugebende Licht. Wenn #PB_All angegeben wird, dann werden alle verbliebenen Lichter freigegeben.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.

Anmerkungen

Alle verbleibenden Lichter werden automatisch freigegeben, wenn das Programm endet.
