PureBasic - VertexAnimation

Généralités

Les animations vertex permettent de gérer les "animations-poses" d'un mesh .

Le mesh doit contenir une liste prédéfinie de poses.

Sommaire des commandes

AddVertexPoseReference
CreateVertexAnimation
CreateVertexPoseKeyFrame
CreateVertexTrack
MeshPoseCount
MeshPoseName
UpdateVertexPoseReference
VertexPoseReferenceCount

Exemple

VertexAnimation.pb

OS Supportés

Tous
