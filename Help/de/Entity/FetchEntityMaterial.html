FetchEntityMaterial()

Syntax

Ergebnis = FetchEntityMaterial(#Entity, #Material [, SubEntity])
Beschreibung
Ermittelt das Material, welches dem angegebenen #Entity mittels @setentitymaterialzugewiesen wurde.

Parameter

#Entity Das zu verwendende Entity.
#Material Die neu erstellte Material-Nummer.
SubEntity (optional) Das Sub-Entity, von welchem das Material ermittelt werden soll. Der erste Sub-Entity-Index beginnt bei 0 (was dem Haupt-Entity entspricht).

Rückgabewert

Gibt bei Erfolg einen Wert ungleich Null zurück, und Null bei einem Fehler.

Unterstützte OS

Alle

<- EntityZ() - Entity Inhaltsverzeichnis - FreeEntity() ->
