MoveBillboardGroup()

Syntax

MoveBillboardGroup(#BillboardGruppe, x, y, z [, Modus])
Beschreibung
Bewegt eine Billboard-Gruppe entsprechend der angegebenen x, y und z Werte. Dies ist standardmäßig eine relative Bewegung, ausgehend von der aktuellen Position der Billboard-Gruppe.

Parameter

#BillboardGruppe Die zu bewegende Billboard-Gruppe.
x, y, z Die neue Position der Billboard-Gruppe.
Modus (optional) Der Bewegungs-Modus. Dies kann einer der folgenden Werte sein: 
  #PB_Relative: relative Bewegung, ausgehend von der aktuellen Position der Billboard-Gruppe (Standard).
  #PB_Absolute: absolute Bewegung an die angegebene Position.
kombiniert mit einem der folgenden Werte: 
  #PB_Local : lokale Bewegung.
  #PB_Parent: Bewegung relativ zur übergeordneten ("parent") Position.
  #PB_World : Bewegung relativ zur Welt.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.
