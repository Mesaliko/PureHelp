GetScriptParticleEmitter()

Syntaxe

Resultat = GetScriptParticleEmitter(#EmetteurParticule, Nom$)
Description
Obtenir un émetteur de particules défini dans un fichier de script OGRE.

Arguments

#EmetteurParticule Le numéro du nouvel émetteur de particules à trouver.

#PB_Any peut être utilisé pour générer automatiquement ce numéro.
Nom$ Le nom de l'émetteur de particules dans le fichier de script OGRE.
Ce nom est sensible à la casse.

Valeur de retour

Renvoie zéro si l'émetteur de particules est introuvable ou ne peut pas être chargé à partir des fichiers de script.

Si #PB_Any a été utilisé alors le numéro auto-généré est renvoyé en cas de succès.

Valeur de retour

Aucune.
