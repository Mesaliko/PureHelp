CreateMaterial()

Syntax

Ergebnis = CreateMaterial(#Material, TexturID [, Farbe])
Beschreibung
Erstellt ein neues Material, basierend auf der angegebenen Textur.

Parameter

#Material Eine Nummer zum Identifizieren des neuen Materials. #PB_Any kann verwendet werden, um diese Nummer automatisch zu generieren.
TexturID Die zu verwendende Textur oder #Null, wenn keine Textur benötigt wird. Eine gültige 'TexturID' kann einfach mittels dem TextureID() Befehl ermittelt werden.
Farbe (optional) Die für Ambient und Diffusion zu verwendende Farbe. Eine gültige RGB-Farbe kann mit RGB() erzeugt werden.

Rückgabewert

Ein Wert ungleich Null, wenn das Material erfolgreich erstellt wurde, andernfalls Null. Wenn #PB_Any als Parameter #Material verwendet wurde, dann wird im Erfolgsfall die generierte Nummer zurückgegeben.
