DesktopResolutionX()

Syntax

Ergebnis.d = DesktopResolutionX()
Beschreibung
Gibt den DPI-Auflösungsfaktor des Desktops auf der 'x'-Achse zurück.

Parameter

Diese Funktion hat keine Parameter.

Rückgabewert

Gibt den DPI-Auflösungsfaktor des Desktops auf der 'x'-Achse zurück. Wenn der Wert '1' lautet, wurde auf die Anzeige auf der 'x'-Achse kein DPI-Faktor angewendet. Wenn der Wert '1.25' lautet, wurde der Anzeige auf der 'x'-Achse ein Faktor von 125% zugewiesen.

Anmerkungen

Die Anwendung muss mit der Option 'DPI-kompatibel' kompiliert werden, damit dieser Befehl einen tatsächlichen DPI-Auflösungsfaktor liefert. Wenn nicht, ist das Ergebnis immer '1'.

Beispiel

  Debug "Desktop DPI 'x' Faktor: " + DesktopResolutionX()
