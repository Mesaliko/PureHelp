EntityID()

Syntax

EntityID = EntityID(#Entity)
Beschreibung
Gibt die einmalige ID zurück, welche das angegebene '#Entity' im Betriebssystem identifiziert. Diese Funktion ist sehr nützlich, wenn eine andere Bibliothek einen Entity-Bezug benötigt.

Unterstützte OS

Alle
