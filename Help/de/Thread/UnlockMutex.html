UnlockMutex()

Syntax

UnlockMutex(Mutex)
Beschreibung
Entsperrt ("unlock") einen zuvor mittels LockMutex() gesperrten Mutex. Der Mutex ist dann für andere Threads erneut zum Sperren verfügbar.

Parameter

Mutex Der zu entsperrende Mutex.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.

Anmerkungen

Ein Mutex kann nur durch den Thread entsperrt werden, welcher ihn auch gesperrt hat.

Siehe CreateMutex() für einen Beispiel-Code.
