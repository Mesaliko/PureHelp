GetNodeAnimationWeight()

Syntax

Ergebnis = GetNodeAnimationWeight(#NodeAnimation)
Beschreibung
Gibt das Gewicht (oder die Wertigkeit) der #NodeAnimation zurück. Das Gewicht ist nützlich beim Abspielen mehrerer Animationen auf einmal. Um zum Beispiel einen sanften Übergang von einer Animation auf die andere zu haben, ist es möglich, schrittweise das Gewicht der ersten Animation zu reduzieren und das Gewicht der zweiten Animation zu erhöhen.

Parameter

#NodeAnimation Die zu verwendende Node-Animation.

Rückgabewert

Das aktuelle Gewicht der Node-Animation (Wert zwischen 0.0 und 1.0). Ist das Gewicht gleich 0, dann hat die Animation keinen Effekt. Ist das Gewicht gleich 1, dann wird die Animation vollständig abgespielt.
