CreateVertexTrack()

Syntax

Ergebnis = CreateVertexTrack(#Mesh, Animation$, Index)
Beschreibung
Erstellt einen neuen Track (Spur, Strecke) mit der Vertex-Animation für das angegebene Mesh. Die Animation sollte bereits mit CreateVertexAnimation() erstellt worden sein, oder vordefiniert im Mesh. Jeder Track hat die gleiche Länge, wie bei CreateVertexAnimation() definiert.

Parameter

#Mesh Das zu verwendende Mesh, um den neuen Animations-Track zu erstellen.
Animation$ Der Animations-Name. Dieser Name beachtet die Groß- und Kleinschreibung. Die Animation sollte bereits mittels CreateVertexAnimation() erstellt worden sein.
Index Der neue Index des Tracks (sollte bei Null starten).

Rückgabewert

Gibt einen Wert ungleich Null zurück, wenn der Animations-Track erfolgreich erstellt wurde, andernfalls Null.
