ZoomSprite()

Syntax

ZoomSprite(#Sprite, Breite, Höhe)
Beschreibung
Zoomt (vergrößert/verkleinert) das angegebene #Sprite auf die angegebenen Dimensionen.

Parameter

#Sprite Das zu zoomende Sprite.
Breite Neue Sprite-Breite, in Pixel. Wenn #PB_Default angegeben wird, dann wird die anfängliche Sprite-Breite wiederhergestellt.
Höhe Neue Sprite-Höhe, in Pixel. Wenn #PB_Default angegeben wird, dann wird die anfängliche Sprite-Höhe wiederhergestellt.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.

Unterstützte OS

Alle
