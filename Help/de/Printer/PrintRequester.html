PrintRequester()

Syntax

Ergebnis = PrintRequester()
Beschreibung
Öffnet einen regulären Druck-Requester, um den Drucker auszuwählen und einige Einstellungen vorzunehmen. Diese Funktion muss vor allen anderen Druckerfunktionen aufgerufen werden. Nach dem erfolgreichen Aufruf von PrintRequester() wird der Druckvorgang mit StartPrinting() gestartet.

Parameter

Diese Funktion hat keine Parameter.

Rückgabewert

Ungleich Null, wenn der Benutzer erfolgreich einen Drucker ausgewählt hat, andernfalls Null.

Unterstützte OS

Alle

<- NewPrinterPage() - Printer Inhaltsverzeichnis - PrinterOutput() ->
