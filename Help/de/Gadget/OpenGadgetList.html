OpenGadgetList()

Syntax

OpenGadgetList(#Gadget [, Eintrag])
Beschreibung
Verwendet das angegebene Gadget als Gadgetliste, um darauf dynamisch neue Gadgets hinzuzufügen.

Parameter

#Gadget Das Gadget, in welchem neue Gadgets erstellt werden sollen.
Eintrag (optional) Für das PanelGadget(): Gibt das Panel (Schalttafel) an, zu welchem die Gadgets hinzugefügt werden sollen. Um dem PanelGadget() dynamisch ein neues Panel hinzuzufügen, muss der Parameter 'Eintrag' dagegen weggelassen werden.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.

Anmerkungen

Die folgenden Gadgets werden von OpenGadgetList() unterstützt:

- ContainerGadget()
- PanelGadget()
- ScrollAreaGadget()

Sobald alle notwendigen Änderungen vorgenommen wurden, sollte CloseGadgetList() aufgerufen werden.
