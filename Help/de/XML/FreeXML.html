FreeXML()

Syntax

FreeXML(#XML)
Beschreibung
Gibt das XML-Objekt und alle enthaltenen Daten frei.

Parameter

#XML Das freizugebende XML-Objekt. Wenn #PB_All angegeben wird, dann werden alle verbliebenen XML-Objekte freigegeben.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.

Anmerkungen

Alle verbleibenden XML-Objekte werden automatisch freigegeben, wenn das Programm endet.

Unterstützte OS

Alle

<- FormatXML() - XML Inhaltsverzeichnis - GetXMLAttribute() ->
