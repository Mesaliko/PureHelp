EnableWorldPhysics()

Syntaxe

EnableWorldPhysics(Etat)
Description
Active ou désactive le moteur physique de l'environnement 3D.

Arguments

Etat Avec une valeur non nulle, le moteur physique est activé (mode par défaut).

Toutes les entités qui ont un corps (body) attribué par la commande CreateEntityBody() seront affectées par le moteur physique.

Les collisions peuvent être activées avec EnableWorldCollisions().

Les collisions ne fonctionnent que si le moteur physique est activé.

Valeur de retour

Aucune.
