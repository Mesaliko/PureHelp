TerrainHeight()

Syntax

Ergebnis = TerrainHeight(#Terrain, x, z)
Beschreibung
Ermittelt die Höhe des Terrains (Geländes) an der angegebenen Position in der Welt.

Parameter

#Terrain Das zu verwendende Terrain.
x Die 'x' Position in der Welt (in Welt-Einheiten), um die Terrain-Höhe zu ermitteln.
z Die 'z' Position in der Welt (in Welt-Einheiten), um die Terrain-Höhe zu ermitteln.

Rückgabewert

Gibt die Terrain-Höhe (in Welt-Einheiten) an der angegebenen Position zurück. Wenn das Terrain an dieser Position nicht gefunden wird, dann wird das Ergebnis null sein.
