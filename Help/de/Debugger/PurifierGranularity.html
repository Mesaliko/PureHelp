PurifierGranularity()

Syntax

PurifierGranularity(GlobaleGranularität, LokaleGranularität, StringGranularität, DynamikGranularität)
Beschreibung
Ändert das Intervall, in welchem der Purifier ("Reiniger") die verschiedenen Bereiche auf Speicherfehler überprüft.

Parameter

GlobaleGranularität Die Anzahl an Quellcode-Zeilen (= Granularität), die zwischen jeder Überprüfung der globalen Variablen ausgeführt werden. Die Verwendung von #PB_Ignore lässt den vorhandenen Intervall-Wert gleich. Ein Wert von 0 deaktiviert die Überprüfung.
LokaleGranularität Die Anzahl an Quellcode-Zeilen (= Granularität), die zwischen jeder Überprüfung der lokalen Variablen ausgeführt werden. Die Verwendung von #PB_Ignore lässt den vorhandenen Intervall-Wert gleich. Ein Wert von 0 deaktiviert die Überprüfung.
StringGranularität Die Anzahl an Quellcode-Zeilen (= Granularität), die zwischen jeder Überprüfung des String-Speichers ausgeführt werden. Die Verwendung von #PB_Ignore lässt den vorhandenen Intervall-Wert gleich. Ein Wert von 0 deaktiviert die Überprüfung.
DynamikGranularität Die Anzahl an Quellcode-Zeilen (= Granularität), die zwischen jeder Überprüfung des reservierten Speichers ausgeführt werden. Die Verwendung von #PB_Ignore lässt den vorhandenen Intervall-Wert gleich. Ein Wert von 0 deaktiviert die Überprüfung.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.

Beispiel

  ; Deaktiviert die Überprüfung für den String-Speicher und überprüft 
  ; den reservierten Speicher alle 10 Zeilen
  PurifierGranularity(#PB_Ignore, #PB_Ignore, 0, 10)
