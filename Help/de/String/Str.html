Str()

Syntax

Ergebnis$ = Str(Wert.q)
Beschreibung
Konvertiert eine vorzeichenbehaftete Ganzzahl (Quad) in einen String.

Parameter

Wert.q Der zu konvertierende Wert.

Rückgabewert

Ein String, der den konvertierten Wert beinhaltet.

Anmerkungen

Fließkommazahlen müssen mit dem Befehl StrF(), Doubles mit StrD() und vorzeichenlose Zahlen mit dem Befehl StrU() konvertiert werden. Es ist möglich, diesen Befehl beim Verketten von Strings und Ganzzahlen wegzulassen, es wird dann das Standardverhalten von Str() verwendet.

Beispiel

  Value.q = 100000000000000001
  Debug "Ergebnis: " + Str(Value)
