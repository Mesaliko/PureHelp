IsPack()

Syntaxe

Resultat = IsPack(#Pack)
Description
Teste si le numéro de pack donné est un pack valide et correctement initialisé.

Arguments

#Pack Le pack à tester.

Valeur de retour

Renvoie une valeur non nulle si #Pack est un pack valide, zéro sinon.

Remarques

Cette fonction est infaillible et peut être utilisée avec n'importe quelle valeur. C'est la bonne façon de s'assurer qu'un pack est prêt à être utilisé.
