MenuTitle()

Syntax

MenuTitle(Titel$)
Beschreibung
Erstellt einen neuen Titel-Eintrag für das Menü.

Parameter

Titel$ Der Text, welcher als Titel-Eintrag des Menüs angezeigt wird. Auf Windows können Sie im Titel$ das spezielle '&' Zeichen benutzen, um einen bestimmten Buchstaben zu unterstreichen, wenn das "Grafik-Theme" dies erlaubt:

"&Datei" ergibt in Wirklichkeit: Datei

Rückgabewert

Diese Funktion gibt keinen Wert zurück.

Anmerkungen

MenuTitle() kann nicht in Popup-Menüs verwendet werden, da es hier keinen Menütitel gibt.

Beispiel

  If OpenWindow(0, 200, 200, 200, 100, "MenuTitle Example")
    If CreateMenu(0, WindowID(0))
      MenuTitle("Project")      ; normaler Menü-Titel mit nachfolgenden Einträgen
        MenuItem(1, "Open")  
        MenuItem(2, "Close")
      MenuTitle("&Edit")        ; Menü-Titel mit unterstrichenem Zeichen, die Unterstreichung wird 
                                ; jedoch nur angezeigt, wenn das Menü mit der F10-Taste aufgerufen wird
        MenuItem(3, "Undo")
        MenuItem(4, "Redo")
      MenuTitle("About")        ; nur Menü-Titel
    EndIf
    Repeat : Until WaitWindowEvent() = #PB_Event_CloseWindow
  EndIf
