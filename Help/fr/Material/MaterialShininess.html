MaterialShininess()

Syntaxe

MaterialShininess(#Matiere, Brillance [, CouleurSpeculaire])
Description
Modifie la brillance d'une matière.

Arguments

#Matiere La matière à utiliser.
Brillance Nouvelle brillance.
CouleurSpeculaire (optionnel) La couleur spéculaire à utiliser.
Une couleur RVB valide peut être créée avec RGB().

Valeur de retour

Aucune.

Remarques

Pour obtenir l'état courant de ce mode, voir GetMaterialAttribute().
