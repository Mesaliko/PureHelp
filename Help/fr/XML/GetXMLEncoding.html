GetXMLEncoding()

Syntaxe

Resultat = GetXMLEncoding(#XML)
Description
Renvoie l'encodage du texte utilisé pour exporter ou enregistrer un arbre XML.

Arguments

#XML L'arbre XML à utiliser.

Valeur de retour

Renvoie une des valeurs suivantes: 
  #PB_Ascii
  #PB_Unicode (UTF16)
  #PB_UTF8

Voir aussi

ExportXML(), SaveXML(), SetXMLEncoding()
