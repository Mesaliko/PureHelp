FindMapElement()

Syntaxe

Resultat = FindMapElement(Map(), Cle$)
Description
Change l'élément courant.

Arguments

Map() La map à utiliser.
Cle$ La clé de l'élément.

Valeur de retour

Renvoie une valeur non nulle en cas de succès ou zéro sinon.
La valeur renvoyée est un pointeur vers l'élément courant.

Exemple

  NewMap Pays.s()

  Pays("US") = "United States"
  Pays("FR") = "France"
  Pays("DE") = "Allemagne"

  If FindMapElement(Pays(), "US")
    Debug "'US' est dans la liste des Pays."    
  Else
    Debug "'US' n'est PAS dans la liste des Pays !"    
  EndIf
  
  If FindMapElement(Pays(), "UK")
    Debug "'UK' est dans la liste des Pays."
  Else
    Debug "'UK' n'est PAS dans la liste des Pays !"
  EndIf
