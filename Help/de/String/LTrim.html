LTrim()

Syntax

Ergebnis$ = LTrim(String$ [, Zeichen$])
Beschreibung
Entfernt alle angegebenen Zeichen vom Anfang des Strings.

Parameter

String$ Der zu kürzende String.
Zeichen$ (optional) Das für das Kürzen des Strings zu verwendende Zeichen. Das Standardzeichen ist ein Leerzeichen ('Space'). 'Zeichen$' muss ein ein Zeichen langer String sein.

Rückgabewert

Ein neuer String, der den originalen String - mit entfernten Anfangszeichen - enthält.

Beispiel

  Debug LTrim("     This is Art    ")  ; wird "This is Art    " anzeigen
  Debug LTrim("!!Hello World!!", "!")  ; Wird "Hello World!!" anzeigen
