DesktopScaledX()

Syntax

Ergebnis = DesktopScaledX(Wert)
Beschreibung
Gibt den skalierten Wert gemäß der aktuellen DPI-Anzeige auf der x-Achse zurück. Dies ist vor allem nützlich, um die reale Pixelposition unabhängig von der angezeigten DPI zu berechnen.

Parameter

Value Der zu verwendende Wert.

Rückgabewert

Gibt den skalierten Wert gemäß der aktuellen DPI-Anzeige auf der x-Achse zurück. Beispielsweise resultiert bei einer Anzeige mit einem zugewiesenen DPI von 125% aus einem Wert von 100 im Ergebnis 125.

Anmerkungen

Die Anwendung muss mit der Option 'DPI-kompatibel' kompiliert werden, damit dieser Befehl tatsächlich einen skalierten DPI-Wert zurückgibt. Wenn nicht, ist das Ergebnis immer gleich dem 'Wert' Parameter.

Beispiel

  Debug "Desktop DPI 'x' skalierter Wert von 100: " + DesktopScaledX(100)
