FinishBoneAssignment()

Syntax

FinishBoneAssignment(#Mesh, SubMesh)
Description
Finish bone assignments for a mesh, previously started with VertexBoneAssignment().

Parameters

#Mesh The mesh to use.
SubMesh The submesh to use. The first submesh index is 0 (main mesh).

Return value

None.

See Also

VertexBoneAssignment()

Supported OS

All
