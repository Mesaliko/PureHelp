DesktopUnscaledY()

Syntax

Ergebnis = DesktopUnscaledY(Wert)
Beschreibung
Gibt den unskalierten Wert gemäß der aktuellen DPI-Anzeige auf der y-Achse zurück. Dies ist vor allem nützlich, um die reale Pixelposition unabhängig von der angezeigten DPI zu berechnen.

Parameter

Value Der zu verwendende Wert.

Rückgabewert

Gibt den unskalierten Wert gemäß der aktuellen DPI-Anzeige auf der y-Achse zurück. Beispielsweise resultiert bei einer Anzeige mit einem zugewiesenen DPI von 125% aus einem Wert von 125 im Ergebnis 100.

Anmerkungen

Die Anwendung muss mit der Option 'DPI-kompatibel' kompiliert werden, damit dieser Befehl tatsächlich einen skalierten DPI-Wert zurückgibt. Wenn nicht, ist das Ergebnis immer gleich dem 'Wert' Parameter.

Beispiel

  Debug "Desktop DPI 'y' unskalierter Wert von 125: " + DesktopUnscaledY(125)
