ConeTwistJoint()

Syntax

Ergebnis = ConeTwistJoint(#Joint, EntityID, TransformX, TransformY, TransformZ, EntityID2, TransformX2, TransformY2, TransformZ2)
Beschreibung
Erstellt eine neue Kegeldrehungs-Verbindung (englisch "cone twist joint") zwischen den zwei angegebenen Entities (Objekten). Eine Kegeldrehung kann verwendet werden, um Arme oder Beine an einem Körper zu befestigen, etc.

Parameter

#Joint Die Nummer zum Identifizieren des neuen Gelenks (Joint). #PB_Any kann verwendet werden, um diese Zahl automatisch zu generieren.
EntityID Das erste mit dem Gelenk (Joint) verknüpfte Entity. Um eine gültige EntityID zu erhalten, verwenden Sie EntityID().
TransformX, TransformY, TransformZ Die auf das erste Entity (Objekt) anzuwendende Transformation.
EntityID2 Das zweite zur Erstellung des Gelenks (Joint) verwendete Entity. Um eine gültige EntityID zu erhalten, verwenden Sie EntityID().
TransformX2, TransformY2, TransformZ2 Die auf das zweite Entity (Objekt) anzuwendende Transformation.

Rückgabewert

Gibt null zurück, wenn die Verbindung bzw. das Gelenk (Joint) nicht erstellt werden konnte. Wenn #PB_Any als Parameter '#Joint' verwendet wird, dann wird die neue Joint-Nummer als 'Ergebnis' zurückgegeben.

Anmerkungen

GetJointAttribute() und SetJointAttribute() können mit den folgenden Attributen verwendet werden, um das Verhalten der Verbindung zu ändern: 
  #PB_ConeTwistJoint_SwingSpan  : Erste Schaukel-Spannweite der Verbindung
  #PB_ConeTwistJoint_SwingSpan2 : Zweite Schaukel-Spannweite der Verbindung
  #PB_ConeTwistJoint_TwistSpan  : Dreh-Spannweite der Verbindung
