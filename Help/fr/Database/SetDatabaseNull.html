SetDatabaseNull()

Syntaxe

SetDatabaseNull(#BaseDeDonnees, IndexDeDeclaration)
Description
Définit une variable de liaison à une valeur NULL pour le prochain appel à DatabaseQuery() ou à DatabaseUpdate().

Arguments

#Database La base de données à utiliser.
IndexDeDeclaration L'indice de la variable de liaison dans la déclaration.

La première variable a l'index 0.

Valeur de retour

Aucune.

Remarques

Voir DatabaseQuery() et DatabaseUpdate() pour des exemples sur comment spécifier les variables de liaison dans une instruction SQL.

Voir aussi

SetDatabaseString(), SetDatabaseLong(), SetDatabaseQuad(), SetDatabaseFloat(), SetDatabaseDouble() SetDatabaseBlob(), DatabaseQuery(), DatabaseUpdate()

OS Supportés
