WriteCGIString()

Syntax

Ergebnis = WriteCGIString(String$ [, Codierung])
Beschreibung
Schreibt einen String in die CGI-Ausgabe.

Parameter

String$ Der zu schreibende String.
Codierung (optional) Die zu verwendende String-Codierung. Dies kann einer der folgenden Werte sein: 
  #PB_UTF8 (Standard)
  #PB_Ascii

Rückgabewert

Gibt einen Wert ungleich Null zurück, wenn der String erfolgreich in die CGI-Ausgabe geschrieben wurde.

Beispiel

  If Not InitCGI() Or Not ReadCGI()
    End
  EndIf
  
  WriteCGIHeader(#PB_CGI_HeaderContentType, "text/html", #PB_CGI_LastHeader)
  
  WriteCGIString("<html><title>PureBasic - test</title><body>" +
                 "Hello from PureBasic CGI<br>" +
                 "Actual time: <b>"+FormatDate("%hh:%ii", Date()) + "</b>" +
                 "</body></html>")
