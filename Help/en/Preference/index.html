PureBasic - Preference

Overview

Preference file contains user defined program parameters stored on the disk and read back when the program start again (like '.INI' files under Windows for example). PureBasic offers the possibility to create hierarchical preference file easily usable on any computer platform. The file format is in unicode UTF-8 with a BOM one preference per line, using the 'Keyword = Value' syntax. Groups can be created for better reading.
