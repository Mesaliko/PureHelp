RotateEntityBone()

Syntaxe

RotateEntityBone(#Entity, Os$, X, Y, Z, Mode)
Description
Fait pivoter un os d'une entité.

Arguments

#Entity L'entité à utiliser.
Os$ Le nom de l'os dans le mesh OGRE, ou tel que spécifié dans CreateBone().
X, Y, Z La valeur de rotation en X, Y, Z, entre 0 à 359 degrés.
Mode (optionnel) 
  #PB_Absolute: Rotation absolue (par défaut).
  #PB_Relative: Rotation relative basée sur la rotation précédente de l'os.

Valeur de retour

Aucune.

Remarques

L'os doit être en mode manuel, défini avec EnableManualEntityBoneControl().

Voir aussi

MoveEntityBone(), ScaleEntityBone()

OS Supportés

Tous
