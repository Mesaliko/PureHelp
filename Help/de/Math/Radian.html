Radian()

Syntax

Ergebnis.f(.d) = Radian(Winkel.f(.d))
Beschreibung
Konvertiert den angegebenen Winkel vom Winkelmaß (Grad) ins Bogenmaß (Radiant).

Parameter

Winkel.f oder Winkel.d Der Eingabe-Winkel in Grad.

Rückgabewert

Gibt den Winkel im Bogenmaß (Radiant) zurück.

Anmerkungen

Es gibt keine Angleichung, um sicher zu gehen, dass der resultierende Winkel zwischen 0 und #PI*2 liegt. Wenn die Eingabe größer als 360 war, dann wird das Ergebnis größer als #PI*2 sein. Ebenso wird eine negative Eingabe in eine negative Ausgabe resultieren.

Die umgekehrte Umwandlung kann mit der Degree() Funktion vorgenommen werden. Diese Funktion verarbeitet und gibt Float oder Double Werte zurück.

Beispiel

  Debug Radian(90) ; wird #PI/2 anzeigen
