FreeStaticGeometry()

Syntaxe

FreeStaticGeometry(#Statique)
Description
Libére une forme géométrique statique.

Arguments

#Statique La forme à libérer.

Si #PB_All est spécifié, toutes les formes géométriqyes statiques restantes sont libérées.

Valeur de retour

Aucune.

Remarques

Toutes les formes géométriques statiques restantes sont automatiquement libérées lorsque le programme se termine.
