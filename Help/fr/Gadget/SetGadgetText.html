SetGadgetText()

Syntaxe

SetGadgetText(#Gadget, Texte$)
Description
Modifie le texte contenu dans un gadget.

Arguments

#Gadget Le gadget à utiliser.
Texte$ Le nouveau texte de l'élément.

Valeur de retour

Aucune.

Remarques

Cette fonction est particulièrement utile pour :

- ButtonGadget()
- ComboBoxGadget()
- DateGadget()
- EditorGadget()
- ExplorerComboGadget()
- ExplorerListGadget()
- ExplorerTreeGadget()
- FrameGadget()
- HyperLinkGadget()
- ListViewGadget()
- StringGadget()
- TextGadget()
- TreeGadget()
- WebGadget()
- WebViewGadget()
