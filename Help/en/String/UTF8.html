UTF8()

Syntax

*Buffer = UTF8(String$)
Description
Creates a buffer with an UTF8 representation of the string. When the buffer is no longer needed, the buffer needs to be freed with FreeMemory().

Parameters

String$ The string to convert.

Return value

The UTF8 representation of the string.

Remarks

This function is mainly useful when interacting with third-party libraries which requires UTF8 as input. Pseudotype 'p-utf8' can also be used to automated the converting process when importing external functions.

The buffer includes a null-terminated character.

Example

  *UTF8 = UTF8("Hélé")
  ShowMemoryViewer(*UTF8, MemorySize(*UTF8))

  Debug PeekS(*UTF8, -1, #PB_UTF8) ; Displays "Hélé"

Example

  Macro Unicode(Mem, Type = #PB_Ascii)
    PeekS(Mem, -1, Type)
  EndMacro

  *Mem1 = Ascii("Test - éàîöÊÜ")
  *Mem2 = UTF8("Test - éàîöÊÜ")

  Text.s = Unicode(*Mem1)
  Debug Text

  Text2.s = Unicode(*Mem2, #PB_UTF8)
  Debug Text2
