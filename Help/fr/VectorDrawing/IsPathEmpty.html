IsPathEmpty()

Syntaxe

Resultat = IsPathEmpty()
Description
Teste si le chemin de dessin vectoriel en cours est vide.

Arguments

Aucun.

Valeur de retour

Renvoie une valeur non nulle si le chemin est vide, zéro sinon (le chemin contient des segments de ligne.)
