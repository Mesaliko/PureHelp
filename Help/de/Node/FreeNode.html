FreeNode()

Syntax

FreeNode(#Node)
Beschreibung
Gibt den angegebenen Knoten frei, welcher zuvor mit CreateNode() erstellt wurde. Sein gesamter zugehöriger Speicher wird freigegeben und dieses Objekt kann nicht mehr verwendet werden.

Die hinzugefügten Objekte werden nicht automatisch freigegeben, und können daher wiederverwendet werden.

Parameter

#Node Der freizugebende Knoten. Wenn #PB_All angegeben wird, dann werden alle verbliebenen Knoten freigegeben.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.

Anmerkungen

Alle verbleibenden Nodes werden automatisch freigegeben, wenn das Programm endet.
