BillboardGroupZ()

Syntax

Ergebnis = BillboardGroupZ(#BillboardGruppe [, Modus])
Beschreibung
Ermittelt die absolute Position der Billboard-Gruppe in der Welt.

Parameter

#BillboardGruppe Die Nummer der Billboard-Gruppe, deren Position ermittelt werden soll.
Modus (optional) Der Modus, nach dem die 'z' Position ermittelt werden soll. Dies kann einer der folgenden Werte sein: 
  #PB_Absolute: ermittelt die absolute 'z' Position der Billboard-Gruppe in der Welt (Standard).
  #PB_Relative: ermittelt die 'z' Position der Billboard-Gruppe relativ zu ihrer übergeordneten (parent) Gruppe.

Rückgabewert

Die absolute Z-Position der Billboard-Gruppe.

Unterstützte OS

Alle

<- BillboardGroupY() - Billboard Inhaltsverzeichnis - BillboardHeight() ->
