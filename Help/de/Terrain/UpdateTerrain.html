UpdateTerrain()

Syntax

UpdateTerrain(#Terrain)
Beschreibung
Aktualisiert das Terrain. Dies ist nötig, nachdem das Terrain mit Befehlen wie SetTerrainTileHeightAtPoint() verändert wurde.

Parameter

#Terrain Das zu aktualisierende Terrain.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.

Siehe auch

CreateTerrain(), SetTerrainTileHeightAtPoint()

Unterstützte OS

Alle

<- TerrainTileSize() - Terrain Inhaltsverzeichnis - UpdateTerrainTileLayerBlend() ->
