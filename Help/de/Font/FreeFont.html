FreeFont()

Syntax

FreeFont(#Font)
Beschreibung
Gibt den angegebenen Zeichensatz frei, der zuvor mittels LoadFont() geladen wurde.

Parameter

#Font Der freizugebende Zeichensatz. Wenn #PB_All angegeben wird, dann werden alle verbliebenen Zeichensätze freigegeben.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.

Anmerkungen

Alle verbleibenden Zeichensätze werden automatisch freigegeben, wenn das Programm endet.
