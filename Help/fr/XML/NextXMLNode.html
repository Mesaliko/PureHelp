NextXMLNode()

Syntaxe

Resultat = NextXMLNode(Noeud)
Description
Renvoie le noeud suivant de même niveau.

Arguments

Noeud Le noeud XML à utiliser.

Valeur de retour

Renvoie le noeud suivant, ou zéro s'il n'y a plus de noeud après celui-ci.
