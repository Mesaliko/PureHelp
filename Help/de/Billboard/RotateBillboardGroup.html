RotateBillboardGroup()

Syntax

RotateBillboardGroup(#BillboardGruppe, x, y, z [, Modus])
Beschreibung
Rotiert eine Billboard-Gruppe entsprechend der angegebenen x, y, z Winkel-Angaben.

Parameter

#BillboardGruppe Die Billboard-Gruppe, welche rotiert (gedreht) werden soll.
x, y, z Die neue Drehung, welche auf die Billboard-Gruppe angewandt werden soll. Alle Winkel werden in Grad angegeben, mit Werten von 0 bis 360.
Modus (optional) Dies kann einer der folgende Werte sein: 
  PB_Absolute: absolute Rotation (Standard).
  PB_Relative: relative Rotation, basierend auf der vorherigen "Billboard"-Rotation.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.

Unterstützte OS

Alle

<- ResizeBillboard() - Billboard Inhaltsverzeichnis
