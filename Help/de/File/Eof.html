Eof()

Syntax

Ergebnis = Eof(#Datei)
Beschreibung
Überprüft, ob das Ende einer Datei erreicht wurde. EOF steht dabei für 'End Of File' (Ende der Datei).

Parameter

#Datei Die zu verwendende Datei.

Rückgabewert

Gibt einen Wert ungleich Null zurück, wenn der Lese-Zeiger am Ende der Datei angelangt ist, oder Null wenn nicht.

Anmerkungen

Für ein Beispiel siehe die ReadFile() Funktion.
