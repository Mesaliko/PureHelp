UseDialogScintillaGadget()

Syntax

UseDialogScintillaGadget()
Description
Enable ScintillaGadget() support in the dialog library. This is not enabled by default to reduce the dialog library size footprint if ScintillaGadget() is not needed.

Return value

None.

See Also

OpenXMLDialog()

Supported OS

All
