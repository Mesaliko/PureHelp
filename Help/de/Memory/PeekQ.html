PeekQ()

Syntax

Wert.q = PeekQ(*Speicherpuffer)
Beschreibung
Liest einen Quad-Wert (8 Bytes) von der angegebenen Speicheradresse.

Parameter

*Speicherpuffer Die Adresse, von der gelesen werden soll.

Rückgabewert

Gibt den Wert des Quad-Werts zurück.
