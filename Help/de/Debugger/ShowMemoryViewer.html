ShowMemoryViewer()

Syntax

ShowMemoryViewer([*Puffer, Länge])
Beschreibung
Öffnet das Speicheranzeige-Fenster oder bringt es in den Vordergrund, wenn es bereits offen ist.

Parameter

*Puffer, Länge (optional) Der in der Speicheranzeige darzustellende Speicherbereich. Wenn diese Parameter nicht angegeben werden, dann wird die Speicheranzeige einfach nur geöffnet, ohne einen bestimmten Speicherbereich anzuzeigen.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.

Beispiel

  *Memory = AllocateMemory(1000)
  If *Memory
    RandomData(*Memory, 1000)       ; den Speicher mit einigen Daten füllen
  
    ShowMemoryViewer(*Memory, 1000) ; die Speicheranzeige öffnen
    CallDebugger     ; das Programm anhalten, damit es nicht sofort beendet wird
  EndIf
