PureBasic - Camera

Einleitung

Kameras (englisch "camera") werden zum Anzeigen der 3D-Welt benutzt. Sie können diese wie lebensechte Kameras verwenden, d.h. Sie können sie drehen, bewegen, das Sichtfeld ändern und einiges mehr. Mindestens eine Kamera wird benötigt, um die Welt auf dem Bildschirm darzustellen ("zu rendern"). Viele Kameras mit verschiedenen Positionen und Ansichten können zur gleichen Zeit benutzt werden, was coole Effekte wie Splitscreen (geteilter Bildschirm, für mehrere Spieler), Rückblick etc. ermöglicht. InitEngine3D() muss erfolgreich aufgerufen worden sein, bevor die Kamera-Befehle benutzt werden.
