CloseNetworkServer()

Syntax

CloseNetworkServer(#Server)
Beschreibung
Schließt den angegebenen laufenden Server. Alle mit diesem Server verbundenen Clients werden automatisch entfernt. Der Port wird freigegeben und kann erneut verwendet werden.

Parameter

#Server Der zu schließende Server.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.
