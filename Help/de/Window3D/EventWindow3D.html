EventWindow3D()

Syntax

Ergebnis = EventWindow3D()
Beschreibung
Nach einem WindowEvent3D() Befehl kann diese Funktion verwendet werden, um das Fenster zu bestimmen, in dem ein Ereignis stattfand.

Parameter

Diese Funktion hat keine Parameter.

Rückgabewert

Die mit dem letzten Ereignis verknüpfte 3D-Window-Nummer.

Unterstützte OS

Alle

<- EventType3D() - Window3D Inhaltsverzeichnis - GetActiveWindow3D() ->
