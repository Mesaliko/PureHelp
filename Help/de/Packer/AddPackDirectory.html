AddPackDirectory()

Syntax

Ergebnis = AddPackDirectory(#Pack, VerzeichnisName$)
Beschreibung
Fügt dem angegebenen Paket, das zuvor mit CreatePack() erstellt wurde, ein neues leeres Verzeichnis hinzu.

Parameter

#Pack Das zu verwendende Paket.
DirectoryName$ Der Name des leeren Verzeichnisses, das dem Paket hinzugefügt werden soll.

Rückgabewert

Ein Wert ungleich Null, wenn das leere Verzeichnis erfolgreich zur Paketdatei hinzugefügt wurde.
