Cos()

Syntax

Result.f(.d) = Cos(Angle.f(.d))
Description
Returns the cosine of the specified angle.

Parameters

Angle.f or Angle.d The input angle in radians. Use Radian() to convert an angle from degrees to radian.

Return value

Returns the cosine of the angle. The result is always between -1.0 and 1.0.

Remarks

The inverse function of Cos() is ACos(). This function handles and returns float or double values.

Example

  Debug Cos(3.141593)

See Also

ACos(), CosH(), Radian()

Supported OS

All
