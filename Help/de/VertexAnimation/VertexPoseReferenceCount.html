VertexPoseReferenceCount()

Syntax

Ergebnis = VertexPoseReferenceCount(#Mesh, Animation$, Track, Keyframe)
Beschreibung
Gibt die Anzahl von Posen-Bezügen (oder auch Referenzen auf Posen) im angegebenen Keyframe zurück. Die Animation sollte bereits mittels CreateVertexAnimation() erstellt worden oder im Mesh vordefiniert sein.

Parameter

#Mesh Das zu verwendende Mesh.
Animation$ Der Animations-Name. Dieser Name beachtet die Groß- und Kleinschreibung. Die Animation sollte bereits mittels CreateVertexAnimation() erstellt worden oder im Mesh vordefiniert sein.
Track Der Track-Index. Der Track (oder auch Spur, Strecke) muss zuvor mit CreateVertexTrack() erstellt worden sein.
Keyframe Der Keyframe-Index. Das Keyframe muss zuvor mittels CreateVertexPoseKeyFrame() erstellt worden sein.

Rückgabewert

Gibt die Anzahl von Posen-Bezügen im angegebenen Keyframe zurück.
