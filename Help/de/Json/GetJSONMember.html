GetJSONMember()

Syntax

Ergebnis = GetJSONMember(JSONWert, Schlüssel$)
Beschreibung
Gibt das JSON-Objektelement mit dem angegebenen Schlüssel$ von einem JSON-Wert des Typs #PB_JSON_Object zurück.

Parameter

JSONWert Der JSON-Wert. Der Wert muss vom Typ #PB_JSON_Object sein.
Schlüssel$ Der Schlüssel ("Key") des zurückzugebenen Elements (Mitglieds).

Der Schlüssel wird unter Beachtung der Groß- und Kleinschreibung verglichen, außer es wurde das #PB_JSON_NoCase Flag beim Erstellen oder Parsen der JSON-Daten angegeben.

Rückgabewert

Die Adresse des JSON-Werts mit dem angegebenen Schlüssel. Wenn der angegebene 'Schlüssel$' im Objekt nicht existiert, wird das Ergebnis 0 sein.

Beispiel

  Input$ = "{ " + Chr(34) + "x" + Chr(34) + ": 10, " + 
                  Chr(34) + "y" + Chr(34) + ": 20, " + 
                  Chr(34) + "z" + Chr(34) + ": 30 }"
  
  ParseJSON(0, Input$)
  
  Debug GetJSONInteger(GetJSONMember(JSONValue(0), "x"))
  Debug GetJSONInteger(GetJSONMember(JSONValue(0), "y"))
  Debug GetJSONInteger(GetJSONMember(JSONValue(0), "z"))
