ApplyEntityTorqueImpulse()

Syntax

ApplyEntityTorqueImpulse(#Entity, x, y, z [, Modus])
Beschreibung
Wendet einen Drehimpuls (englisch "rotation impulse") auf das Entity an. Der neue Impuls wird zusätzlich zur Drehkraft angewandt, die vorher auf das Entity angewendet wurde.

Parameter

#Entity Das zu verwendende Entity-Objekt.
x, y, z Die Dreh-Impuls-Werte.
Modus (optional) Der anzuwendende Drehimpuls-Modus. Dies kann einer der folgenden Werte sein: 
  #PB_Local : lokaler Drehimpuls.
  #PB_Parent: Drehimpuls relativ zur übergeordneten ("parent") Position.
  #PB_World : Drehimpuls relativ zur Welt.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.
