IsToolBar()

Syntax

Ergebnis = IsToolBar(#ToolBar)
Beschreibung
Überprüft, ob die angegebene #ToolBar Nummer gültig und eine korrekt initialisierte Werkzeugleiste ist.

Parameter

#ToolBar Die zu verwendende Werkzeugleiste.

Rückgabewert

Gibt einen Wert ungleich Null zurück, wenn #ToolBar eine gültige Werkzeugleiste ist, andernfalls Null.

Anmerkungen

Diese Funktion ist "kugelsicher" und kann mit jedem Wert benutzt werden. Ist das 'Ergebnis' ungleich Null, dann ist das Objekt gültig und initialisiert, andernfalls wird Null zurückgegeben. Dies ist der richtige Weg, um sicher zu stellen, dass eine ToolBar bereit zur Verwendung ist.
