CopyMesh()

Syntax

Ergebnis = CopyMesh(#Mesh, #NewMesh)
Beschreibung
Erstellt ein neues Objekt '#NewMesh', welches eine exakte Kopie des angegebenen '#Mesh' ist. Wenn #PB_Any als '#NewMesh' Parameter verwendet wird, dann wird die Nummer des neuen Mesh als 'Ergebnis' zurückgegeben.

Ist das 'Ergebnis' gleich 0, ist das Kopieren des Mesh's fehlgeschlagen. Wenn das neue Objekt '#NewMesh' bereits existiert, wird dieses automatisch freigegeben und durch das neue ersetzt.
