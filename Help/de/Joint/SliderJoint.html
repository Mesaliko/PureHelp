SliderJoint()

Syntax

Ergebnis = SliderJoint(#Joint, EntityID, TransformX, TransformY, TransformZ, EntityID2, TransformX2, TransformY2, TransformZ2)
Beschreibung
Erstellt eine neue Schieberegler-Verbindung (englisch "slider joint") zwischen den beiden angegebenen Entities (Objekten). Ein "Slider" kann verwendet werden, um ein Entity mit Einschränkungen auf einer ebenen Fläche etc. zu bewegen.

Parameter

#Joint Die Nummer zum Identifizieren des neuen Gelenks (Joint). #PB_Any kann verwendet werden, um diese Zahl automatisch zu generieren.
EntityID Das erste mit der Verbindung (Joint) verknüpfte Entity. Um eine gültige EntityID zu erhalten, verwenden Sie EntityID().
TransformX Der X-Wert der Transformation für das erste Entity (Objekt).
TransformY Der Y-Wert der Transformation für das erste Entity.
TransformZ Der Z-Wert der Transformation für das erste Entity.
EntityID2 Das zweite zur Erstellung der Verbindung (Joint) verwendete Entity. Um eine gültige EntityID zu erhalten, verwenden Sie EntityID().
TransformX2 Der X-Wert der Transformation für das zweite Entity (Objekt).
TransformY2 Der Y-Wert der Transformation für das zweite Entity.
TransformZ2 Der Z-Wert der Transformation für das zweite Entity.

Rückgabewert

Gibt null zurück, wenn die Verbindung bzw. das Gelenk (Joint) nicht erstellt werden konnte. Wenn #PB_Any als Parameter '#Joint' verwendet wird, dann wird die neue Joint-Nummer als 'Ergebnis' zurückgegeben.

Anmerkungen

GetJointAttribute() und SetJointAttribute() können mit den folgenden Attributen verwendet werden, um das Verhalten der Verbindung zu ändern: 
  #PB_SliderJoint_LowerLimit : Untere Grenze der Verbindung
  #PB_SliderJoint_UpperLimit : Obere Grenze der Verbindung
