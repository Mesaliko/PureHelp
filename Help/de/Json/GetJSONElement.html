GetJSONElement()

Syntax

Ergebnis = GetJSONElement(JSONWert, Index)
Beschreibung
Gibt das JSON-Arrayelement am angegebenen 'Index' von einem JSON-Wert des Typs #PB_JSON_Array zurück.

Parameter

JSONWert Der JSON-Wert. Der Wert muss vom Typ #PB_JSON_Array sein.
Index Der Index des zurückzugebenden Array-Elements. Der Index muss zwischen 0 und JSONArraySize() - 1 liegen.

Rückgabewert

Die Adresse des JSON-Werts am angegebenen Array-Index. Liegt der angegebene 'Index' außerhalb des gültigen Bereichs, wird das Ergebnis 0 sein.

Beispiel

  ParseJSON(0, "[1, 2, 3, 4, 5]")
    
  For i = 0 To JSONArraySize(JSONValue(0)) - 1
    Debug GetJSONInteger(GetJSONElement(JSONValue(0), i))
  Next i
