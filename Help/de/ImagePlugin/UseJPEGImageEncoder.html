UseJPEGImageEncoder()

Syntax

UseJPEGImageEncoder()
Beschreibung
Aktiviert die Unterstützung für JPEG (Join Picture Expert Group) Bilder bei den SaveImage() und SaveSprite() Funktionen.

Parameter

Diese Funktion hat keine Parameter.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.

Anmerkungen

Das JPEG-Format verwendet eine destruktive (verlustbehaftete) Kompression, was bedeutet, dass ein Bild beim Komprimieren einen bestimmten Teil seiner Originalinformationen verliert. Dieser Kompressions-Algorithmus ist einer der besten verfügbaren, welcher die Bildgröße - ohne eine Menge von sichtbaren Artefakten zu produzieren - dramatisch vermindern kann.
