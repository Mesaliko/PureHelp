ChildXMLNode()

Syntaxe

Resultat = ChildXMLNode(Noeud [, Index])
Description
Renvoie un noeud fils.

Arguments

Noeud Le noeud XML qui contient le noeud fils.
Index (optionnel) L'index du noeud fils.

Le premier index commence à 1.
S'il est omis alors le premier noeud est renvoyé.

Valeur de retour

Renvoie le noeud fils indiqué, ou zéro s'il n'existe pas.
