ErrorAddress()

Syntaxe

Resultat = ErrorAddress()
Description
Renvoie l'adresse mémoire de l'instruction assembleur qui a causé l'erreur.

Arguments

Aucun.

Valeur de retour

Renvoie l'adresse mémoire de l'instruction assembleur qui a causé l'erreur.

Cette commande renvoie une valeur valide uniquement si l'erreur est gérée par OnErrorCall() ou OnErrorGoto().

OS Supportés

Tous
