FreeDialog()

Syntax

FreeDialog(#Dialog)
Beschreibung
Gibt den angegebenen Dialog und seinen zugehörigen Speicher frei. Wenn das Dialog-Fenster noch geöffnet ist, wird es automatisch geschlossen.

Parameter

#Dialog Der freizugebende Dialog. Wenn #PB_All angegeben wird, dann werden alle verbliebenen Dialoge freigegeben.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.

Anmerkungen

Alle verbliebenen Dialoge werden automatisch freigegeben, wenn das Programm endet.
