PackEntryDate()

Syntax

Ergebnis.q = PackEntryDate(#Pack [, DatumsTyp])
Beschreibung
Gibt das Dateidatum des aktuellen Paket-Eintrags zurück, festgelegt mit NextPackEntry().

Parameter

#Pack Das zu verwendende Paket.
DatumsTyp Der Typ des zurückzugebenden Datums. Dies kann einer der folgenden Werte sein: 
  #PB_Date_Created : gibt das Erstellungsdatum der Datei zurück.
  #PB_Date_Accessed: gibt das Datum des letzten Dateizugriffs zurück.
  #PB_Date_Modified: gibt das Datum der letzten Dateiänderung zurück.

Rückgabewert

Gibt das Dateidatum oder -1 zurück, wenn kein Datum verfügbar ist.
