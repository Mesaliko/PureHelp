SelectedFontName()

Syntax

Name$ = SelectedFontName()
Beschreibung
Gibt den Namen des vom Benutzer im FontRequester() ausgewählten Zeichensatzes zurück.

Parameter

Diese Funktion hat keine Parameter.

Rückgabewert

Der Name des vom Benutzer im FontRequester() ausgewählten Zeichensatzes. Dieser Name kann direkt vom LoadFont() Befehl verwendet werden.
