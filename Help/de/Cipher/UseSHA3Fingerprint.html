UseSHA3Fingerprint()

Syntax

UseSHA3Fingerprint()
Beschreibung
Registriert das SHA3 Fingerprint Plugin für die weitere Nutzung. Die standardmäßigen 224-Bit, 256-Bit, 384-Bit und 512-Bit Varianten werden unterstützt.

Parameter

Diese Funktion hat keine Parameter.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.

Beispiel

  UseSHA3Fingerprint()

  Debug StringFingerprint("yourpassword", #PB_Cipher_SHA3, 512) ; verwende die SHA3-512 Variante
