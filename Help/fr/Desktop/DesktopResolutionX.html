DesktopResolutionX()

Syntaxe

Resultat.d = DesktopResolutionX()
Description
Renvoie le facteur de résolution DPI horizontal du bureau.

Arguments

Aucun.

Valeur de retour

Renvoie le facteur de résolution DPI du bureau sur l'axe 'x'.
Si la valeur est '1', aucun facteur DPI n'a été appliqué sur l'axe 'x'.
Si la valeur est '1.25', un facteur de 125% a été appliqué sur l'axe 'x'.

Remarques

L'application doit être compilée avec le commutateur 'DPI' pour que cette commande renvoie le facteur de résolution DPI réel. Sinon, le résultat sera toujours '1'.

Exemple

  Debug "DPI du bureau: Facteur d'échelle d'affichage horizontal : " + DesktopResolutionX()
