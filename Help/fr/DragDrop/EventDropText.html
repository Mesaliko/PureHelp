EventDropText()

Syntaxe

Resultat$ = EventDropText()
Description
Renvoie le texte déposé.

Arguments

Aucun.

Valeur de retour

Renvoie le texte déposé après avoir reçu #PB_Event_GadgetDrop ou #PB_Event_WindowDrop avec WaitWindowEvent() ou WindowEvent() et un format (récupérable avec EventDropType()) de type #PB_Drop_Text.

Voir aussi

EnableGadgetDrop(), EnableWindowDrop(), EventDropType(), EventDropAction(), EventDropX(), EventDropY()

OS Supportés

Tous
