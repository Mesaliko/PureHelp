CameraFollow()

Syntax

CameraFollow(#Kamera, ObjektID, Winkel, Höhe, Distanz, RotationProzent, PositionProzent [, Modus])
Beschreibung
Foglt dem angegebenen Objekt auf sanfte Weise, durch Verwendung von Interpolation.

Parameter

#Kamera Die zu verwendende Kamera.
ObjektID Das zu verfolgende Objekt. Dies kann einer der folgenden Typen sein: 
  - Entity         : verwenden Sie EntityID() zur Ermittlung einer gültigen ID.
  - Light          : verwenden Sie LightID() zur Ermittlung einer gültigen ID.
  - Node           : verwenden Sie NodeID() zur Ermittlung einer gültigen ID.
  - ParticleEmitter: verwenden Sie ParticleEmitterID() zur Ermittlung einer gültigen ID.
  - BillboardGroup : verwenden Sie BillboardGroupID() zur Ermittlung einer gültigen ID.
  - Text3D         : verwenden Sie Text3DID() zur Ermittlung einer gültigen ID.
Winkel Der Winkel der Kamera, relativ zum verfolgten Objekt.
Höhe Die absolute Höhe der Kamera.
Distanz Die Entfernung der Kamera relativ zum verfolgten Objekt.
RotationProzent Anzuwendender Wert, wenn sich die Kamera dreht, um diese wieder im richtigen Winkel zu haben. Gültige Werte reichen von 0 bis 1.
PositionProzent Anzuwendender Wert, wenn sich die Kamera bewegt, um diese wieder an der richtigen Position zu haben. Gültige Werte reichen von 0 bis 1. Wenn der Wert 0 lautet, dann bewegt sich die Kamera nicht. Wenn der Wert 1 beträgt, dann wird die Kamera an die endgültige Position gesetzt, ohne Interpolation.
Modus (optional) Dies kann einer der folgenden Werte sein: 
  #True : die Kamera schaut automatisch auf das verfolgte Entity-Objekt (Standard).
  #False: die Kamera schaut nicht automatisch auf das verfolgte Objekt.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.
