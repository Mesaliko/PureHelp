GetHomeDirectory()

Syntax

Ergebnis$ = GetHomeDirectory()
Beschreibung
Gibt den 'Home'-Verzeichnispfad (Anwender-bezogen) des aktuell eingeloggten Anwenders zurück.

Parameter

Diese Funktion hat keine Parameter.

Rückgabewert

Gibt den vollständigen Pfad des 'Home'-Verzeichnisses zurück. Dieser wird mit einem Verzeichnis-Trennzeichen #PS, #PS$ ('\' auf Windows, ansonsten '/') enden.

Es ist sehr unwahrscheinlich, aber wenn diese Funktion fehlschlägt, wird ein leerer String zurückgegeben.

Anmerkungen

Das 'Home'-Verzeichnis ist das Verzeichnis für die Anwender-Daten (Voreinstellungen, Plugins, etc.) des aktuellen Anwenders. Lese- und Schreib-Zugriff sollte in diesem Verzeichnis möglich sein.
