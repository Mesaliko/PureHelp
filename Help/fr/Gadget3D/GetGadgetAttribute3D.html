GetGadgetAttribute3D()

Syntaxe

Resultat = GetGadgetAttribute3D(#Gadget3D, Attribut)
Description
Renvoie la valeur de l'attribut d'un gadget 3D.

Arguments

#Gadget3D Le gadget 3D à utiliser.
Attribut L'attribut à tester.

Valeur de retour

Cette fonction est disponible avec les gadgets suivants:

- EditorGadget3D()
- PanelGadget3D()
- ProgressBarGadget3D()
- ScrollAreaGadget3D()
- ScrollBarGadget3D()
- SpinGadget3D()

OS Supportés

Tous

<- GetActiveGadget3D() - Gadget3D Index - GetGadgetData3D() ->
