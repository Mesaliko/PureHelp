InsertJSONMap()

Syntax

InsertJSONMap(JSONWert, Map())
Beschreibung
Fügt die angegebene Map() in den angegebenen JSON-Wert sein. Der JSON-Wert wird auf den Typ #PB_JSON_Object geändert.

Parameter

JSONWert Der JSON-Wert. Der vorherige Inhalt des Werts wird auf den Inhalt der Map geändert.
Map() Die in den JSON-Wert einzufügende Map.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.

Beispiel

  NewMap Colors()
  Colors("red")   = $0000FF
  Colors("green") = $00FF00
  Colors("blue")  = $FF0000
  
  If CreateJSON(0)
    InsertJSONMap(JSONValue(0), Colors())
    Debug ComposeJSON(0, #PB_JSON_PrettyPrint)
  EndIf
