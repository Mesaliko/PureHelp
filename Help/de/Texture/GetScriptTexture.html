GetScriptTexture()

Syntax

Ergebnis = GetScriptTexture(#Textur, Name$)
Beschreibung
Ermittelt eine Textur, die in einer OGRE-Skriptdatei definiert wurde. Skripte werden geladen und geparst, wenn Parse3DScripts() aufgerufen wird.

Parameter

#Textur Eine Nummer zum Identifizieren der neuen Textur. #PB_Any kann verwendet werden, um diese Zahl automatisch zu generieren.
Name$ Der Name der Textur in den Skript-Dateien.

Rückgabewert

Ein Wert ungleich Null, wenn die Textur erfolgreich erstellt wurde, andernfalls Null. Wenn #PB_Any als #Textur Parameter verwendet wurde, dann wird im Erfolgsfall die generierte Nummer zurückgegeben.

Unterstützte OS

Alle

<- FreeTexture() - Texture Inhaltsverzeichnis - IsTexture() ->
