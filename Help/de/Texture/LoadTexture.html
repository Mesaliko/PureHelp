LoadTexture()

Syntax

Ergebnis = LoadTexture(#Textur, Dateiname$)
Beschreibung
Lädt eine neue Textur von Disk. Vor dem Laden einer Textur muss ein Archiv mittels Add3DArchive() definiert werden.

Das Textur-Format kann als PNG, TGA oder JPG vorliegen. Es ist äußerst empfehlenswert, die Textur als Quadrat mit Breite/Höhe als Vielfaches von 2 zu erstellen: 64x64, 128x128, 256x256 ... Ältere Grafikkarten können Einschränkungen haben, begrenzen Sie daher die Textur-Größe möglichst auf 256x256 Pixel.

Parameter

#Textur Eine Nummer zum Identifizieren der neuen Textur. #PB_Any kann verwendet werden, um diese Zahl automatisch zu generieren.
Dateiname$ Der Dateiname der Textur.

Rückgabewert

Ein Wert ungleich Null, wenn die Textur erfolgreich geladen wurde, andernfalls Null. Wenn #PB_Any als #Textur Parameter verwendet wurde, dann wird im Erfolgsfall die generierte Nummer zurückgegeben.
