RemoveEnvironmentVariable()

Syntaxe

RemoveEnvironmentVariable(Nom$)
Description
Supprime la variable d'environnement spécifiée du bloc d'environnement du programme.

Arguments

Nom$ La variable d'environnement à supprimer.

Valeur de retour

Aucune.

OS Supportés

Tous
