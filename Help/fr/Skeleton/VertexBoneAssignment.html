VertexBoneAssignment()

Syntaxe

VertexBoneAssignment(#Mesh, SousMesh, VertexIndex, OsIndex, Poids.f)
Description
Attribue un sommet à un os.

Arguments

#Mesh Le mesh à utiliser.
SousMesh Le sous-mesh à partir duquel obtenir les données.
Le premier indice du sous-mesh est 0 (mesh principal).
VertexIndex L'indice du sommet.
Le premier indice est 0.
OsIndex L'index de l'os, dans l'ordre de création.
Le premier indice est 0.
Poids Le poids de l'os doit être compris entre 0 et 1.

Valeur de retour

Aucune.

Remarques

Un même sommet peut être attribué à plusieurs os : la somme des poids des os doit être égal à 1.
Une fois l'affectation des sommets terminée, FinishBoneAssignment() doit être appelé.
