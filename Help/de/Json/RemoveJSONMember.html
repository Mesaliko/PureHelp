RemoveJSONMember()

Syntax

RemoveJSONMember(JSONWert, Schlüssel$)
Beschreibung
Entfernt ein Element (Mitglied) mit dem angegebenen Schlüssel von einem JSON-Wert des Typs #PB_JSON_Object.

Parameter

JSONWert Der JSON-Wert. Der Wert muss vom Typ #PB_JSON_Object sein.
Schlüssel$ Der Schlüssel ("Key") des zu entfernenden Elements.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.

Beispiel

  Input$ = "{ " + Chr(34) + "x" + Chr(34) + ": 10, " + 
                  Chr(34) + "y" + Chr(34) + ": 20, " + 
                  Chr(34) + "z" + Chr(34) + ": 30 }"
  
  ParseJSON(0, Input$)
  RemoveJSONMember(JSONValue(0), "x")
  Debug ComposeJSON(0, #PB_JSON_PrettyPrint)
