NextDraggedItem()

Syntax

Ergebnis = NextDraggedItem()
Beschreibung
Diese Funktion muss nach ExamineDraggedItems() aufgerufen werden. Sie durchläuft die gezogenen Elemente.

Rückgabewert

Gibt einen Wert ungleich Null zurück, wenn ein weiteres gezogenes Element verfügbar ist, oder Null, wenn kein weiteres gezogenes Element vorhanden ist.
