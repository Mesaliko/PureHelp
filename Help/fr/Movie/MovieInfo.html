MovieInfo()

Syntaxe

Resultat = MovieInfo(#Video, Options)
Description
Renvoie des informations additionnelles concernant une vidéo.

Arguments

#Video La vidéo à utiliser.
Options Valeur supportée actuellement: 
  0: Renvoie le nombre de trames par seconde (*1000).

Valeur de retour

Renvoie la valeur spécifiée en fonction du paramètre 'Options'.
