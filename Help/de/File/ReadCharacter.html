ReadCharacter()

Syntax

Zahl.c = ReadCharacter(#Datei [, Format])
Beschreibung
Liest ein Zeichen aus einer Datei, beginnend an der aktuellen Dateiposition.

Parameter

#Datei Die Datei, aus der gelesen werden soll.
Format (optional) Das Format des zu lesenden Zeichens (englisch "Character"). Dies kann einer der folgenden Werte sein: 
  #PB_Ascii  : 1 Byte Zeichen.
  #PB_Unicode: 2 Bytes Zeichen (Standard im Unicode-Modus).
  #PB_UTF8   : Multi-Bytes Zeichen (von 1 bis zu 4 Bytes).
Wenn dieses Flag nicht gesetzt wird, dann hängt das Format für das Lesen des Zeichens von der entsprechenden Einstellung des zuvor verwendeten CreateFile(), OpenFile() bzw. ReadFile() Befehls ab.

Rückgabewert

Gibt das gelesene Zeichen zurück, oder Null bei einem Fehler.

Anmerkungen

Für ein Beispiel, wie aus einer Datei gelesen wird, siehe den ReadFile() Befehl - mit ReadCharacter() wird einfach ein Zeichen (Character) anstelle einer kompletten Zeile (String) gelesen.
