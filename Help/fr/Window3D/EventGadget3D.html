EventGadget3D()

Syntaxe

Resultat = EventGadget3D()
Description
Utiliser cette fonction après un évènement de type #PB_Event3D_Gadget (renvoyé par WindowEvent3D()) pour connaitre le numéro du gadget qui a été utilisé.

Arguments

Aucun.

Valeur de retour

Le numéro du gadget 3D qui a été utilisé.
