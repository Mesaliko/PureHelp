SetActiveWindow()

Syntax

SetActiveWindow(#Window)
Beschreibung
Aktiviert das angegebene Fenster. Dies bedeutet, dass der Fokus auf dieses Fenster gesetzt wird.

Parameter

#Window Das zu aktivierende Fenster.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.

Anmerkungen

Der Befehl wird nur den Fokus innerhalb Ihres Programms verändern. Sie können damit nicht das Programm in den Vordergrund bringen, wenn ein anderes Programm den Fokus hat.
