Mid()

Syntax

Ergebnis$ = Mid(String$, StartPosition [, Länge])
Beschreibung
Ermittelt einen String der angegebenen Länge aus dem vorgegebenen String.

Parameter

String$ Der zu verwendende String.
StartPosition Gibt die Zeichenposition an, bei der das Extrahieren starten soll. Die Position des ersten Zeichens lautet 1.
Länge (optional) Gibt an, wieviele Zeichen ausgeschnitten werden sollen. Wird dieser Parameter weggelassen, werden alle Zeichen bis zum Ende des Strings extrahiert.

Rückgabewert

Ein neuer String, der die ausgeschnittenen Zeichen beinhaltet.

Beispiel

  Debug Mid("Hello", 2)    ; wird "ello" anzeigen
  Debug Mid("Hello", 2, 1) ; wird "e" anzeigen

Unterstützte OS

Alle

<- Len() - String Inhaltsverzeichnis - RSet() ->
