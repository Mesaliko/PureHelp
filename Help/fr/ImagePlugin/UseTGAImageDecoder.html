UseTGAImageDecoder()

Syntaxe

UseTGAImageDecoder()
Description
Active le support du format TGA (Targa) pour les commandes CatchImage(), LoadImage(), CatchSprite() et LoadSprite().

Arguments

Aucun.

Valeur de retour

Aucune.

Remarques

Le format TGA utilise un algorithme de compression non-destructif, ce qui veut dire que l'intégralité de l'information est conservée. La taille des images TGA est souvent très élévée car l'algorithme de compression (quand il est utilisé) est peu performant (RLE).

Si une couche alpha (transparence) est présente dans l'image, elle ne sera pas prise en compte (pour l'instant).
