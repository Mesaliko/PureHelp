ErrorRegister()

Syntaxe

Resultat = ErrorRegister(Registre)
Description
Renvoie le contenu du registre spécifié du microprocesseur lorsque l'erreur est survenue.

Arguments

Registre Le registre à afficher.

Les registres disponibles dépendent du CPU pour lequel le programme est compilé. Les valeurs suivantes sont disponibles:

x86: 
  #PB_OnError_EAX
  #PB_OnError_EBX
  #PB_OnError_ECX
  #PB_OnError_EDX
  #PB_OnError_EBP
  #PB_OnError_ESI
  #PB_OnError_EDI
  #PB_OnError_ESP
  #PB_OnError_Flags
x64: 
  #PB_OnError_RAX
  #PB_OnError_RCX
  #PB_OnError_RDX
  #PB_OnError_RBX
  #PB_OnError_RSP
  #PB_OnError_RBP
  #PB_OnError_RSI
  #PB_OnError_RDI
  #PB_OnError_R8
  #PB_OnError_R9
  ...
  #PB_OnError_R15
  #PB_OnError_Flags

Valeur de retour

Renvoie le contenu du registre spécifié du microprocesseur lorsque l'erreur est survenue.

Cette commande renvoie une valeur valide uniquement si l'erreur est gérée par OnErrorCall() ou OnErrorGoto().

OS Supportés

Tous
