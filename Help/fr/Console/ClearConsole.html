ClearConsole()

Syntaxe

ClearConsole()
Description
Efface tout le contenu de la console.

Arguments

Aucun.

Valeur de retour

Aucune.

Remarques

Le fond d'écran est déterminé par ConsoleColor().
La console doit être en mode graphique.

Exemple

  If OpenConsole()
    EnableGraphicalConsole(1)
  
    PrintN("Vous ne me verrez jamais")
    ClearConsole()
    
    PrintN("Appuyez sur [Entree] pour quitter")
    Input()
  EndIf
