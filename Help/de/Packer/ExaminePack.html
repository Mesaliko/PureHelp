ExaminePack()

Syntax

Ergebnis = ExaminePack(#Pack)
Beschreibung
Beginnt den Paket-Inhalt zu untersuchen. NextPackEntry() muss aufgerufen werden, um die in der Paket-Datei gefundenen Einträge aufzulisten. Das Paket muss zuvor mit OpenPack() oder CatchPack() geöffnet worden sein. Pakete, welche gerade mit CreatePack() erstellt wurden, können nicht untersucht werden.

Parameter

#Pack Das zu verwendende Paket.

Rückgabewert

Gibt einen Wert ungleich Null zurück, wenn das Paket untersucht werden kann, andernfalls Null.

Beispiel

  UseZipPacker()

  ; Öffnet die gepackte Datei
  If OpenPack(0, "mycompressedfiles.zip") 
    
    ; Listet alle Einträge auf
    If ExaminePack(0)
      While NextPackEntry(0)
        Debug "Name: " + PackEntryName(0) + ", Size: " + PackEntrySize(0)
      Wend
    EndIf
    
    ClosePack(0)
  EndIf
