PureBasic - Joystick

Einleitung

PureBasic bietet vollen Zugriff auf Joysticks, die am Computer angeschlossen sind. Diese Bibliothek unterstützt Joysticks und komplexe Gamepads mit mehreren Pads, Triggern und vielen Tasten.

Anmerkungen

Auf Windows verwendet diese Bibliothek die DirectX Technologie. Auf Linux verwendet sie SDL.
