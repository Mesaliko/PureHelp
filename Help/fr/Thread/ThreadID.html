ThreadID()

Syntaxe

Resultat = ThreadID(Thread)
Description
Renvoie l'identifiant système d'un thread.

Arguments

Thread Le thread à utiliser.

Valeur de retour

Renvoie l'identifiant système du thread spécifié.

Remarques

L'identifiant est parfois appelé 'Handle'. Pour plus d'informations consultez le chapitre Numéros et Identifiants (Handles).
