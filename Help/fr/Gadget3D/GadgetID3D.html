GadgetID3D()

Syntaxe

Resultat = GadgetID3D(#Gadget3D)
Description
Renvoie l'identifiant d'un gadget 3D.

Arguments

#Gadget3D Le gadget 3D à utiliser.

Valeur de retour

Renvoie l'identifiant système unique du gadget 3D.

OS Supportés

Tous

<- GadgetHeight3D() - Gadget3D Index - GadgetToolTip3D() ->
