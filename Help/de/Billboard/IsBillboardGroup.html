IsBillboardGroup()

Syntax

Ergebnis = IsBillboardGroup(#BillboardGruppe)
Beschreibung
Überprüft, ob die angegebene BillboardGruppen-Nummer gültig und eine korrekt initialisierte Billboard-Gruppe ist.

Parameter

#BillboardGruppe Die Nummer der zu verwendenden Billboard-Gruppe.

Rückgabewert

Ein Wert ungleich Null, wenn die Billboard-Gruppe gültig ist, andernfalls Null.

Anmerkungen

Diese Funktion ist "kugelsicher" und kann mit jedem Wert verwendet werden. Dies ist der korrekte Weg, um sicherzugehen, dass eine Billboard-Gruppe bereit zur Benutzung ist.

Unterstützte OS

Alle

<- HideBillboardGroup() - Billboard Inhaltsverzeichnis - MoveBillboard() ->
