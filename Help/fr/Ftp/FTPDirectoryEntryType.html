FTPDirectoryEntryType()

Syntaxe

Resultat = FTPDirectoryEntryType(#FTP)
Description
Renvoie le type de l'élément courant.

Arguments

#FTP La connexion FTP à utiliser.

Valeur de retour

Renvoie le type de l'élément courant sur le #FTP en cours d'examen avec ExamineFTPDirectory() et NextFTPDirectoryEntry().
Peut être une des valeurs suivantes: 
  #PB_FTP_File     : L'élément est un fichier.
  #PB_FTP_Directory: L'élément est un répertoire.

Voir aussi

ExamineFTPDirectory(), NextFTPDirectoryEntry(), FTPDirectoryEntryName(), FTPDirectoryEntryDate(), FTPDirectoryEntrySize(), FTPDirectoryEntryRaw(), FTPDirectoryEntryAttributes()
