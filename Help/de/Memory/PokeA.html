PokeA()

Syntax

PokeA(*Speicherpuffer, Nummer)
Beschreibung
Schreibt ein ASCII-Zeichen (1 Byte) an die angegebene Speicheradresse.

Parameter

*Speicherpuffer Die Adresse, an die geschrieben werden soll.
Nummer Der zu schreibende Wert.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.
