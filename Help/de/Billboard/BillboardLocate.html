BillboardLocate()

Syntax

BillboardLocate(#Billboard, #BillboardGruppe, x, y, z)
Beschreibung
Bewegt ein Billboard an eine neue Position innerhalb der Billboard-Gruppe, zu der es hinzugefügt wurde. Die neu angegebene Position ist relativ zur Position der Billboard-Gruppe. Verwenden Sie den MoveBillboard() Befehl, um ein Billboard relativ zu seinen eigenen Koordinaten zu bewegen.

Parameter

#Billboard Die Nummer des Billboards, welches bewegt werden soll.
#BillboardGruppe Die Nummer der Billboard-Gruppe, welche das Billboard enthält.
x, y, z Die neue Position des Billboards.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.

Unterstützte OS

Alle

<- BillboardHeight() - Billboard Inhaltsverzeichnis - BillboardWidth() ->
