GetGadgetItemData()

Syntax

Ergebnis = GetGadgetItemData(#Gadget, Eintrag)
Beschreibung
Gibt den Wert zurück, welcher zuvor für diesen #Gadget-Eintrag mittels SetGadgetItemData() gespeichert wurde. Dies ermöglicht es, jedem Eintrag in einem Gadget einen individuellen Wert zuzuweisen.

Parameter

#Gadget Das zu verwendende Gadget.
Eintrag Der Eintrag, von welchem die Daten gelesen werden sollen. Der erste Eintrag im Gadget hat den Index 0.

Rückgabewert

Gibt die gespeicherten Daten zurück. Wenn für den Eintrag noch kein Wert gespeichert wurde, wird der Rückgabewert gleich 0 sein.

Anmerkungen

Dieser Befehl arbeitet mit folgenden Gadgets:
- ComboBoxGadget()
- ListIconGadget()
- ListViewGadget()
- PanelGadget()
- TreeGadget()

Siehe SetGadgetItemData() für ein Beispiel.
