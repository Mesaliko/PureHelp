GetActiveGadget3D()

Syntaxe

Resultat = GetActiveGadget3D()
Description
Renvoie le numéro du gadget 3D qui a le focus clavier.

Arguments

Aucun.

Valeur de retour

Le numéro du gadget 3D qui a actuellement le 'focus' clavier. Si aucun gadget a le focus, -1 est renvoyé.
