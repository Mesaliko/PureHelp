ErrorFile()

Syntaxe

Resultat$ = ErrorFile()
Description
Renvoie le nom du fichier du code source où l'erreur est survenue.

Arguments

Aucun.

Valeur de retour

Renvoie le nom du fichier du code source où l'erreur est survenue.

Cette commande renvoie une valeur valide uniquement si l'erreur est gérée par OnErrorCall() ou OnErrorGoto().

Remarques

Le suivi des lignes en temps réel a besoin d'être activé à la compilation pour que cette fonction renvoie la ligne actuelle. Pour l'activer, il faut cocher "Activer le numéro de ligne pour OnError" dans les options de compilation ou spécifier /LINENUMBERING (Windows) ou --linenumbering (Linux, MacOS X) en ligne de commande. Si ce support est désactivé, cette fonction renverra "OnError line support disabled".

OS Supportés

Tous
