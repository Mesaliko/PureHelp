GetToolBarButtonState()

Syntaxe

Resultat = GetToolBarButtonState(#BarreOutils, Bouton)
Description
Renvoie l'état d'un Bouton.

Arguments

#BarreOutils La barre d'outils à utiliser.
Bouton Le bouton à tester.

Valeur de retour

Renvoie une valeur non nulle si le bouton de la barre d'outils est activé (enfoncé), zéro sinon.

Remarques

Attention, le bouton doit avoir été créé avec l'option #PB_ToolBar_Toggle.

SetToolBarButtonState() permet de changer l'état d'un bouton.

Voir aussi

SetToolBarButtonState()
