BuildMeshShadowVolume()

Syntax

BuildMeshShadowVolume(#Mesh)
Beschreibung
Erstellt das Ausmaß des Schattens für das angegebene #Mesh. Dies wird benötigt, wenn das Mesh einen Schatten werfen soll. Erfolgen sollte dies, sobald die Mesh-Erstellung vollständig abgeschlossen ist, oder der Schatten entspricht nicht dem (fertigen) Mesh.

Parameter

#Mesh Gibt das betroffene Mesh-Objekt an.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.
