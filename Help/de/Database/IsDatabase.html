IsDatabase()

Syntax

Ergebnis = IsDatabase(#Datenbank)
Beschreibung
Diese Funktion überprüft, ob die angegebene #Datenbank Nummer gültig und eine korrekt initialisierte Datenbank ist.

Parameter

#Datenbank Die zu verwendende Datenbank.

Rückgabewert

Gibt einen Wert ungleich Null zurück, wenn #Datenbank eine gültige Datenbank-Verbindung ist, andernfalls wird Null zurückgegeben.

Anmerkungen

Diese Funktion ist "kugelsicher" und kann mit jedem Wert benutzt werden. Ist das 'Ergebnis' ungleich Null, dann ist das Objekt gültig und initialisiert, andernfalls wird Null zurückgegeben. Dies ist ein guter Weg, um sicher zu stellen, dass eine Datenbank bereit zur Verwendung ist.
