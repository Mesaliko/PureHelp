InputRequester()

Syntax

Text$ = InputRequester(Titel$, Mitteilung$, StandardText$ [, Flags] [, ParendID]])
Beschreibung
Öffnet einen (das Programm) "blockierenden" Eingabe-Requester, um eine Texteingabe zu ermöglichen.

Parameter

Titel$ Titel des Requesters.
Mitteilung$ Die vor dem Eingabefeld angezeigte Mitteilung.
StandardText$ Standard-Text im Eingabefeld.
Flags (optional) Kann der folgende Wert sein: 
  #PB_InputRequester_Password: setzt den Feld-Typ auf 'Passwort'. Der angezeigte Text wird versteckt.
ParentID (optional) Die Kennung des übergeordneten Fensters. Eine gültige Fensterkennung kann mit WindowID() abgerufen werden.

Rückgabewert

Gibt den Text im Eingabefeld zurück, oder einen leeren String, wenn der Requester ohne Anklicken des "OK"-Schalters geschlossen wurde.

Beispiel

  Input$ = InputRequester("Titel", "Bitte machen Sie Ihre Eingabe:", "Ich bin die Standardeingabe.")

  If Input$ > ""
    a$ = "Sie haben folgendes in den Requester geschrieben:" + Chr(10)  ; Chr(10) wird nur für
    a$ + Input$                                                   ; den Zeilenumbruch benötigt
  Else  
    a$ = "Der Requester wurde abgebrochen oder es wurde nichts eingegeben."
  EndIf
  MessageRequester("Information", a$, 0)
