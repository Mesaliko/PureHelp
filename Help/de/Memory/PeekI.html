PeekI()

Syntax

Wert.i = PeekI(*Speicherpuffer)
Beschreibung
Liest einen Ganzzahl-Wert (auch "Integer", 4 Bytes in einem 32-Bit Executable, 8 Bytes in einem 64-Bit Executable) von der angegebenen Speicheradresse.

Parameter

*Speicherpuffer Die Adresse, von der gelesen werden soll.

Rückgabewert

Gibt den Wert des Integer-Werts (Ganzzahl) zurück.
