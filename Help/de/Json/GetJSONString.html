GetJSONString()

Syntax

Ergebnis$ = GetJSONString(JSONWert)
Beschreibung
Gibt den Wert eines JSON-Werts vom Typ #PB_JSON_String als String zurück.

Parameter

JSONWert Der JSON-Wert. Der Wert muss vom Typ #PB_JSON_String sein.

Rückgabewert

Der im JSON-Wert beinhaltete String.

Beispiel

  ParseJSON(0, Chr(34) + "The quick brown fox jumped over the lazy dog" + Chr(34))
  
  Debug GetJSONString(JSONValue(0))
