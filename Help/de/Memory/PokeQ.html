PokeQ()

Syntax

PokeQ(*Speicherpuffer, Nummer)
Beschreibung
Schreibt einen Quad-Wert (8 Bytes) an die angegebene Speicheradresse.

Parameter

*Speicherpuffer Die Adresse, an die geschrieben werden soll.
Nummer Der zu schreibende Wert.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.
