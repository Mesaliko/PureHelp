IsFont()

Syntax

Ergebnis = IsFont(#Font)
Beschreibung
Überprüft, ob die angegebene #Font Nummer gültig und ein korrekt initialisierter Zeichensatz ist.

Parameter

#Font Der zu verwendende Zeichensatz.

Rückgabewert

Gibt einen Wert ungleich Null zurück, wenn #Font ein gültiger Zeichensatz ist, andernfalls Null.

Anmerkungen

Diese Funktion ist "kugelsicher" und kann mit jedem Wert benutzt werden. Ist das Ergebnis ungleich Null, dann ist das Objekt gültig und initialisiert, andernfalls wird Null zurückgegeben. Dies ist der richtige Weg, um sicher zu stellen, dass ein Zeichensatz bereit zur Verwendung ist.
