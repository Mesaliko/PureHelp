DisableMenuItem()

Syntax

DisableMenuItem(#Menü, MenueEintrag, Status)
Beschreibung
Deaktiviert (oder aktiviert) einen Menü-Eintrag im angegebenen Menü.

Parameter

#Menü Das zu verwendende Menü.
MenueEintrag Die Nummer des Menü-Eintrags, welcher deaktiviert oder aktiviert werden soll.
Status Der neue Status des Menü-Eintrags. Ein Wert von 1 deaktiviert den Menü-Eintrag und ein Wert von 0 aktiviert ihn.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.

Beispiel

  If OpenWindow(0, 200, 200, 200, 100, "DisableMenuItem Example")
    If CreateMenu(0, WindowID(0))
      MenuTitle("Project")
        MenuItem(1, "Open")
        MenuItem(2, "Save")
        DisableMenuItem(0, 2, 1)    ; deaktiviere den zweiten Menü-Eintrag (Save)
    EndIf
    
    Repeat
    Until WaitWindowEvent() = #PB_Event_CloseWindow
  EndIf
