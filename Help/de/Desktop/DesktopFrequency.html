DesktopFrequency()

Syntax

Ergebnis = DesktopFrequency(#Desktop)
Beschreibung
Gibt die Frequenz des angegebenen Desktops zurück.

Parameter

#Desktop Der Index des Desktops. Der erste Index spezifiziert immer den Hauptdesktop (-monitor). Der erste Index-Wert ist Null.

Rückgabewert

Gibt die Frequenz des angegebenen Desktops in Hertz zurück. Ist das Ergebnis gleich 0, dann wird die Standard Hardware-Frequenz verwendet (der Anwender hat diese nicht gegenüber der Standardeinstellung verändert), oder die aktuelle Frequenz konnte nicht ermittelt werden.

Anmerkungen

ExamineDesktops() muss vor diesem Befehl aufgerufen werden, um die Informationen über vorhandene Desktops zu ermitteln. Hinweis: Auf Linux gibt diese Funktion immer 0 zurück.

Beispiel

  ExamineDesktops()
  f = DesktopFrequency(0)
  If f = 0
    MessageRequester("Display Information", "Es wurde keine Desktop-Frequenz eingestellt, die Standard-Hardwarefrequenz wird verwendet.")
  Else
    MessageRequester("Display Information", "Frequenz des Desktops: "+Str(f)+" Hz.")
  EndIf
