ResizeBillboard()

Syntax

ResizeBillboard(#Billboard, #BillboardGruppe, Breite, Höhe)
Beschreibung
Verändert die Größe (englisch: "resizes") eines Billboards, welches aktuell in einer Billboard-Gruppe enthalten ist, auf eine neue Breite und Höhe, angegeben in Welt-Einheiten. Beachten Sie, dass es - auch wenn Sie Billboards mit diesem Befehl unabhängig in der Größe verändern können - einige Leistungseinbußen gibt, wenn die Billboards innerhalb einer Billboard-Gruppe alle verschiedene Größen haben.

Parameter

#Billboard Die Nummer des Billboards, dass in der Größe verändert werden soll.
#BillboardGruppe Die Nummer der Billboard-Gruppe, welche aktuell das Billboard enthält.
Breite, Höhe Die neue Größe des Billboards, in Welt-Einheiten.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.

Unterstützte OS

Alle
