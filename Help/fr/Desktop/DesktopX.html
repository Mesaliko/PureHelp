DesktopX()

Syntaxe

Resultat = DesktopX(#Bureau)
Description
Renvoie la position 'X' d'un bureau.

Arguments

#Bureau Le numéro du bureau à tester.

Le bureau principal a toujours la valeur 0.

Valeur de retour

Renvoie la position 'X' en pixels, à partir du coin supérieur gauche du bureau.

La coordonnée est relative au coin supérieur gauche de l'écran principal.
Elle est négative si le bureau est indiqué à gauche de l'écran principal.

Remarques

ExamineDesktops() doit être appelée avant cette commande pour mettre à jour les informations concernant les bureaux.

Exemple

  ExamineDesktops()
  MessageRequester("Position du bureau principal", "Position en X = "+Str(DesktopX(0)) + #LF$ + "Position en Y = "+Str(DesktopY(0)))

Voir aussi
