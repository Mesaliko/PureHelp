UseOGGSoundDecoder()

Syntax

UseOGGSoundDecoder()
Beschreibung
Aktiviert den OGG (OGG Vorbis) Sound Support für CatchSound() und LoadSound().
Sound-Streaming wird bei diesem Plugin unterstützt.

Parameter

Diese Funktion hat keine Parameter.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.

Anmerkungen

Das OGG-Format verwendet eine verlustbehaftete Komprimierung. Dies bedeutet, dass der Sound beim Komprimieren einige seiner Originalinformationen verliert. Dies ist aktuell einer der besten verfügbaren verlustbehafteten Kompressionsalgorithmen, welcher die Soundgröße - ohne eine Menge von hörbaren Artefakten zu produzieren - dramatisch vermindern kann.

Für weitere Informationen: Wikipedia - OGG.
