AvailableProgramOutput()

Syntax

Ergebnis = AvailableProgramOutput(Programm)
Beschreibung
Gibt die Anzahl an Bytes zurück, die zum Lesen von der Programmausgabe zur Verfügung stehen.

Parameter

Programm Das zu verwendende Programm. Dieses muss mit RunProgram() unter Verwendung des #PB_Program_Read Flag gestartet worden sein.

Rückgabewert

Die Anzahl an Bytes, die zum Lesen von der Programmausgabe zur Verfügung stehen.

Anmerkungen

Die Ausgabe kann entweder mittels ReadProgramString() oder ReadProgramData() gelesen werden.

Allerdings bleiben ReadProgramString() bzw. ReadProgramData() unbegrenzt gesperrt, wenn das Programm einen Eingriff des Nutzers anfordert. Geben Sie z.B. eine Ja/Nein-Auswahl an das Programm zurück, um fortzufahren.
