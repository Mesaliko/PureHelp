JSONArraySize()

Syntax

Ergebnis = JSONArraySize(JSONWert)
Beschreibung
Gibt die Anzahl der in einem JSON-Wert vom Typ #PB_JSON_Array beinhalteten Elemente zurück.

Parameter

JSONWert Der JSON-Wert. Der Wert muss vom Typ #PB_JSON_Array sein.

Rückgabewert

Die Anzahl an Elementen im JSON-Array.

Beispiel

  ParseJSON(0, "[1, 2, null, true]")
  Debug JSONArraySize(JSONValue(0))
