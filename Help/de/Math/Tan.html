Tan()

Syntax

Ergebnis.f(.d) = Tan(Winkel.f(.d))
Beschreibung
Gibt den Tangens des angegebenen Winkels zurück.

Parameter

Winkel.f oder Winkel.d Der Eingabe-Winkel im Bogenmaß (Radiant). Die Radian() Funktion kann verwendet werden, um einen Winkel von Grad in das Bogenmaß (Radiant) umzuwandeln.

Rückgabewert

Gibt den Tangens des angegebenen Winkels zurück.

Anmerkungen

Die Umkehrfunktion von Tan() ist ATan(). Diese Funktion verarbeitet und gibt Float oder Double Werte zurück.

Beispiel

  Debug Tan(0.785398) ; wird ungefähr '1.0' ergeben
