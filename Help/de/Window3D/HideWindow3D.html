HideWindow3D()

Syntax

HideWindow3D(#Window3D, Status)
Beschreibung
Versteckt oder zeigt das angegebene Fenster #Window3D.

Parameter

#Window3D Das zu verwendende 3D-Fenster.
Status Dieser kann folgende Werte annehmen: 
  #True : das Fenster #Window3D wird versteckt
  #False: das Fenster #Window3D wird angezeigt

Rückgabewert

Diese Funktion gibt keinen Wert zurück.

Unterstützte OS

Alle

<- GetWindowTitle3D() - Window3D Inhaltsverzeichnis - InputEvent3D() ->
