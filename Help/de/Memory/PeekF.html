PeekF()

Syntax

Wert.f = PeekF(*Speicherpuffer)
Beschreibung
Liest einen Float-Wert (4 Bytes) von der angegebenen Speicheradresse.

Parameter

*Speicherpuffer Die Adresse, von der gelesen werden soll.

Rückgabewert

Gibt den Wert des Float-Werts zurück.
