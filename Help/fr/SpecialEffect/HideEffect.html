HideEffect()

Syntaxe

HideEffect(#Effet, Etat)
Description
Masque ou affiche un effet.

Arguments

#Effet L'effet à utiliser.
Etat 
  #True  : L'effet sera caché.
  #False : L'effet sera affiché.

Valeur de retour

Aucune.
