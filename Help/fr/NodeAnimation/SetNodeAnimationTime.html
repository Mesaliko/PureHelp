SetNodeAnimationTime()

Syntaxe

SetNodeAnimationTime(#NodeAnimation, Temps)
Description
Modifie le temps courant d'une animation .

Arguments

#NodeAnimation L'animation à utiliser.
Temps Le temps absolu (en millisecondes).

Valeur de retour

Aucune.

Remarques

Il s'agit d'une position temporelle absolue.
Pour changer le temps, utilisez AddNodeAnimationTime().

Voir aussi

StartNodeAnimation(), AddNodeAnimationTime(), GetNodeAnimationTime()

OS Supportés

Tous
