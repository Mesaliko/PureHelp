Text3DY()

Syntaxe

Resultat = Text3DY(#Text3D)
Description
Renvoie la position absolue 'y' du texte dans le monde.

Arguments

#Text3D Le texte 3D à utiliser.

Valeur de retour

Renvoie la position absolue 'y' du texte.
