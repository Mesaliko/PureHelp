CountMaterialLayers()

Syntaxe

Resultat = CountMaterialLayers(#Matiere)
Description
Renvoie le nombre de couches d'une matière.

Arguments

#Matiere La matière à utiliser.

Valeur de retour

Renvoie le nombre de couches que contient la matière.
