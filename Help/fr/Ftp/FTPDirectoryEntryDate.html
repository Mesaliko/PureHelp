FTPDirectoryEntryDate()

Syntaxe

Resultat = FTPDirectoryEntryDate(#FTP)
Description
Renvoie la date d'un fichier

Arguments

#FTP La connexion FTP à utiliser.

Valeur de retour

Renvoie la date du fichier courant sur le FTP en cours d'examen avec ExamineFTPDirectory() et NextFTPDirectoryEntry().
La date est dans le même format que celui utilisé dans la bibliothèque Date, donc toutes les commandes s'y rapportant comme FormatDate() sont utilisables.
