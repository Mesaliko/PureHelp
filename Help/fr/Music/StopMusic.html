StopMusic()

Syntaxe

StopMusic(#Music)
Description
Arrête la lecture.

Arguments

#Music Le module musical à utiliser.

Si '#Music' est égal à #PB_All, alors tous les modules musicaux en cours de lecture sont arrêtés.

Valeur de retour

Aucune.
