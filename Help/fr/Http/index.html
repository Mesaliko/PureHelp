PureBasic - Http

Généralités

Http est le nom du protocole utilisé par les navigateurs Web pour accéder aux sites distants, comme par exemple une page Web. Chaque donnée distante a son adresse unique: l'URL (Uniform Resource Locator). Cette bibliothèque permet de manipuler facilement les URLs et de télécharger des fichiers distants.

Sous Linux, 'libcurl' doit être installé pour que certaines commandes HTTP fonctionnent (déjà installé dans la plupart des distributions Linux).
