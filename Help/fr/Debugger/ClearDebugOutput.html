ClearDebugOutput()

Syntaxe

ClearDebugOutput()
Description
Efface le contenu de la fenêtre de déboguage.

Arguments

Aucun.

Valeur de retour

Aucune.

Exemple

  ; Montre 10 valeurs dans le debugger tout les 500 ms
  Repeat
    ClearDebugOutput()
    For i = 1 To 10
      Debug x
      x + 1
    Next i
    
    Delay(500)
  ForEver
