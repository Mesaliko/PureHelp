ExamineKeyboard()

Syntaxe

ExamineKeyboard()
Description
Met à jour l'état du clavier.

Arguments

Aucun.

Valeur de retour

Aucune.

Remarques

Cette fonction doit être appelée avant d'utiliser les commandes KeyboardInkey(), KeyboardPushed() et KeyboardReleased().

En mode plein écran, la commande FlipBuffers() doit être appelée dans la boucle d'événements pour gérer correctement tous les événements.
