ASinH()

Syntax

Ergebnis.f(.d) = ASinH(Wert.f(.d))
Beschreibung
Gibt den Areasinus (Area Sinus Hyperbolicus) des angegebenen Wertes zurück.

Parameter

Wert.f oder Wert.d Der Eingabewert. Der Wertebereich von 'Wert.f(.d)' ist nicht begrenzt.

Rückgabewert

Gibt den hyperbolischen Winkel zurück.

Anmerkungen

Dies ist die Umkehrfunktion von SinH(). Diese Funktion verarbeitet und gibt Float oder Double Werte zurück.

Beispiel

  Debug ASinH(0)        ; wird '0.0' anzeigen
  Debug Exp(ASinH(0.5)) ; wird '1.618033...' anzeigen (den goldenen Schnitt)
