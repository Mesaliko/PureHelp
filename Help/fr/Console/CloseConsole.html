CloseConsole()

Syntaxe

CloseConsole()
Description
Ferme la console.

Arguments

Aucun.

Valeur de retour

Aucune.

Remarques

Une fois que la console est fermée il n'est plus possible d'utiliser les fonctions de cette bibliothèque.
La console se ferme automatiquement à la fin de l'exécution du programme.
Cette fonction n'a pas d'effet sous Linux et MacOS.

Exemple

  For i = 0 To 4
    If OpenConsole()
      PrintN("C'est la console #"+Str(i))
      PrintN("Appuyez sur [Entree] pour quitter")
      Input()
      CloseConsole()
    EndIf
  Next
