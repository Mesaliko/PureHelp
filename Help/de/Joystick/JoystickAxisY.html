JoystickAxisY()

Syntax

Ergebnis = JoystickAxisY(#Joystick [, Pad [, Modus]])
Beschreibung
Gibt den Status der Y-Achse des Joysticks zurück.

Parameter

#Joystick Der zu verwendende Joystick. Der Index des ersten Joysticks lautet 0. Die Anzahl der verfügbaren Joysticks wird von InitJoystick() zurückgegeben.
Pad (optional) Der zu verwendende Pad, wenn der Joystick mehrere Pads besitzt. Der Index des ersten Pads lautet 0.
Modus (optional) Der Modus kann einer der folgenden Werte sein: 
  #PB_Absolute: Der zurückgegebene Wert ist entweder -1 (oben), 0 (keine Bewegung) oder 1 (unten). (Standard)
  #PB_Relative: Der zurückgegebene Wert liegt im Bereich zwischen -1000 (oben) und 1000 (unten). Wenn das 
                Gamepad keine relative Bewegung unterstützt, wird das Ergebnis -1000, 0 oder 1000 lauten.

Rückgabewert

Gibt den Wert der Y-Achse des Joysticks zurück, abhängig vom angegebenen Modus.

Anmerkungen

ExamineJoystick() muss vor der Verwendung dieser Funktion aufgerufen werden, um den momentanen Joystick-Status zu aktualisieren.
