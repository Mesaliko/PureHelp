SpriteOutput()

Syntax

OutputID = SpriteOutput(#Sprite)
Beschreibung
Ermittelt die OutputID des angegebenen Sprites, um darauf 2D-Zeichenoperationen durchzuführen.

Parameter

#Sprite Das Sprite, auf dem gezeichnet werden soll.

Rückgabewert

Die OutputID - oder Null, wenn Zeichnen nicht möglich ist. Dieser Wert sollte direkt an die StartDrawing() Funktion übergeben werden, um die Zeichenoperation zu starten. Der Rückgabewert ist nur für eine Zeichenoperation gültig und kann nicht wiederverwendet werden.

Anmerkungen

SpriteOutput() muss im selben Thread aufgerufen werden, in dem OpenScreen() aufgerufen wurde.

Beispiel

  StartDrawing(SpriteOutput(#Sprite))
    ; hier folgen die Zeichenoperationen...
  StopDrawing()
