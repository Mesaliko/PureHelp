ParseXML()

Syntax

Ergebnis = ParseXML(#XML, Eingabe$)
Beschreibung
Erstellt einen neuen XML-Baum ausgehend von den XML-Daten im String. Vom XML wird erwartet, dass es in dem vom Executable verwendeten String-Format (ASCII oder Unicode) kodiert ist.

Wenn eine andere Kodierung analysiert ("parsed") werden soll, kann stattdessen die CatchXML() Funktion verwendet werden.

Parameter

#XML Eine Nummer zum Identifizieren des neuen XML. #PB_Any kann verwendet werden, um diese Zahl automatisch zu generieren.
Eingabe$ Der String, welcher die zu analysierenden ("zu parsenden") XML-Daten enthält.

Rückgabewert

Diese Funktion gibt nur bei Speicherfehlern Null zurück.
Um Parser-Fehler zu überprüfen, sollte XMLStatus() verwendet werden. Im Fall eines Parser-Fehlers sind alle vor dem Fehler bereits analysierten Daten im XML-Baum verfügbar.
