ListViewGadget3D()

Syntax

Ergebnis = ListViewGadget3D(#Gadget3D, x, y, Breite, Höhe)
Beschreibung
Erstellt ein ListView Gadget (Auswahlliste) innerhalb der aktuellen Gadgetliste. Sobald ein ListView Gadget erstellt wurde, ist dessen Listeninhalt leer.

Parameter

#Gadget3D Eine Nummer zum Identifizieren des neuen 3D-Gadgets. #PB_Any kann verwendet werden, um diese Zahl automatisch zu generieren.
x, y, Breite, Höhe Die Position und Abmessungen des neuen Gadgets.

Rückgabewert

Gibt im Erfolgsfall einen Wert ungleich Null zurück, andernfalls Null. Wenn #PB_Any als #Gadget3D Parameter verwendet wurde, dann enthält bei Erfolg der Rückgabewert die automatisch generierte Nummer.

Anmerkungen

Zu diesem Gadget kann mittels GadgetToolTip3D() eine 'Mini-Hilfe' hinzugefügt werden.

Die folgenden Befehle können benutzt werden, um auf den Listen-Inhalt zuzugreifen:

- AddGadgetItem3D(): fügt einen Eintrag hinzu.
- RemoveGadgetItem3D(): entfernt einen Eintrag.
- ClearGadgetItems3D(): entfernt alle Einträge.
- CountGadgetItems3D(): ermittelt die Anzahl der aktuellen Einträge im #Gadget3D.

- GetGadgetItemData3D(): ermittelt den Wert, der mit dem Gadget-Eintrag gespeichert wurde.
- GetGadgetItemState3D(): gibt einen Wert ungleich Null zurück, wenn der Eintrag ausgewählt ist, andernfalls Null.
- GetGadgetItemText3D(): ermittelt den Inhalt des angegebenen Eintrags.
- GetGadgetState3D(): ermittelt den Index vom ausgewählten Eintrag oder -1, wenn kein Eintrag selektiert ist.
- GetGadgetText3D(): ermittelt den Textinhalt vom ausgewählten Eintrag.
- SetGadgetItemData3D(): speichert einen Wert mit dem angegebenen Eintrag.
- SetGadgetItemState3D(): selektiert den angegebenen Eintrag oder entfernt die Auswahl.
- SetGadgetItemText3D(): Set the text of the given item.
- SetGadgetState3D(): ändert den ausgewählten Eintrag. Wird -1 angegeben, dann wird die Auswahl entfernt.
- SetGadgetText3D(): wählt den Eintrag mit dem angegebenen Text aus (der Text muss exakt übereinstimmen).
