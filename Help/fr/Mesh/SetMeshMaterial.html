SetMeshMaterial()

Syntaxe

SetMeshMaterial(#Mesh, MateriauID [, SousMesh])
Description
Définit le matériau par défaut du mesh.

Arguments

Mesh Le mesh à utiliser.
MateriauID Spécifie le matériau à utiliser par défaut pour le mesh. Pour obtenir un 'MateriauID' valide, utiliser MaterialID().
SousMesh (optionnel) S'il est spécifié, le matériau ne sera appliqué que sur le sous-mesh. Le premier indice de sous-mesh est 0 (mesh principal).

Valeur de retour

Aucune.

CreateMesh(), LoadMesh()

OS Supportés

Tous
