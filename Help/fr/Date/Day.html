Day()

Syntaxe

Resultat = Day(Date.q)
Description
Renvoie le jour.

Arguments

Date.q La date à utiliser.

Valeur de retour

Renvoie le jour de la date spécifiée ou -1 en cas d'erreur.
Le résultat est compris entre 1 et 31.

Exemple

  Debug Day(Date(2022, 10, 3, 0, 0, 0))  ; Le résultat est '3'.
