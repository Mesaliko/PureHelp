ParseDate()

Syntaxe

Resultat.q = ParseDate(Masque$, Date$)
Description
Transforme une date donnée sous la forme d'une chaîne de caractères en valeur numérique.

Arguments

Masque$ Une chaîne de caractères indiquant comment formater la date. 
  %yyyy: L'année sera affichée avec 4 chiffres.
    %yy: L'année sera affichée avec 2 chiffres.
    %mm: Le mois sera affiché avec 2 chiffres.
    %dd: Le jour sera affiché avec 2 chiffres.
    %hh: L'heure sera affichée avec 2 chiffres.
    %ii: Les minutes seront affichées avec 2 chiffres.
    %ss: Les secondes seront affichées avec 2 chiffres.
Date$ La date à tester.

Valeur de retour

Renvoie la date représentant la chaîne à analyser.

Si la date est incorrecte, la valeur renvoyée est -1.

Cette fonction est particulièrement utile pour retrouver et traiter des dates stockées, dans un fichier de type journal d'évènements par exemple.

Exemple

  Debug ParseDate("%yy/%mm/%dd", "20/12/01") ; Renvoie la date numérique correspondant à "01/12/20"
  Debug ParseDate("%dd/%mm/%yyyy", "01/07/2020") ; Renvoie la date numérique correspondant à "01/07/2020"
