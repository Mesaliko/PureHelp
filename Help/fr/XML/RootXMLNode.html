RootXMLNode()

Syntaxe

Resultat = RootXMLNode(#XML)
Description
Renvoie le noeud racine.

Arguments

#XML L'arbre XML à utiliser.

Valeur de retour

Renvoie toujours un noeud XML valide si l'arbre #XML est correctement initialisé.

Remarques

Ce noeud est toujours présent. Il représente le document XML au complet. Le texte de ce noeud contient les espaces en dehors de tout noeud XML (il ne peut pas y avoir de texte en dehors des noeuds). Les fils de ce noeud sont le noeud principal et tous les commentaires situés en dehors du noeud principal. Le type de ce noeud est #PB_Xml_Root.
