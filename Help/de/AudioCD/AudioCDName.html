AudioCDName()

Syntax

Ergebnis = AudioCDName()
Beschreibung
Gibt den systemabhängigen Namen zurück, der dem aktuellen Audio-CD Laufwerk zugewiesen wurde.

Parameter

Diese Funktion hat keine Parameter.

Rückgabewert

Gibt einen String mit dem Namen zurück. Zum Beispiel wird auf dem Windows-System 'D:\' zurückgegeben, wenn die Audio-CD dem System als Laufwerks-Buchstabe 'D' zugewiesen ist.
