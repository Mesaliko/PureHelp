NextDatabaseRow()

Syntax

Ergebnis = NextDatabaseRow(#Datenbank)
Beschreibung
Ermittelt Informationen über den nächsten Datenbank-Eintrag (Zeile) in der #Datenbank.

Um auf die Informationen innerhalb des Eintrags (bzw. der Zeile) zuzugreifen, können GetDatabaseLong(), GetDatabaseFloat() und GetDatabaseString() verwendet werden.

Parameter

#Datenbank Die zu verwendende Datenbank.

Rückgabewert

Ist das Ergebnis gleich 0, dann sind keine weiteren Einträge mehr verfügbar (d.h. das Ende der Tabelle wurde erreicht).
