SetJSONFloat()

Syntax

SetJSONFloat(JSONWert, Wert.f)
Beschreibung
Ändert den Typ des JSON-Wertes auf #PB_JSON_Number und speichert den angegebenen Fließkommawert (Float).

Parameter

JSONWert Der JSON-Wert.
Wert.f Der zu speichernde Wert.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.

Anmerkungen

Beachte, dass JSON nicht die speziellen Fließkommawerte +Infinity, -Infinity oder NaN in den JSON-Daten ermöglicht. Wenn ein solcher Wert mit dieser Funktion gesetzt wird, dann wird dieser beim Speichern oder Kodieren durch ein JSON 'null' Literal ersetzt. Die Funktionen IsInfinity() bzw. IsNaN() können verwendet werden, um dies festzustellen.

Beispiel

  If CreateJSON(0)
    ArrayValue = SetJSONArray(JSONValue(0))
    SetJSONFloat(AddJSONElement(ArrayValue), 1.23)
    SetJSONFloat(AddJSONElement(ArrayValue), 4.56)
    
    Debug ComposeJSON(0)
  EndIf
