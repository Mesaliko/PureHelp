IsProgram()

Syntax

Ergebnis = IsProgram(Programm)
Beschreibung
Überprüft, ob das angegebene 'Programm' ein Programm ist, welches mittels der RunProgram() Funktion gestartet wurde.

Parameter

Programm Das zu überprüfende Programm.

Rückgabewert

Ein Wert ungleich Null, wenn das Programm durch die RunProgram() Funktion ausgeführt wird, andernfalls Null.

Anmerkungen

Diese Funktion ist "kugelsicher" und kann mit jedem Wert verwendet werden.
