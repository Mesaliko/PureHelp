PackEntryName()

Syntaxe

Resultat$ = PackEntryName(#Archive)
Description
Renvoie le nom de l'entrée courante trouvée avec NextPackEntry().

Arguments

#Archive L'archive à utiliser.

Valeur de retour

Renvoie le nom de l'entrée courante.

Voir aussi

OpenPack(), CatchPack(), NextPackEntry(), PackEntryType(), PackEntrySize().

OS Supportés

Tous
