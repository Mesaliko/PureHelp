InsertString()

Syntax

Ergebnis$ = InsertString(String$, StringToInsert$, Position)
Beschreibung
Fügt den String 'StringToInsert$' in den 'String$' an der angegebenen 'Position' ein.

Parameter

String$ Der zu verwendende String.
StringToInsert$ Der einzufügende String.
Position Die Position im String, an der der neue String eingefügt werden soll. Der Index der ersten Position lautet 1.

Rückgabewert

Ein neuer String mit dem an der angegebenen Position eingefügten String.

Beispiel

  Debug InsertString("Hello !", "World", 7)  ; Wird "Hello World!" anzeigen
  Debug InsertString("Hello !", "World", 1)  ; Wird "WorldHello !" anzeigen
