MeshPoseName()

Syntax

Ergebnis$ = MeshPoseName(#Mesh, PoseIndex)
Beschreibung
Gibt den Namen der Pose im Mesh zurück. Eine Pose ist eine vordefinierte Vertex-Animation im Mesh.

Parameter

#Mesh Das zu verwendende Mesh.
PoseIndex Der Posen-Index, von dem der Name ermittelt werden soll. Der erste Index-Wert für eine Pose beginnt bei Null. Der Index muss niedriger als das Ergebnis von MeshPoseCount() sein.

Rückgabewert

Gibt den Namen der Pose beim angegebenen Index zurück.
