RemoveJSONElement()

Syntaxe

RemoveJSONElement(JSONValeur, Index)
Description
Retire l'élément à l'index spécifié d'une valeur de JSON de type #PB_JSON_Array.

Arguments

JSONValeur La valeur JSON.

La valeur doit être de type #PB_JSON_Array.
Index L'indice de l'élément à supprimer.

La valeur doit être comprise entre 0 et JSONArraySize() - 1.

Valeur de retour

Aucune.

Exemple

  ParseJSON(0, "[1, 2, 3, 4, 5]")
  RemoveJSONElement(JSONValue(0), 2)
  Debug ComposeJSON(0)
  
  ; Resultat : [1, 2, 4, 5]
