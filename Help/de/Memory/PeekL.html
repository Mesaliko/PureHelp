PeekL()

Syntax

Wert.l = PeekL(*Speicherpuffer)
Beschreibung
Liest einen Long-Wert (4 Bytes) von der angegebenen Speicheradresse.

Parameter

*Speicherpuffer Die Adresse, von der gelesen werden soll.

Rückgabewert

Gibt den Wert des Long-Werts zurück.
