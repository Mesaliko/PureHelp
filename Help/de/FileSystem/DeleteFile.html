DeleteFile()

Syntax

Ergebnis = DeleteFile(DateiName$ [, Modus])
Beschreibung
Löscht die angegebene Datei.

Parameter

DateiName$ Die zu löschende Datei.
Modus (optional) Optionen für die Lösch-Operation. Dies kann einer der folgenden Werte sein: 
  #PB_FileSystem_Force: Löscht auch die geschützten Dateien ("read-only" / schreibgeschützt).

Rückgabewert

Gibt einen Wert ungleich Null zurück, wenn die Operation erfolgreich war, und Null bei einem Fehler.
