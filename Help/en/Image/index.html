PureBasic - Image

Overview

Images are graphics objects which can be displayed in a window or in several gadgets. PureBasic supports beside BMP and ICON (.ico, Windows only) image types all other image types, which are supported by the ImagePlugin Library.

Transparent PNG images can be used to enable transparency in the gadgets, menu and toolbars images. On Windows, transparent ICON images can be used as well but PNG should be preferred as it works on all platforms. Transparency of BMP images is not supported.
