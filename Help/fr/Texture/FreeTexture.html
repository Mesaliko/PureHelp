FreeTexture()

Syntaxe

FreeTexture(#Texture)
Description
Supprime une texture.

Arguments

#Texture La texture à utiliser.

Si #PB_All est spécifié, toutes les textures restantes sont libérées.

Valeur de retour

Aucune.

Remarques

Toutes les textures restantes sont automatiquement supprimées quand le programme se termine.

Voir aussi

CreateTexture(), LoadTexture().

OS Supportés

Tous
