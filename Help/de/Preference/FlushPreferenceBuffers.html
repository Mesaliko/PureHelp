FlushPreferenceBuffers()

Syntax

Ergebnis = FlushPreferenceBuffers()
Beschreibung
Stellt sicher, dass alle Änderungen an den Einstellungen (englisch "Preferences") auf Disk geschrieben werden.

Parameter

Diese Funktion hat keine Parameter.

Rückgabewert

Ein Wert ungleich Null, wenn die Preferences erfolgreich auf Disk geschrieben wurden. Wenn ein Fehler auftrat (z.B. die Disk war voll, ein Disk-Fehler), dann wird Null zurückgegeben.
