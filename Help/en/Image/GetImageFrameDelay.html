GetImageFrameDelay()

Syntax

Result = GetImageFrameDelay(#Image)
Description
Returns the display delay (in milliseconds) for the current image frame. Each frame can have its own delay.

Parameters

#Image The image to get the frame delay.

Return value

Returns the display delay (in milliseconds) for the current image frame.

See Also

CreateImage(), AddImageFrame(), RemoveImageFrame(), SetImageFrame(), GetImageFrame(), SetImageFrameDelay()

Supported OS

All
