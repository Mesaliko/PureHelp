CloseGadgetList3D()

Syntax

CloseGadgetList3D()
Beschreibung
Beendet die aktuelle Gadgetlisten-Erstellung und geht zurück zur vorherigen Gadgetliste.

Parameter

Diese Funktion hat keine Parameter.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.

Anmerkungen

Diese Funktion wird von den folgenden 3D-Gadgets unterstützt:

- ContainerGadget3D()
- PanelGadget3D()
- ScrollAreaGadget3D()

Unterstützte OS

Alle

<- ClearGadgetItems3D() - Gadget3D Inhaltsverzeichnis - ComboBoxGadget3D() ->
