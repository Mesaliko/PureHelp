MoveMemory()

Syntax

MoveMemory(*QuellMemoryID, *ZielMemoryID, Größe)
Beschreibung
Kopiert einen Speicherbereich beginnend bei der '*QuellMemoryID' an die '*ZielMemoryID'. Das Überlappen der beiden Speicherbereiche ist erlaubt.

Parameter

*QuellMemoryID Die Adresse, von der die Bytes kopiert werden.
*ZielMemoryID Die Adresse, wohin die Bytes kopiert werden.
Größe Die Anzahl an zu kopierenden Bytes.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.

Anmerkungen

Dieser Befehl kann langsamer als CopyMemory() sein, aber er stellt sicher, dass die Bytes korrekt kopiert werden, auch wenn sich die zwei Speicherbereiche überschneiden.
