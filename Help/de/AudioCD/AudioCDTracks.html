AudioCDTracks()

Syntax

Ergebnis = AudioCDTracks()
Beschreibung
Gibt die Gesamtzahl der Tracks auf der CD zurück, die zum Abspielen verfügbar sind.

Parameter

Diese Funktion hat keine Parameter.

Rückgabewert

Gibt die Anzahl an Tracks zurück.
