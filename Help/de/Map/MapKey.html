MapKey()

Syntax

Key$ = MapKey(Map())
Beschreibung
Gibt den Schlüssel des aktuellen Map-Elements zurück.

Parameter

Map() Die zu verwendende Map.

Rückgabewert

Gibt den Schlüssel 'Key$' des aktuellen Map-Elements zurück. Gibt es kein aktuelles Element, dann wird ein leerer String zurückgegeben.

Beispiel

  NewMap Country.s()

  Country("US") = "United States"
  Country("FR") = "France"
  Country("GE") = "Germany"

  ForEach Country()
    Debug MapKey(Country())
  Next
