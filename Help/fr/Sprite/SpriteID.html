SpriteID()

Syntaxe

Resultat = SpriteID(#Sprite)
Description
Renvoie l'identifiant système d'un Sprite.

Arguments

#Sprite Le sprite à utiliser.

Valeur de retour

L'ID du sprite.
Ce résultat est parfois aussi appelé 'Handle'. Voir le chapitre Numéros et Identifiants (Handles) pour plus d'informations.

Voir aussi

CreateSprite(), LoadSprite(), CatchSprite()

OS Supportés

Tous

<- SpriteHeight() - Sprite Index - SpriteOutput() ->
