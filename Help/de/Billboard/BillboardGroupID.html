BillboardGroupID()

Syntax

BillboardGroupID = BillboardGroupID(#BillboardGruppe)
Beschreibung
Gibt die einmalige ID zurück, welche die angegebene '#BillboardGruppe' im Betriebssystem identifiziert. Diese Funktion ist sehr nützlich, wenn eine andere Bibliothek einen Bezug auf die Billboard-Gruppe benötigt.

Parameter

#BillboardGruppe Die zu verwendende Billboard-Gruppe.

Rückgabewert

Die einmalige ID, welche die angegebene '#BillboardGruppe' im Betriebssystem identifiziert.

Unterstützte OS

Alle

<- BillboardGroupCommonUpVector() - Billboard Inhaltsverzeichnis - BillboardGroupMaterial() ->
