Year()

Syntaxe

Resultat = Year(Date.q)
Description
Renvoie l'année.

Arguments

Date.q La date à utiliser.

Valeur de retour

Renvoie l'année de la date spécifiée ou -1 en cas d'erreur.
Le résultat est compris entre 1970 et 2034.

Exemple

  Debug Year(Date(2022, 10, 3, 0, 0, 0))  ; Le résultat est '2022'.
