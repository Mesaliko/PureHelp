FreeSprite()

Syntaxe

FreeSprite(#Sprite)
Description
Supprime un Sprite de la mémoire.

Arguments

#Sprite Le sprite à libérer.

Si #PB_All est spécifié, tous les sprites restants sont libérés.

Valeur de retour

Aucune.

Remarques

Tous les sprites restants sont automatiquement libérés quand le programme se termine.
