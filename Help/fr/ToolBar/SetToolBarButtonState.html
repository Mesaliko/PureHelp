SetToolBarButtonState()

Syntaxe

SetToolBarButtonState(#BarreOutils, Bouton, Etat)
Description
Change l'état d'un Bouton.

Arguments

#BarreOutils La barre d'outils à utiliser.
Bouton Le bouton à utiliser.
Etat 
  #False : Etat relâché
  #True  : Etat enfoncé

Valeur de retour

Aucune.

Remarques

Attention, le bouton doit avoir été créé avec l'option #PB_ToolBar_Toggle.

GetToolBarButtonState() permet de connaître l'état d'un bouton.

OS Supportés

Tous
