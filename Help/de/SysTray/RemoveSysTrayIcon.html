RemoveSysTrayIcon()

Syntax

RemoveSysTrayIcon(#SysTrayIcon)
Beschreibung
Entfernt das angegebene SysTray-Icon.

Parameter

#SysTrayIcon Das zu entfernende SysTray-Icon. Wenn #PB_All angegeben wird, dann werden alle verbliebenen SysTray-Icons freigegeben.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.

Anmerkungen

Alle verbleibenden SysTray-Icons werden automatisch entfernt, wenn das Programm endet.

Unterstützte OS

Alle

<- IsSysTrayIcon() - SysTray Inhaltsverzeichnis - SysTrayIconMenu() ->
