EntityCollide()

Syntax

Ergebnis = EntityCollide(#Entity, #Entity2)
Beschreibung
Überprüft, ob die zwei angegebenen Entity-Objekte kollidieren.

Um ihre Kollisionen durch die Physik-Engine verwalten zu können, benötigt das Entity einen mittels CreateEntityBody() erstellten Körper ("body").

Um einen Effekt zu haben, muss die Physik-Engine mittels EnableWorldPhysics() aktiviert sein.

Parameter

#Entity Das erste zu testende Entity.
#Entity2 Das zweite zu testende Entity.

Rückgabewert

Gibt einen Wert ungleich Null zurück, wenn die zwei Entity-Objekte kollidieren.

Unterstützte OS

Alle

<- EntityBoundingBox() - Entity Inhaltsverzeichnis - EntityCustomParameter() ->
