GetMailBody()

Syntaxe

Resultat$ = GetMailBody(#Courrier)
Description
Renvoie le contenu d'un courrier.

Arguments

#Courrier Le courrier à utiliser.

Valeur de retour

Renvoie une chaîne de caractères qui est le contenu du courrier, préalablement défini par SetMailBody().

Exemple

  If CreateMail(0, "test@purebasic.com", "Salut")
    SetMailBody(0, "C'est le contenu")
    Debug GetMailBody(0) ; Affiche "C'est le contenu"
  EndIf
