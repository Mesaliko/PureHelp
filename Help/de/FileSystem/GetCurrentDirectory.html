GetCurrentDirectory()

Syntax

Ergebnis$ = GetCurrentDirectory()
Beschreibung
Gibt das aktuelle Verzeichnis des Programms zurück.

Parameter

Diese Funktion hat keine Parameter.

Rückgabewert

Gibt den vollständigen Pfad des aktuellen Verzeichnisses zurück. Dieser wird mit einem Verzeichnis-Trennzeichen #PS, #PS$ ('\' auf Windows, ansonsten '/') enden.

Es ist sehr unwahrscheinlich, aber wenn diese Funktion fehlschlägt, wird ein leerer String zurückgegeben.

Anmerkungen

Alle Datei-Zugriffe erfolgen relativ zu diesem Verzeichnis, wenn kein absoluter Pfad angegeben wird. Zum Ändern des aktuellen Verzeichnisses verwenden Sie SetCurrentDirectory().
