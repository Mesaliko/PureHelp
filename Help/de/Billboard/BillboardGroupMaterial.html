BillboardGroupMaterial()

Syntax

BillboardGroupMaterial(#BillboardGruppe, MaterialID)
Beschreibung
Weist der angegebenen Billboard-Gruppe ein Material zu. Dieses Material von allen Billboards verwendet, welche zu dieser Gruppe hinzugefügt werden. Einer Gruppe kann zur gleichen Zeit nur ein Material zugeordnet sein.

Parameter

#BillboardGruppe Die Nummer der Billboard-Gruppe, der ein neues Material zugewiesen werden soll.
MaterialID Die ID-Nummer des Materials, welches der Billboard-Gruppe (und den Billboards) zugewiesen werden soll. Eine gültige 'MaterialID' kann einfach mit dem MaterialID() Befehl ermittelt werden.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.

Unterstützte OS

Alle

<- BillboardGroupID() - Billboard Inhaltsverzeichnis - BillboardGroupX() ->
