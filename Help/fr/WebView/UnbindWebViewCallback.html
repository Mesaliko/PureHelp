UnbindWebViewCallback()

Syntaxe

UnbindWebViewCallback(#Gadget, FonctionJavaScript$)
Description
Dissocie une fonction JavaScript précédemment liée avec BindWebViewCallback().

Arguments

#Gadget Le gadget d’affichage Web à utiliser.
FonctionJavaScript$ Nom sensible à la casse de la fonction JavaScript à dissocier.

Valeur de retour

Aucune.

Remarques

La fonction JavaScript sera automatiquement supprimée du gadget WebView et ne sera plus disponible dans le code JavaScript.
