InitJoystick()

Syntaxe

Resultat = InitJoystick()
Description
Initialise l'environnement de gestion des joysticks.

Arguments

Aucun.

Valeur de retour

Renvoie le nombre de joysticks disponibles.

Remarques

Vous devez appeler cette fonction avant toute autre fonction de cette bibliothèque.
