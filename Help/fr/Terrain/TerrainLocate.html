TerrainLocate()

Syntaxe

TerrainLocate(#Terrain, X, Y, Z)
Description
Modifie l'emplacement absolu d'un terrain dans le monde.

Arguments

#Terrain Le terrain à utiliser.
X, Y, Z Nouvel emplacement absolu dans le monde (dans l'unité du monde).

Valeur de retour

Aucune.
