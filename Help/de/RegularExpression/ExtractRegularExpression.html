ExtractRegularExpression()

Syntax

Ergebnis = ExtractRegularExpression(#RegularExpression, String$, Array$())
Beschreibung
Extrahiert Strings abhängig vom regulären Ausdruck #RegularExpression in ein Array.

Parameter

#RegularExpression Der zu verwendende reguläre Ausdruck.
String$ Der String, auf den der reguläre Ausdruck angewendet werden soll.
Array$() Die extrahierten Strings werden in diesem Array gespeichert. Die Größe des Arrays wird automatisch an die Anzahl der Elemente im 'String$', welche mit dem Ausdruck zusammenpassen (englisch "matching"), angepasst.

Rückgabewert

Gibt die Anzahl der Elemente zurück, welche auf den regulären Ausdruck im String passen.

Beispiel

  ; Dieser Ausdruck passt auf jedes Wort mit 3 Buchstaben, welches mit einem
  ; Kleinbuchstaben beginnt, gefolgt vom Zeichen 'b' und welches mit einem
  ; Großbuchstaben endet. Beispiel: abC
  ;    
  If CreateRegularExpression(0, "[a-z]b[A-Z]")
    Dim Result$(0)
    NbFound = ExtractRegularExpression(0, "abC ABc zbA abc", Result$())
    For k = 0 To NbFound-1
      Debug Result$(k)
    Next
  Else
    Debug RegularExpressionError()
  EndIf
