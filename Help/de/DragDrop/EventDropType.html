EventDropType()

Syntax

Ergebnis = EventDropType()
Beschreibung
Nachdem ein #PB_Event_GadgetDrop oder #PB_Event_WindowDrop von WaitWindowEvent() oder WindowEvent() empfangen wurde, gibt diese Funktion das Format der abgelegten Daten zurück.

Parameter

Diese Funktion hat keine Parameter.

Rückgabewert

Gibt einen der folgenden Werte oder eine OS-spezifische ID für ein anderes Format (siehe DragOSFormats() für weitere Informationen) zurück: 
  #PB_Drop_Text   : Text wurde abgelegt.        (verwenden Sie EventDropText() um ihn zu erhalten)
  #PB_Drop_Image  : Ein Bild wurde abgelegt.    (verwenden Sie EventDropImage() um es zu erhalten)
  #PB_Drop_Files  : Dateinamen wurden abgelegt. (verwenden Sie EventDropFiles() um sie zu erhalten)
  #PB_Drop_Private: Eine "private" Operation wurde abgeschlossen. (verwenden Sie EventDropPrivate() um deren Typ zu erfahren)

Anmerkungen

Für den Umgang mit OS-spezifischen Formaten können EventDropBuffer() und EventDropSize() verwendet werden.
