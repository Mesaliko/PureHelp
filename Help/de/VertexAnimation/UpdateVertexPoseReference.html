UpdateVertexPoseReference()

Syntax

UpdateVertexPoseReference(#Mesh, Animation$, Track, Keyframe, PoseIndex, Einfluss)
Beschreibung
Aktualisiert einen neuen Posen-Bezug (auch Referenz auf eine Pose) der Animation. Die Animation sollte bereits mittels CreateVertexAnimation() erstellt worden oder vordefiniert im Mesh sein.

Parameter

#Mesh Das zu verwendende Mesh.
Animation$ Der Animations-Name. Dieser Name beachtet die Groß- und Kleinschreibung. Die Animation sollte bereits mittels CreateVertexAnimation() erstellt worden sein.
Track Der Track-Index. Der Track (oder auch Spur, Strecke) muss zuvor mit CreateVertexTrack() erstellt worden sein.
Keyframe Der Keyframe-Index. Das Keyframe muss zuvor mittels CreateVertexPoseKeyFrame() erstellt worden sein.
PoseIndex Der Posen-Index. Der erste Index-Wert für eine Pose beginnt bei Null.
Influence Der neue Einfluss (oder auch das Gewicht) der Pose. Dieser Wert reicht von 0.0 (keinen Einfluss) bis 1.0 (voller Einfluss).

Rückgabewert

Diese Funktion gibt keinen Wert zurück.
