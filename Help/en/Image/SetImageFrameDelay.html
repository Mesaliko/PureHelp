SetImageFrameDelay()

Syntax

Result = SetImageFrameDelay(#Image, Delay)
Description
Sets the display delay (in milliseconds) for the current image frame. Each frame can have its own delay.

Parameters

#Image The image to set the frame delay.

Return value

Sets the display delay (in milliseconds) for the current image frame.

See Also

CreateImage(), AddImageFrame(), RemoveImageFrame(), SetImageFrame(), GetImageFrame(), GetImageFrameDelay()

Supported OS

All
