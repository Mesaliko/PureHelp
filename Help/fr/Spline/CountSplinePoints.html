CountSplinePoints()

Syntaxe

Resultat = CountSplinePoints(#Spline)
Description
Renvoie le nombre de points d'une spline.

Arguments

#Spline La spline à utiliser.

Valeur de retour

Renvoie le nombre de points de la spline.
