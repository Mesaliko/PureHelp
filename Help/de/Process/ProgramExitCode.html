ProgramExitCode()

Syntax

Ergebnis = ProgramExitCode(Programm)
Beschreibung
Gibt den "Exitcode" zurück, der vom angegebenen Programm bei dessen Beendigung zurückgegeben wurde.

Parameter

Programm Das zu verwendende Programm. Dieses muss zuvor mittels RunProgram() gestartet worden sein.

Rückgabewert

Der 'ExitCode', der zurückgegeben wurde, als das angegebene Programm endete.

Anmerkungen

Dieser Befehl sollte nur verwendet werden, nachdem das angegebene Programm beendet wurde. Verwenden Sie ProgramRunning() oder WaitProgram(), um hierbei sicher zu gehen.

Der Exitcode ermöglicht die Rückgabe eines einzelnen Wertes durch das Programm an das Programm, welches es ausgeführt hat. Dieser wird üblicherweise verwendet, um einen Fehler oder Erfolg des Programms anzuzeigen. Bitte beachten, dass aufgrund von OS-Limitierungen auf OS X und Linux die unterstützten Exitcodes zwischen 0 und 255 liegen.

Um einen "Exitcode" durch ein PB Programm zurückgeben zu lassen, verwenden Sie den optionalen Wert bei der End Anweisung: 
  End 1   ; gibt den Exitcode 1 zurück
