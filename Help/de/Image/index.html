PureBasic - Image

Einleitung

Images (Bilder) sind grafische Objekte, welche in einem Fenster oder in verschiedenen Gadgets dargestellt werden können. PureBasic unterstützt neben den Bild-Typen BMP und Icon (.ico, nur auf Windows) auch alle von der ImagePlugin Bibliothek unterstützten Bild-Formate.

Transparente PNG-Bilder können verwendet werden, um in den Gadget-, Menü- und Toolbar-Images Transparenz zu aktivieren. Auf Windows können auch transparente ICON-Bilder verwendet werden. PNG sollte aber bevorzugt werden, da es auf allen Plattformen funktioniert. Transparenz bei BMP-Bildern wird nicht unterstützt.
