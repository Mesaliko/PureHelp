LoadMovie()

Syntaxe

Resultat = LoadMovie(#Video, Fichier$)
Description
Charge une vidéo depuis un fichier.

Arguments

#Video Le numéro d'identification de la vidéo.

PB_Any # peut être utilisé pour générer automatiquement ce numéro.
Fichier$ Le nom et le chemin du fichier vidéo.

Valeur de retour

Renvoie une valeur non nulle en cas de succès, zéro sinon.

Voir aussi

PlayMovie(), MovieInfo(), MovieLength(), MovieWidth(), MovieHeight(), FreeMovie(), InitMovie()

OS Supportés

Tous
