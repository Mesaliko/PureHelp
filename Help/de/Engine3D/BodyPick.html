BodyPick()

Syntax

Ergebnis = BodyPick(#Camera, Ausgewählt, x, y, Gesperrt)
Beschreibung
Simuliert einen Mausklick und beginnt das Entity-Objekt an der angegebenen Koordinate zu manipulieren.

Parameter

#Camera Die zu verwendende Kamera.
Ausgewählt Wenn #True, dann wird der ausgewählte Körper (englisch "picked body") nicht freigegeben bis ein neuer Funktionsaufruf mit diesem Parameter diesen auf #False setzt. Es kann nur einen ausgewählten Körper zur gleichen Zeit geben.
x, y Die Koordinaten (in Pixel) des Punktes.
Gesperrt Wenn auf #True gesetzt, wird der ausgewählte Körper sich nicht frei drehen können, wenn er bewegt wird. Wenn auf #False gesetzt, wird sich der Körper frei drehen entsprechend der Körperbewegung.

Rückgabewert

Das ausgewählte ("picked") #Entity, oder -1 wenn an diesen Koordinaten kein Entity-Objekt gefunden wurde.

Unterstützte OS

Alle

<- AntialiasingMode() - Engine3D Inhaltsverzeichnis - CheckObjectVisibility() ->
