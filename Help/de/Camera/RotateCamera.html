RotateCamera()

Syntax

RotateCamera(#Kamera, x, y, z [, Modus])
Beschreibung
Rotiert die #Kamera entsprechend der angegebenen x,y,z Winkel-Angaben.

Parameter

#Kamera Die zu rotierende (drehende) Kamera.
x, y, z Die neue Rotation, welche auf die Kamera angewandt werden soll. Alle Winkel werden in Grad angegeben, mit Werten von 0 bis 360.
Modus (optional) Dies kann einer der folgende Werte sein: 
  PB_Absolute: absolute Rotation (Standard).
  PB_Relative: relative Rotation, basierend auf der vorherigen Kamera-Rotation.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.

Unterstützte OS

Alle

<- ResizeCamera() - Camera Inhaltsverzeichnis
