PureBasic - Joystick

Généralités

PureBasic permet la gestion complète des joysticks connectés à l'ordinateur.
Tous les joysticks et les pads sont acceptés, même les manettes multi-boutons, triggers, etc.

Remarques

Sur Windows l'interface est assurée par la technologie DirectX et sur Linux avec SDL.

Sommaire des commandes

ExamineJoystick
InitJoystick
JoystickAxisX
JoystickAxisY
JoystickAxisZ
JoystickButton
JoystickName

Exemple

Joystick.pb

OS Supportés

Tous
