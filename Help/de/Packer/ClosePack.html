ClosePack()

Syntax

ClosePack(#Pack)
Beschreibung
Schließt die angegebene Paket-Datei.

Parameter

#Pack Das zu schließende Paket. Wenn #PB_All angegeben wird, dann werden alle verbliebenen Pakete geschlossen.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.

Anmerkungen

Alle noch offenen Paket-Dateien werden automatisch geschlossen, wenn das Programm endet.

Unterstützte OS

Alle

<- CatchPack() - Packer Inhaltsverzeichnis - CompressMemory() ->
