CameraCustomParameter()

Syntax

CameraCustomParameter(#Kamera, ParameterIndex, Wert1, Wert2, Wert3, Wert4)
Beschreibung
Legt einen benutzerdefinierten Parameterwert im Kamera-Shader-Skript (entweder GLSL oder HLSL) fest.

Parameter

#Kamera Die zu verwendende Kamera.
ParameterIndex Der Parameter-Index im Shader-Skript.
Wert1 Der erste Parameter-Wert.
Wert2 Der zweite Parameter-Wert (sofern der Parameter nur einen Wert akzeptiert, wird dieser Wert ignoriert).
Wert3 Der dritte Parameter-Wert (sofern der Parameter nur einen Wert akzeptiert, wird dieser Wert ignoriert).
Wert4 Der vierte Parameter-Wert (sofern der Parameter nur einen Wert akzeptiert, wird dieser Wert ignoriert).

Rückgabewert

Diese Funktion gibt keinen Wert zurück.

Unterstützte OS

Alle

<- CameraBackColor() - Camera Inhaltsverzeichnis - CameraDirection() ->
