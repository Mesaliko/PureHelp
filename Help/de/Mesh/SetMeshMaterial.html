SetMeshMaterial()

Syntax

SetMeshMaterial(#Mesh, MaterialID [, SubMesh])
Beschreibung
Legt das Standard-Material für das Mesh-Objekt fest.

Parameter

Mesh Das zu verwendende Mesh-Objekt.
MaterialID Gibt das Standard-Material an, welches für das Mesh verwendet werden soll. Um eine gültige 'MaterialID' zu erhalten, verwenden Sie MaterialID().
SubMesh (optional) Wenn angegeben, wird das Material nur auf das Sub-Mesh angewendet. Der Index des ersten Sub-Mesh lautet 0 (Haupt-Mesh).

Rückgabewert

Diese Funktion gibt keinen Wert zurück.
