WindowID()

Syntaxe

Resultat = WindowID(#Fenetre)
Description
Renvoie l'identifiant unique de la fenêtre dans le système d'exploitation.

Arguments

#Fenetre La fenêtre à utiliser.

Remarques

Cette fonction est très utile car FenetreID est souvent demandé par d'autres fonctions.

Le résultat FenetreID est aussi appelé 'Handle'. Voir le chapitre de l'aide Handles et Nombres pour plus d'informations.

Exemple

  If OpenWindow(0, 0, 0, 220, 100, "Exemple...", #PB_Window_SystemMenu | #PB_Window_ScreenCentered)
    
   ButtonGadget  (1, 10, 60, 200, 30, "Identifiant de la fenêtre")
   
   Repeat
     Event = WaitWindowEvent()
          
     Select Event
     
       Case #PB_Event_Gadget
         Select EventGadget()
           Case 1
             
             ID=WindowID(0)
             Debug "Identifiant de la fenêtre : " + Str(ID)
                                                          
           EndSelect
            
     EndSelect
   Until Event = #PB_Event_CloseWindow
 EndIf
