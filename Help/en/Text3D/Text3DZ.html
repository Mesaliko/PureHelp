Text3DZ()

Syntax

Result = Text3DZ(#Text3D)
Description
Returns the absolute 'z' position of the text in the world.

Parameters

#Text3D The text to use.

Return value

Returns the absolute 'z' position of the text.

See Also

Text3DX(), Text3DY(), MoveText3D()

Supported OS

All
