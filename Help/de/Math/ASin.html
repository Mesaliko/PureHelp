ASin()

Syntax

Ergebnis.f(.d) = ASin(Wert.f(.d))
Beschreibung
Gibt den Arkussinus des angegebenen Wertes zurück.

Parameter

Wert.f oder Wert.d Der Eingabewert. Dieser muss zwischen -1.0 und 1.0 liegen (-1.0 und 1.0 eingeschlossen).

Rückgabewert

Gibt den resultierenden Winkel im Bogenmaß (Radiant) zurück. Dieser kann mittels der Funktion Degree() in Grad umgewandelt werden.

Anmerkungen

Dies ist die Umkehrfunktion von Sin(). Diese Funktion verarbeitet und gibt Float oder Double Werte zurück.

Beispiel

  Debug ASin(1) ; wird '1.570796...' (Pi/2) anzeigen
  Debug ASin(0) ; wird '0.0' anzeigen
