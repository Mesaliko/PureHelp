IsDialog()

Syntax

Ergebnis = IsDialog(#Dialog)
Beschreibung
Überprüft, ob die angegebene Dialog-Nummer ein gültiger Dialog ist.

Parameter

#Dialog Der zu testende Dialog.

Rückgabewert

Gibt einen Wert ungleich Null zurück, wenn #Dialog ein gültiger Dialog ist, und Null wenn nicht.

Anmerkungen

Diese Funktion ist "kugelsicher" und kann mit jedem Wert verwendet werden. Dies ist der korrekte Weg, um sicherzustellen, dass ein Dialog fertig zur Verwendung ist.
