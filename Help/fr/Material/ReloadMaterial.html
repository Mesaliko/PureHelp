ReloadMaterial()

Syntaxe

ReloadMaterial(NomMatiere$, NomFichierScript$, ParseScript)
Description
Recharge une matière à partir d'un script OGRE.

Arguments

NomMatiere$ Le nom de la matière à utiliser.
NomFichierScript$ Le nom du fichier script à utiliser.
ParseScript 
  #True : Le 'NomFichierScript$' est analysé à nouveau pour obtenir des informations de mises à jour sur la matière.

Valeur de retour

Aucune.

Remarques

Ceci est utile lors de l'utilisation de matériaux (matières) personnalisé(e)s stocké(e)s dans des fichiers de script.

Un matériau peut être obtenu à partir des scripts OGRE par GetScriptMaterial().
