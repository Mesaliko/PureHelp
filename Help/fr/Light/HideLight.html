HideLight()

Syntaxe

HideLight(#Lumiere, Etat)
Description
Affiche ou cache une lumière.

Arguments

#Lumiere La lumière à utiliser.
Etat 
  #True : la lumière est cachée
  #False: la lumière est affichée

Valeur de retour

Aucune.
