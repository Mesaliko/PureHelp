UCase()

Syntax

Ergebnis$ = UCase(String$)
Beschreibung
Gibt den originalen String - konvertiert in Großbuchstaben (wenn möglich) - zurück.

Parameter

String$ Der in Großbuchstaben zu konvertierende String.

Rückgabewert

Der in Großbuchstaben konvertierte String.

Anmerkungen

Diese Funktion unterstützt auch Buchstaben mit Betonung; wenn also ein kleines 'é' gefunden wurde, wird dieses in ein großes 'É' konvertiert.

Beispiel

  Debug UCase("This is Art") ; wird "THIS IS ART" anzeigen
