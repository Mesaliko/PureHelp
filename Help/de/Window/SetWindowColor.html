SetWindowColor()

Syntax

SetWindowColor(#Window, Farbe)
Beschreibung
Ändert die Hintergrundfarbe des angegebenen Fensters.

Parameter

#Window Das zu verwendende Fenster.
Farbe Die neue Farbe, welche für den Fenster-Hintergrund verwendet werden soll. RGB() kann zum Ermitteln eines gültigen Farbwertes benutzt werden. Eine Farbtabelle mit gebräuchlichen Farben finden Sie hier.

Wenn der Farbwert auf #PB_Default gesetzt wird, dann wird der Hintergrund auf die Standardfarbe zurückgesetzt.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.

Anmerkungen

GetWindowColor() kann verwendet werden, um die aktuelle Hintergrundfarbe des Fensters zu ermitteln.
