UseDialogWebViewGadget()

Syntaxe

UseDialogWebViewGadget()
Description
Active la prise en charge de WebViewGadget() dans la bibliothèque de dialogue.
Ceci n'est pas activé par défaut pour réduire la taille de la bibliothèque de dialogue si WebViewGadget() n'est pas nécessaire.

Valeur de retour

Aucune.
