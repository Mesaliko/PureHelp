GetGadgetData3D()

Syntax

Ergebnis = GetGadgetData3D(#Gadget3D)
Beschreibung
Gibt den 'Data' Wert zurück, welcher zuvor für dieses Gadget mittels SetGadgetData3D() gespeichert wurde. Dies ermöglicht es, jedem Gadget einen individuellen Wert zuzuweisen.

Parameter

#Gadget3D Das zu verwendende 3D-Gadget.

Rückgabewert

Der aktuelle 'Data'-Wert, der diesem Gadget zugewiesen wurde - oder Null, wenn kein Wert festgelegt wurde.

Anmerkungen

Diese Funktion wird von allen 3D-Gadgets unterstützt. Siehe SetGadgetData3D() für ein Beispiel.
