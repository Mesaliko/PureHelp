GetGadgetAttribute()

Syntaxe

Resultat = GetGadgetAttribute(#Gadget, Attribut)
Description
Renvoie la valeur de l'attribut d'un gadget.

Arguments

#Gadget Le gadget à utiliser
Attribut L'attribut à tester.

Valeur de retour

Renvoie la valeur de l'attribut spécifié ou 0 si le gadget ne supporte pas l'attribut.

Remarques

Cette fonction est disponible avec les gadgets suivants:

- ButtonImageGadget()
- CalendarGadget()
- CanvasGadget()
- DateGadget()
- EditorGadget()
- ExplorerListGadget()
- ListIconGadget()
- OpenGLGadget()
- PanelGadget()
- ProgressBarGadget()
- ScrollAreaGadget()
- ScrollBarGadget()
- SpinGadget()
- SplitterGadget()
- StringGadget()
- TrackBarGadget()
- WebGadget()
