GetImageFrameDelay()

Syntax

Ergebnis = GetImageFrameDelay(#Image)
Beschreibung
Gibt die Anzeigeverzögerung (in Millisekunden) für das aktuelle "Frame" (Einzelbild) zurück. Jedes Frame kann seine eigene Verzögerung haben.

Parameter

#Image Das Bild, von dem die Frame-Verzögerung ermittelt werden soll.

Rückgabewert

Gibt die Anzeigeverzögerung (in Millisekunden) für das aktuelle Bild-Frame zurück.
