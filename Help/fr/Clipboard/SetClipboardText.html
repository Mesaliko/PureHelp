SetClipboardText()

Syntaxe

SetClipboardText(Texte$)
Description
Place un texte dans le presse-papier.

Arguments

Texte$ Le texte à placer dans le presse-papier

Valeur de retour

Aucune.

Remarques

Si le presse-papier contient déjà un texte celui-ci est remplacé.

Exemple

  ; On place un texte dans le presse papier
  SetClipboardText("Un exemple de texte")
  MessageRequester("Presse-papier","Le presse papier contient le texte suivant : " + GetClipboardText())
