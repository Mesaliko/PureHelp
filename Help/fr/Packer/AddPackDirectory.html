AddPackDirectory()

Syntaxe

Resultat = AddPackDirectory(#Archive, Repertoire$)
Description
Ajoute un nouveau répertoire vide à l'archive créée avec CreatePack().

Arguments

#Pack L'archive à utiliser.
NomRépertoire$ Le nom du répertoire vide à ajouter à l'archive.

Valeur de retour

Renvoie une valeur non nulle si le répertoire vide a été ajouté avec succès à l'archive.
