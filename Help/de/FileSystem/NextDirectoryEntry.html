NextDirectoryEntry()

Syntax

Ergebnis = NextDirectoryEntry(#Verzeichnis)
Beschreibung
Diese Funktion muss nach einem ExamineDirectory() aufgerufen werden. Sie wird Schritt für Schritt durch das Verzeichnis gehen und dessen Inhalt auflisten.

Parameter

#Verzeichnis Das Verzeichnis, welches mit ExamineDirectory() untersucht wird.

Rückgabewert

Gibt einen Wert ungleich Null zurück, wenn ein neuer Eintrag vom Verzeichnis gelesen wurde, und Null, wenn es keine weiteren Einträge gibt.

Anmerkungen

Der Name des Eintrags kann mit Hilfe der Funktion DirectoryEntryName() ermittelt werden. Wenn Sie wissen möchten, ob ein Eintrag ein Unterverzeichnis oder eine Datei ist, verwenden Sie die DirectoryEntryType() Funktion.
