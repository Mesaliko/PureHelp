CloseLibrary()

Syntaxe

CloseLibrary(#Bibliotheque)
Description
Ferme une bibliotheque et libère la mémoire associée.

Arguments

#Bibliotheque Le numéro de la bibliothèque.

Si #PB_All est spécifié, toutes les autres bibliothèques seront fermées.

Valeur de retour

Aucune.

Remarques

Toutes les bibliothèques ouvertes sont automatiquement fermées quand le programme se termine.
