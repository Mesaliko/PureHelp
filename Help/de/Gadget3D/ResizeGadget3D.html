ResizeGadget3D()

Syntax

ResizeGadget3D(#Gadget3D, x, y, Breite, Höhe)
Beschreibung
Ändert das angegebene 3D-Gadget auf die angegebene neue Position und Dimensionen.

Parameter

#Gadget3D Das zu verwendende 3D-Gadget.
x, y, Breite, Höhe Die neue Position und Dimensionen des Gadgets.

Um das Erstellen von in Echtzeit größenveränderbaren ("realtime resizeable") Graphical User Interface (GUI) zu vereinfachen, kann jeweils #PB_Ignore als Parameter (x, y, Breite oder Höhe) übergeben werden und dieser Parameter wird dann nicht verändert.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.

Beispiel

  [...]

  ResizeGadget3D(0, #PB_Ignore, #PB_Ignore, 300, #PB_Ignore) ; Ändert nur die Gadgetbreite.
