IsThread()

Syntax

Ergebnis = IsThread(Thread)
Beschreibung
Überprüft, ob die angegebene Thread-Nummer ein von CreateThread() erzeugter Thread ist, und ob dieser noch läuft.

Parameter

Thread Der zu verwendende Thread.

Rückgabewert

Ein Wert ungleich Null, wenn der Thread noch gültig ist und läuft, andernfalls Null.

Unterstützte OS

Alle

<- FreeSemaphore() - Thread Inhaltsverzeichnis - KillThread() ->
